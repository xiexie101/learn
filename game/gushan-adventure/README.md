# 🏔️ 鼓山·漫步奇遇记 | Gushan Adventure

一个户外爬山互动体验应用，将鼓山爬山变成有趣的探险任务。

## 🎯 游戏玩法

结合福州鼓山文化，设计了 5 个闯关节点：

| 节点 | 地点 | 任务类型 |
|------|------|----------|
| 1️⃣ 山脚启程 | 闽山第一亭 | 📸 拍照打卡 |
| 2️⃣ 摩崖石刻 | 文人墨客区 | 🎮 找字挑战 |
| 3️⃣ 半山亭 | 休息区 | 💬 话题卡抽取 |
| 4️⃣ 眺望台 | 观景位 | 📸 拍照打卡 |
| 5️⃣ 涌泉寺 | 登顶终点 | 🎉 完成庆祝 |

## 🚀 快速开始

```bash
cd gushan-adventure
python3 -m http.server 8080
```

用手机浏览器访问：http://[电脑IP]:8080

## ✨ 功能特性

- 📚 **文化锦囊** - 每个节点都有福州本地文化介绍
- 🎯 **互动任务** - 拍照、找字、话题讨论等多种玩法
- 📊 **进度保存** - 使用 localStorage 保存进度
- 🎉 **登顶庆祝** - 完成所有任务后的庆祝动画

## 📱 适用场景

- 亲子户外活动
- 情侣爬山约会
- 朋友团建出游
