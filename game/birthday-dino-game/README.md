# 🦖 恐龙大作战 - 小小英雄的冒险

一个基于手势识别的体感打恐龙游戏，专为儿童生日派对设计。

## 🎮 游戏玩法

1. **开始游戏** - 点击「开始战斗」按钮
2. **允许摄像头** - 需要摄像头权限来识别手势
3. **握拳出击** - 握紧拳头 + 快速挥动 = 打击恐龙
4. **完成任务** - 打败 10 只恐龙即获胜！

## 🚀 快速开始

### 方式一：本地服务器
```bash
cd birthday-dino-game
python3 -m http.server 8899
```
打开浏览器访问：http://localhost:8899

### 方式二：直接打开
部分浏览器支持直接打开 `index.html`（Chrome 需要本地服务器）

## 📺 投屏到电视玩

1. 电脑打开游戏网页
2. 使用投屏功能（AirPlay / 有线连接）投到电视
3. 孩子站在摄像头前，挥拳打恐龙！

## ⚙️ 自定义配置

编辑 `index.html` 中的 `CONFIG` 对象：

```javascript
const CONFIG = {
    targetScore: 10,        // 需要击败的恐龙数量
    spawnInterval: 2000,    // 恐龙生成间隔(毫秒)
    dinoLifetime: 4000,     // 恐龙存活时间(毫秒)
    punchThreshold: 0.15,   // 出拳灵敏度(越小越灵敏)
    hitDistance: 150,       // 击中判定距离(像素)
};
```

## 🎯 功能特性

| 功能 | 说明 |
|------|------|
| 🎥 手势识别 | MediaPipe 追踪双手位置 |
| 👊 出拳检测 | 握拳 + 快速移动触发攻击 |
| 🦖 多种恐龙 | 随机出现不同恐龙表情 |
| 💥 击中特效 | 粒子爆炸 + 消失动画 |
| 🔥 连击系统 | 连续击中显示 Combo |
| 🎵 音效 | 击中和胜利音效 |
| 🏆 胜利庆祝 | 烟花 + 英雄称号 |

## 📱 设备要求

- 现代浏览器（Chrome / Safari / Edge）
- 摄像头
- 网络连接（加载 MediaPipe 库）
