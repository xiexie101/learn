# 设计方案：水墨八段锦 (Ink Wash Baduanjin)

**日期**: 2025-12-31
**作者**: Antigravity
**状态**: 已确认

## 1. 设计概述
本方案旨在彻底重构“禅意花园运动追踪器”的视觉风格。抛弃原有的通用 Glassmorphism（玻璃拟态），转而采用极具中国传统文化韵味的**“水墨写意 + 翠竹绿”**风格。目标是打造一个既古典雅致，又符合现代 Web 审美的高级感应用，让“八段锦”的修身养性在界面上得到视觉延伸。

## 2. 视觉风格指南

### 2.1 核心意象
*   **水墨 (Ink Wash)**：墨分五色（焦、浓、重、淡、清），通过深浅不一的灰阶和自然的笔触边缘，营造流动、通透的质感。
*   **宣纸 (Rice Paper)**：背景使用带有细腻纤维纹理的暖米色，还原纸张的温润触感，避免屏幕冷光的刺眼。
*   **翠竹 (Bamboo)**：作为唯一的强调色，象征君子之风、生机勃勃。用于点亮关键交互和积极反馈。

### 2.2 配色方案 (Color Palette)

| 用途 | 色彩名称 | 颜色代码 (Hex) | 说明 |
| :--- | :--- | :--- | :--- |
| **背景** | 宣纸白 | `#F7F4ED` | 基础背景，需叠加纹理杂点 |
| **主墨色** | 焦墨 | `#1A1A1A` | 标题、强调文字 |
| **副墨色** | 浓墨 | `#333333` | 正文文字 |
| **淡墨色** | 淡墨 | `#666666` | 次要信息、边框细节 |
| **渲染色** | 清墨 | `rgba(0,0,0,0.05)` | 阴影、容器背景底色 |
| **强调色** | **翠竹绿** | `#4A7050` | 主按钮、进度条、Active 状态 |
| **辅助色** | 丹砂红 | `#BC412B` | 少量点缀（如印章风格的徽章） |

### 2.3 排版 (Typography)
*   **标题字体**：优先使用楷体 (`KaiTi`, `STKaiti`) 或行楷，找不到则回退到宋体 (`SimSun`, `Noto Serif SC`)。CSS 设置应保留书法笔触的韵律感。
*   **正文字体**：现代衬线体或无衬线体 (`Noto Serif SC`, `PingFang SC`)，保证小字号下的易读性。

## 3. UI 组件设计

### 3.1 布局容器
*   **放弃圆角矩形卡片**：改为富有“留白”意境的布局。
*   **边框处理**：使用类似于毛笔勾勒的不规则边缘或极细的淡墨线条。
*   **背景**：容器背景可以是极淡的水墨晕染，或者直接利用大背景的留白。

### 3.2 交互元素
*   **按钮**：
    *   **主按钮**：墨色或竹绿色底，带有些微的笔触材质感。Hover 时有水墨扩散的动效。
    *   **复选框 (Move Item)**：选中后变为竹绿色的水墨点或竹叶图标。
*   **进度条**：
    *   底槽为淡墨笔触，进度填充为翠竹绿色的一笔挥洒。

### 3.3 禅意花园 (Zen Garden)
*   完全重绘 Garden 内的 SVG 元素。
*   **石头**：水墨皴法勾勒。
*   **植物**：写意画法，竹子、幽兰、松树。
*   **动效**：元素出现时如墨滴入水般晕开 (Ink spread animation)。

### 3.4 图标与徽章
*   **成就徽章**：设计为传统的“印章”样式（朱红或墨色），解锁后颜色鲜艳（丹砂或翠竹），未解锁为淡墨水印。
*   **功能图标**：细线条的水墨风格图标 (Lucide icons 可通过 CSS 滤镜或 stroke 调整模拟)。

## 4. 技术实现路径

### 4.1 CSS 策略
*   **Tailwind 配置**：扩展 `tailwind.config.js`，添加自定义颜色和字体族。
*   **纹理实现**：使用 CSS `background-image` 和 `blend-mode` 叠加噪点图片或 SVG pattern 来模拟宣纸质感。
*   **字体降级**：合理设置 `font-family` 栈，确保在不同系统（Win/Mac/Mobile）上都能尽可能呈现书法韵味。

### 4.2 关键代码片段预演

```css
/* 宣纸背景 */
.bg-rice-paper {
  background-color: #F7F4ED;
  background-image: url("data:image/svg+xml,..."); /* 噪点纹理 */
}

/* 水墨容器 */
.card-ink {
  background: rgba(255, 255, 255, 0.6);
  border: 1px solid rgba(0, 0, 0, 0.1);
  box-shadow: 0 4px 20px rgba(0, 0, 0, 0.05);
  backdrop-filter: blur(4px); /* 保留一点点磨砂感，但更像薄纸 */
}

/* 书法标题 */
.font-shufa {
  font-family: "KaiTi", "STKaiti", "Noto Serif SC", serif;
}
```
