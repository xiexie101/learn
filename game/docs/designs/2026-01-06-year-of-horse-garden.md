# 2026 马年庆典：禅意花园“龙马精神”更新设计

**日期**：2026-01-06
**状态**：已确认
**标签**：UI, Feature, Animation

## 1. 概述
响应“马年”主题，为禅意花园运动追踪器增添节日氛围。核心是引入具有生命力的“水墨骏马”，象征用户在健康修行路上的活力与坚持。

## 2. 核心体验：水墨骏马

### 2.1 视觉风格
- **技法**：采用没骨画法（Mogu）绘制马身，表现墨色晕染的体积感；用干笔焦墨（Gugfa）勾勒鬃毛和马尾，体现苍劲有力。
- **色彩**：以水墨黑灰为主，鬃毛或马蹄处可带有极淡的赭石色，保持禅意淡雅，不抢夺视觉焦点。
- **动态呼吸**：利用 SVG `<animate>` 或 CSS 配合 Filter，让马的尾巴和鬃毛有微风拂动的效果（墨韵流动）。

### 2.2 行为逻辑
- **入住条件**：作为 2026 马年福利，**无等级门槛，直接入住**。
- **默认状态**：
    - 位置：随机出现在草地较空旷区域。
    - 动作：低头吃草、或者抬头眺望远方（SVG 路径变形）。
- **交互反馈**：
    - **点击/悬停**：马儿会发出嘶鸣（视觉上的震动或文字气泡 "嘶~"），或者轻快地小跳一下。
    - **完成今日练习**：这是高光时刻。当用户点击“完成今日练习”时，马儿会绕着花园（或者在屏幕特定区域）奔跑一圈。
        - 动画：SVG Path 沿路径运动 + 蹄子摆动。
        - 寓意：马到成功，一日千里。

## 3. 氛围装饰

### 3.1 界面元素
- **闲章**：标题栏右侧增加一枚红色的“丙午”或“大吉”印章，采用古朴的篆书风格。
- **配色微调**：在现有的“米纸白 + 竹青”基调上，极其克制地引入一点点“朱砂红”（Vermilion），仅用于印章和重点高亮，提气而不俗艳。

### 3.2 成就
- **新成就**：【龙马精神】
- **获取条件**：在马年期间（2026年）累计打卡达到一定天数（如 10 天），或只需在春节期间登录练习一次。
- **图标**：一个小马驹的抽象线条 ICON。

## 4. 技术实现要点
- **SVG 优化**：马的 SVG 代码需要精简，利用 `feTurbulence` 和 `feDisplacementMap` 实现水墨边缘的动态“毛刺感”，避免复杂的逐帧动画。
- **兼容性**：确保在低性能模式下（Low Profile），马儿依然可见，但可以关闭复杂的滤镜效果。
