# 「画中游」— AI赋能传统古画互动体验
**日期:** 2025-12-24

## 背景

用户希望探索 AI 赋能传统古画的新玩法，灵感来源于上海交通大学的互动投影项目（AI修复传世名画 + 实时互动投影）。目标是创建一个能让古画"活"起来的 Web 应用，面向儿童/亲子和 Z 世代用户群体。

## 讨论过程

### 关键问题与决策

| 问题 | 选项 | 最终选择 |
|-----|------|---------|
| 应用场景 | 线下展览/线上Web/教育/文创 | **线上 Web/App** |
| 核心玩法 | 走进画中/古画复活/风格迁移/互动问答/共创 | **古画复活** |
| 古画类型 | 人物/山水/花鸟/风俗/通用 | **通用平台（MVP单画）** |
| 互动方式 | 一键自动/点触互动/语音控制/AR触发 | **手势点触互动** |
| 目标用户 | 儿童/Z世代/文化爱好者/教育/大众 | **儿童/亲子 + Z世代** |
| MVP范围 | 单画Demo/小型画廊/开放平台 | **单画 Demo** |
| 选定古画 | 千里江山图/清明上河图/百骏图等 | **《清明上河图》** |
| 动画方案 | AI生成(Runway/Pika) / 手工Sprite | **AI 生成动画** |

### 交互方式迭代
1. **初版**：点击触发局部动画
2. **用户反馈**：希望用"擦除迷雾"方式
3. **再迭代**：用户明确只有手指当前位置显示动画，手指离开恢复静态
4. **最终方案**：魔法聚光灯效果

## 最终方案

### 核心体验
用户手指在《清明上河图》上滑动，**手指所在位置**的画面"活"过来（人物行走、船只划动、水波流动），手指离开后该区域**立即回归静态**。

### 视觉效果
- 以手指为圆心，半径约 100-150px 的圆形区域显示动画
- 边缘有柔和渐变过渡
- 动静对比形成强烈的"魔法"感

### 选择理由
- 符合儿童的探索天性（到处摸一摸看看会发生什么）
- 对 Z 世代有新奇感，适合录屏分享
- 技术实现可控，MVP 可快速验证

## 架构设计

### 技术栈
- **前端**：HTML5 + CSS3 + JavaScript（无框架）
- **动画**：Canvas 遮罩 + Video 播放
- **手势**：Touch Events + Mouse Events
- **音频**：Web Audio API（跟随手指位置的空间音效）

### 分层结构
```
Layer 3: 遮罩层（手指位置挖洞）
Layer 2: 动画层（AI生成的动画视频）
Layer 1: 静态层（高清原图）
```

### 动画素材制作
1. 将《清明上河图》分割为多个区域
2. 使用 Runway Gen-3 / Pika 生成 3-5 秒循环动画
3. 拼接成完整动画长卷
4. 导出 WebM/MP4 格式
