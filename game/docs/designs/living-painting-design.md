# 🎨 「画中游」— AI赋能传统古画互动体验

> **项目代号**：Living Painting / 活画  
> **创建日期**：2024-12-24  
> **状态**：头脑风暴完成，待开发

---

## 📌 项目概述

### 核心理念
用手指唤醒千年古画——用户的指尖就是穿越时空的钥匙，触碰到哪里，北宋汴京的生活就在哪里复活。

### 产品形态
- **平台**：Web应用（移动端优先，兼容桌面端）
- **MVP选定古画**：《清明上河图》

### 目标用户
| 用户群体 | 需求特征 | 设计考量 |
|---------|---------|---------|
| 儿童/亲子 | 趣味性强、易上手、有教育意义 | 简洁UI、引导提示、知识彩蛋 |
| Z世代年轻人 | 新奇好玩、可分享社交媒体 | 潮流视觉、分享功能、短视频友好 |

### 具体体验
1. **默认状态**：整幅《清明上河图》显示为静态高清古画，可横向滚动浏览
2. **手指触碰**：触碰位置为圆心的区域"活"过来：
   - 河水流动、波光粼粼
   - 船只缓缓划行
   - 行人走动、交谈、买卖
   - 店铺招牌摇晃、店员招呼
   - 动物奔跑嬉戏
3. **手指离开**：该区域立即回归静态，魔法消失
4. **声音配合**：手指所在区域播放对应环境音（流水、人声、鸟鸣等）

## 🛠️ 技术方案

### 架构设计

```
┌─────────────────────────────────────────────────────┐
│                    视觉分层结构                       │
├─────────────────────────────────────────────────────┤
│  Layer 3: 遮罩层                                     │
│           - 默认完全遮挡动画层                        │
│           - 手指位置挖出圆形透明"洞"                  │
│           - 边缘有羽化渐变效果                        │
├─────────────────────────────────────────────────────┤
│  Layer 2: 动画层                                     │
│           - 带动效的《清明上河图》                    │
│           - 使用AI生成的视频/动画序列                 │
│           - 循环播放                                  │
├─────────────────────────────────────────────────────┤
│  Layer 1: 静态层                                     │
│           - 原始《清明上河图》高清静态图              │
│           - 作为底层兜底显示                          │
└─────────────────────────────────────────────────────┘
```

### 技术栈
| 层面 | 技术选型 | 说明 |
|-----|---------|------|
| 前端框架 | 纯HTML5 + CSS3 + JS | 保持轻量，无需复杂框架 |
| 动画实现 | Canvas + Video | Canvas绘制遮罩，Video播放动画 |
| 手势识别 | Touch/Mouse Events | 原生API，兼容移动端和桌面端 |
| 音频 | Web Audio API | 空间化音效，跟随手指位置 |

### 动画素材制作
**采用方案A：AI生成动画**

| 工具 | 用途 | 优势 |
|-----|------|-----|
| Runway Gen-3 | 将静态古画片段生成动画视频 | 效果自然，速度快 |
| Pika Labs | 补充生成特定动效 | 细节控制好 |
| Kling AI | 备选方案 | 国产工具，访问方便 |

**制作流程**：
1. 将《清明上河图》分割为多个区域片段
2. 对每个片段使用AI生成3-5秒循环动画
3. 拼接成完整的动画版长卷
4. 导出为Web友好的视频格式（WebM/MP4）
