# 设计方案：漂移的双人舞 (The Dérive of Two)

**日期**: 2026-01-15
**类型**: Web App / 移动端网页
**风格**: 极简主义 / 瑞士风格 (Swiss Style)

## 1. 核心概念
这是一个专为**福清利桥古街**与**石竹山**设计的双人 CityWalk 辅助工具。它不指示路线，而是通过随机抽取的“心理地理学”任务，引导两人在空间中构建独特的“情境”，从而打破尴尬，深化关系。

## 2. 交互流程 (User Flow)

### 2.1 启动 (The Landing)
*   **界面**: 极干净的纯色背景（或微微的噪点质感）。
*   **元素**: 标题《漂移的双人舞》，简短的引导语“让城市成为你们关系的迷宫”。
*   **动作**: 只有一个显眼的“开始漂移”按钮。

### 2.2 命运轮盘 (The Spin)
*   **核心隐喻**: 轮盘赌 / 罗盘。
*   **视觉**: 极简线条构成的圆环，分为四个扇区，对应四个花色：
    *   【共振】 (Attunement)
    *   【异化】 (Détournement)
    *   【共谋】 (Conspiracy)
    *   【祈梦】 (The Dream)
*   **交互**: 点击或滑动屏幕，轮盘快速旋转，伴随清脆的机械咔哒声（可选），最终停在某个花色上。

### 2.3 任务卡片 (The Card)
*   **展示**: 轮盘停止后，该花色扇区放大，翻转出一张具体的任务卡。
*   **内容**: 显示任务编号（如 #03）和具体指令（如“寻找一种属于‘利桥古街’特有的红色...”）。
*   **版式**: 瑞士风格排版，大字号标题，左对齐，清晰易读。

### 2.4 决策时刻 (The Veto)
*   **界面**: 任务下方有两个明显的按钮。
    *   **[接受挑战] (Accept)**: 确认并在心中（或行动上）锁定该任务。点击后收起卡片，回到待机状态或显示“进行中”。
    *   **[使用否决权] (Veto)**: 拒绝执行该任务。
*   **惩罚机制**: 如果点击“Veto”，屏幕瞬间变色（如变为警告红或深黑），弹出一个“真心话”问题。
    *   *规则*: “跳过任务的代价：必须诚实回答这个问题。”
    *   *示例*: “你对我的第一印象是什么？” / “分享一个童年的秘密。”

## 3. 视觉风格指南 (Visual Style)

*   **关键词**: 冷静、精密、高对比度、无装饰。
*   **色彩**: 
    *   背景: 纯白 (#FFFFFF) or 极浅灰 (#F5F5F5)
    *   文字: 墨黑 (#111111)
    *   点缀色 (用于花色区分): 克莱因蓝 (共振), 砖红 (异化), 森林绿 (共谋), 梦幻紫 (祈梦)。
*   **字体**: 无衬线字体 (Helvetica, Inter, 或类似风格)，强调粗细对比。

## 4. 内容数据
直接复用 `books/citywalk/福清石竹山.md` 中的 20 个任务和花色分类。

## 5. 技术建议
*   **框架**: 纯 HTML/CSS/JS 或轻量级框架 (Vue/React)。如果追求极致轻量，原生 JS 即可。
*   **动效**: 轮盘旋转的物理惯性、卡片弹出的流畅度是体验关键。
*   **适配**: 必须完美适配手机竖屏浏览器。
