# ✍️ 小学生汉字仿影练字板

使用 Hanzi Writer 库实现的汉字练习工具，支持笔画演示和描红练习。

## 🎯 功能特性

- 📖 **笔画演示** - 动画展示汉字书写顺序
- ✏️ **描红练习** - 跟着笔画轮廓练习书写
- 🔄 **自定义汉字** - 输入任意汉字进行练习

## 🚀 快速开始

```bash
cd handwriting-practice
python3 -m http.server 8080
```

打开浏览器访问：http://localhost:8080

## 📱 使用说明

1. **输入汉字** - 在输入框输入想要练习的汉字
2. **点击生成** - 点击「生成练字卡」按钮
3. **观看演示** - 点击「演示笔画」查看书写顺序
4. **开始描红** - 点击「开始描红」跟着练习

## ⚙️ 项目结构

```
handwriting-practice/
├── index.html  # 主页面
├── style.css   # 样式
└── script.js   # 主逻辑（Hanzi Writer 配置）
```

## 📱 设备要求

- 现代浏览器（Chrome / Safari / Edge）
- 网络连接（加载 Hanzi Writer 库）
- 触屏设备效果更佳
