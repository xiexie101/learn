# 福清行 (Fuqing Walk)

> 把城市当成迷宫去探索。
> An urban exploration game based on Situationist International.

## 🗺️ 游戏介绍
这是一个为两个人设计的城市漫步指南。在这个数字化早已接管生活的时代，我们试图通过“异化”的视角，重新夺回对物理空间的感知权。

游戏不依赖于打卡景点，而是关注**此时此刻**的感官体验与人际连接。

## 🎮 玩法规则

1.  **开始游戏**：在首页点击 **“开始探索”**。
2.  **转盘抽卡**：
    *   应用会随机为你抽取一张任务卡。
    *   颜色代表不同的心理维度：
        *   🔵 **共振 (Attunement)**：同步两人的注意力和节奏。
        *   🔴 **异化 (Detournement)**：打破常规，行为艺术。
        *   🟢 **共谋 (Conspiracy)**：建立只属于你们的秘密结界。
        *   🟣 **祈梦 (Dream)**：深层对话与潜意识探索。
3.  **执行任务**：
    *   按照卡片指令行动。
    *   没有输赢，只有体验。
4.  **否决权**：
    *   如果任务太尴尬或无法执行，可以使用 **“否决权”**。
    *   代价是必须诚实回答一个 **“真心话”** 问题。

## 📚 任务图鉴
点击首页的 **“任务图鉴”**，可以阅览所有的任务卡片。就像收集塔罗牌一样，去发现每一个角落的惊喜。

## 💡 设计灵感
灵感来源于情境主义国际（Situationist International）的漂移理论（Dérive）。我们鼓励在熟悉的城市中迷失，寻找心理地理学的浪漫。
