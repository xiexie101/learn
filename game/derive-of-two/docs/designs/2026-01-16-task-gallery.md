# 设计方案：任务图鉴 (Task Gallery)

**日期**: 2026-01-16
**作者**: Antigravity

## 1. 概述
在现有“福清行”应用中增加一个“任务图鉴”页面，允许用户主动浏览和查询所有任务卡片。

## 2. 核心功能
*   **浏览模式**：以卡片网格形式展示所有 20 个任务。
*   **筛选功能**：用户可以通过点击顶部的四个花色图标（黑桃/红心/梅花/方块）来筛选特定类别的任务。
*   **翻牌体验**：卡片默认显示背面（仅显示编号），点击后翻转显示正面（标题与详情），模拟“抽卡”或“翻开命运”的仪式感。

## 3. 界面设计

### 3.1 入口
*   **位置**：首页 (`#landing`) 底部或“开始探索”按钮下方。
*   **样式**：次级按钮 "任务图鉴" (View Gallery)。

### 3.2 图鉴页面 (`#gallery-screen`)
*   **布局**：
    *   **顶部**：返回按钮 (Back) + 筛选栏 (Filter Bar)。
    *   **主体**：滚动视图，包含任务卡片网格。
*   **筛选栏**：
    *   四个圆形按钮，分别包含四个花色的 SVG 图标。
    *   **选中状态**：高亮显示，且对应颜色的卡片保持完全不透明。
    *   **非选中状态**：按钮变灰，对应颜色的卡片在网格中隐藏或半透明（建议直接隐藏以减少杂乱）。

### 3.3 卡片样式
*   **尺寸**：宽高比约为 2:3（类似扑克牌或塔罗牌）。
*   **背面 (Back)**：
    *   **背景色**：使用任务对应的 `color` (深蓝/火红/森林绿/紫)。
    *   **纹理**：右下角叠加半透明的大号花色图标（如 20% 透明度的黑桃）。
    *   **内容**：正中央显示白色大号文字 `#01`。
*   **正面 (Front)**：
    *   **背景色**：白色 (#FFFFFF)。
    *   **装饰**：顶部有一条细的颜色条（对应分类颜色）。
    *   **内容**：
        *   `Category` (小字，右上角)
        *   `Title` (大字，居中或靠上)
        *   `Description` (正文)

### 3.4 交互动效
*   **翻转**：CSS 3D Transform (`rotateY(180deg)`). 点击卡片触发。
*   **入场**：进入页面时，卡片依次浮现 (Staggered fade-in)。

## 4. 技术实现要点
*   复用 `index.html` 中的 SVG Symbols。
*   在 `style.css` 中增加 `.gallery-grid`, `.gallery-card`, `.card-face` 等样式。
*   在 `script.js` 中增加 `renderGallery()` 函数，动态生成 DOM 节点。
