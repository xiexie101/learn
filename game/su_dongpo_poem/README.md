# 🌊 庐山烟雨浙江潮 - 苏东坡诗意可视化

将苏东坡名诗《观潮》通过水墨动画呈现，体验诗中的意境变化。

## 📜 诗文内容

```
庐山烟雨浙江潮，未至千般恨不消。
到得还来别无事，庐山烟雨浙江潮。
```

## 🎯 功能特性

- 🎨 **水墨画效果** - Canvas 渲染的动态水墨背景
- ✨ **诗句渐现** - 点击交互逐步展示诗句
- 🔄 **重游体验** - 重置后可再次体验

## 🚀 快速开始

```bash
cd su_dongpo_poem
python3 -m http.server 8080
```

打开浏览器访问：http://localhost:8080

## 📱 使用说明

1. **观看前两句** - 页面加载后展示前两句诗
2. **点击感悟** - 点击屏幕继续展示后两句
3. **体会意境** - 首尾呼应的禅意："看山还是山"
4. **点击重游** - 重新体验
