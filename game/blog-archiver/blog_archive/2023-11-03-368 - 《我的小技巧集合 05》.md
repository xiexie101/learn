# 368 - 《我的小技巧集合 05》

> 每凑够 10 条发一篇。

*   如果 Node 没有退出，可以用 why-is-node-running 或 wtfnode 查看原因。
*   `echo $?` 可以查看上一条命令的 exit code。
*   升级 iOS 到 17。试试 iOS 17 系列语音输入法（官方名称叫「听写」）。我已经把它用起来了，感觉有几个可以用的方向，一是用来写日根，二是可以快速记录一些事情，比如日记，三是可以用来练习英语。这个功能在 Mac 上也有，开启在「Keyboard - Dictation」里，可以快捷键开启，我是用「双击 Control」 开启，但是效果很一般，应该不是同一套东西。
*   语雀文档加 `/markdown?attachment=true&latexcode=false&anchor=false&linebreak=false` 可以以 Markdown 的格式下载文档。
*   VSCode 配置里可以开「terminal.integrated.copyOnSelection」，让 Terminal 里选择既复制，和 iTerm 保持体验一致。
*   利用 ffmpeg 可以一键把视频转成 gif，ffmpeg -i remix-parallel-loaders-demo.webm -s 898x412 -vf “fps=10” test-5.gif，需要注意的是，微信公众号对 gif 的 fps 有要求，别大于 20，我先是搞了个 30，发现不让传，于是又全部重新转成 10 fps 的了。
*   mac apple id 被锁可以用「defaults delete MobileMeAccounts && rm -rf ~/Library/Preferences/MobileMeAccounts.plist && killall accountsd」退。（未验证）。
*   Copilot Chat 小技巧。借助 fix 功能，实现 TypeScript to JavaScript，以及自动添加 console.log。
*   vercel 站点如果被墙，这么配。A 记录地址：76.223.126.88 ，CNAME 记录地址：[cname-china.vercel-dns.com](http://cname-china.vercel-dns.com) 。
*   ChatGPT 如果持续遇到「network error」问题，不用怀疑，是机场问题，换一个就好了。
