# 177 - 《最近新收获的工具、技巧和经验 07》

1、正式启用 Obsidian 管理我的所有 Markdown 文档，包括日更、MDH 周刊、任务管理、Daily Notes 等。感觉这就是我想要的笔记软件，相见恨晚！（每次都这么说…）基础模板和插件基于 [https://t.zsxq.com/05Nz76EmA](https://t.zsxq.com/05Nz76EmA) 懒人包；文件夹分类基于 P.A.R.A 分类法；备份没有用官方 Sync，而是定时提交到 Github 私库；Tag 管理目前还比较随意。没有刻意学如何使用 Obsidian，感觉已上手，更多高级功能还在熟悉中。

![](https://img.alicdn.com/imgextra/i3/O1CN01GTdUUG1IGfuuAkQxh_!!6000000000866-0-tps-1979-1080.jpg)

2、我家楼下的小区门卡是 IC 卡，每次带一个卡片有点麻烦，之前用小米手环是支持 NFC 写入的，换到 Apple Watch 之后发现 Apple Watch 不支持，几次被卡在门外。。于是买了「RFID迷你复卡机」和「PM3」来复制卡（实际只用了前者），写入到「很薄的卡贴」上，然后把卡贴贴在手机壳上使用。

3、新安装了「[Noptcha: ReCAPTCHA & hCAPTCHA Solver](https://chrome.google.com/webstore/detail/noptcha-recaptcha-hcaptch/dknlfmjaanfblgfdfebhijalfmhmjjjo/related)」Chrome 插件， ，用机器人打败机器人

4、读完三本书，《纳瓦尔宝典》、《教授与年轻人》和《前端架构师》。

5、新学到一个小技巧，给 Github Clone 加速，参考 [https://gist.github.com/laispace/666dd7b27e9116faece6?permalink\_comment\_id=3686810#gistcomment-3686810](https://gist.github.com/laispace/666dd7b27e9116faece6?permalink_comment_id=3686810#gistcomment-3686810)
