# 384 - 《用 GPT-4 写前端文章》

尝试了下，做下记录。5 分钟即可完成一篇。感觉如果没人和我说这是 AI 写的，可能不会轻易分辩出来。有兴趣的可以往社区上发发试试。

步骤如下。

0、确定主题，本文以新出的原子化 CSS in JS 方案 StyleX 为例。

1、安装 gpt-crawler 。

```bash
git clone git@github.com:BuilderIO/gpt-crawler.git && cd gpt-crawler && npm i
```

2、修改 config.ts，配置如下。

```ts
import { Config } from "./src/config";
export const defaultConfig: Config = {
  url: "https://stylexjs.com/",
  match: "https://stylexjs.com/docs/**",
  maxPagesToCrawl: 50,
  outputFileName: "output.json",
};
```

3、执行 npm start，让他去爬 stylex 官网，你会拿到一个 output-1.json 。

![](https://img.alicdn.com/imgextra/i4/O1CN01llnpVv1vulaOkSPgt_!!6000000006233-2-tps-2834-1788.png)

4、在 [https://chat.openai.com/](https://chat.openai.com/) 里点 Explorer，然后创建 GPTs。把前面的 output-1.json 传上去，然后在右边写 PROMPT 让他输出文章。我的 PROMPT 如下。（注：如果要以 API 的方式访问，可以创建 assistant [https://platform.openai.com/assistants](https://platform.openai.com/assistants) ）

```
你是一名专业的前端程序员，同时拥有非常好的写作能力。
现在需要写一篇以「介绍 StyleX」为题的文章。
写作方式是，
1、先提出一些问题
2、然后对此做解答
3、以「1、2、3、...」的形式输出文章，总共不超过 10 点，每一点的内容在 100 字到 500 字之间
4、文中如果需要，可以包含一些代码例子
5、可以穿插一些你的感受，加入一些以「我」开头的句子
6、段落可以有长有短
注意：输出的内容请在中文字符和其他字符之间添加空格。
```

![](https://img.alicdn.com/imgextra/i4/O1CN01da9bWo1EP8l6ZrNpT_!!6000000000343-2-tps-3822-1748.png)
