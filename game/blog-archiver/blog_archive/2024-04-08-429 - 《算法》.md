4、个人刷 LeetCode 题的步骤如下。

0）用 TypeScript 解，本地 Copilot 生成用例，验证后再提交  
1）先尝试自己解  
2）解不出的用 Github Copilot 自动完整一版，看是否能理解  
3）不能理解的话有两个办法，a）让 ChatGPT 给解释一遍，b）看官方题解（文字版），我更倾向前者，因为，a）官方题解有时不会给 JavaScript 版解，b）官方题解是先将原理，再给方法，没有 ChatGPT 直接对着代码将原理直观  
4）关闭 VSCode 的 Copilot 自动完成，自己写一遍  
5）拿出本子对着「鸭子」讲解一遍  
6）隔几周复习一遍
