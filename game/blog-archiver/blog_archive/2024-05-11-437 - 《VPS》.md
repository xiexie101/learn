# 437 - 《VPS》

> VPS 新手，欢迎补充。

1、两个论坛，[大佬论坛](https://www.dalao.net/) 和 [NodeSeek 论坛](https://www.nodeseek.com/)，前者更偏域名交易，我逛后者比较多。NodeSeek 主要是各种 VPS 资讯和交易，此外还有些奇怪的资讯，比如域名、Telegram 账号、平台和软件合租、edu 账号、国外电话卡、学生权益、等等。我这周还在这上面上了个 Netflix 合租的车，尼区，5 人车，一个季度 30 块钱。

2、几台我在用的主机是：

1）[CloudCone](https://app.cloudcone.com/?ref=10409)，US、2C2G，年付 $29，[sorrycc.com 博客](https://sorrycc.com/)在用，国内访问感觉速度偏慢，ping 值在 100+ 到 200+，准备把服务迁到 VKVM。

2）[搬瓦工](https://bandwagonhost.com/aff.php?aff=74975)，JP、1C2G，年付 $74.57，[mako 官网](https://makojs.dev/)部署在这，国内访问感觉较快，ping 值在 100 以内。

3）[VKVM](https://www.vkvm.info/aff/NPUGEOQM)，HK、2C2G，年付 ¥210。国内访问感觉最快，ping 值在 50 上下。（不推荐！性能不如搬瓦工的 1C2G ）

4）阿里云机器，之前年付 ￥99 买的，空着。

3、机器到手后的几件事。

1）配 ssh 密码免登，一次性操作，方便后续管理机器。（注：遇到过 ssh 登不上的问题，应该是和我开了 Surge 的 Enhanced Mode 有关，让 ssh 走 Direct 就好了）

```bash
ssh-keygen
ssh-copy-id -i ~/.ssh/xxx -p xxxx root@x.x.x.x
edit ~/.ssh/config
ssh xxx
```

2）装 [1Panel 面板](https://1panel.cn/)。就用过 1Panel 和宝塔面板，对比用过之后，感觉 1Panel 更好用一些。

![](https://res.cloudinary.com/sorrycc/image/upload/v1715411035/blog/0amldfrx.png)

附：1Panel 如何安装 WordPress？a）安装 WordPress 应用，b）新建反向代理网站，配置到 WordPress 应用端口（如果 504，反向代理时别用 127.0.0.1，改用真实 IP 试试）。

3）装 [哪吒探针](https://nezha.wiki/)。我的探针地址是 [https://tz.sorrycc.com/](https://tz.sorrycc.com/) 。

![](https://res.cloudinary.com/sorrycc/image/upload/v1715410050/blog/7xzvz502.png)

4、常见操作。

如何配置 Node 服务。1）scp 传文件到 `/opt/1panel/apps/openresty/openresty/www/sites/` 下找个目录，但别用域名，域名目录给网站用，2）新建运行环境，3）新建网站，4）新建证书，5）编辑网站，启用证书。
