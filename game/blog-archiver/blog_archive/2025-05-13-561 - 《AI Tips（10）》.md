# 561 - 《AI Tips（10）》

> 上一篇 [560 - 《AI Tips（9）》](https://sorrycc.com/ai-tips-09)。

1、Zed [支持 Coding Agent](https://zed.dev/agentic)。免费版每个月 50 prompts 和 2000 edit prediction。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/xZTJvt.png)

2、用 [VikParuchuri/marker](https://github.com/VikParuchuri/marker) 把 PDF 转为 Markdown 或 Html 。

```bash
uv init
uv add marker-pdf
source .venv/bin/activate
marker_single /path/to/pdf --output_format html
```

3、Figma 在 Config 大会上推出多项新产品，1）网站创建与托管平台 Figma Sites，2）矢量绘图工具 Figma Draw，3）社交素材生成工具 Figma Buzz，4）将设计自动转化为代码 Figma Make。上述产品均集成AI聊天功能，并开始提供“文本生成应用”能力。

4、记住这条 o3 咒语：「fact-check this!」用于「核查事实」。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/okXxSC.png)

5、[MODSetter/SurfSense](https://github.com/MODSetter/SurfSense)，开源的 NotebookLM / Perplexity / Glean 平替，连接如搜索引擎、Slack、Notion、YouTube 和 GitHub 等。

6、[infiolab/infio-copilot](https://github.com/infiolab/infio-copilot)。Obsidian AI 插件。提供交互式对话、内联编辑、智能补全、全库检索问答等强大功能。相见恨晚。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/640.gif)

7、[ryoppippi/sitemcp](https://github.com/ryoppippi/sitemcp)。一款将目标网站页面转化为 MCP 服务器的工具，方便 AI 获取信息。其核心价值在于将未提供 MCP 服务器的网站内容接入 AI 知识库，降低 token 消耗，提升效率。支持小众库最新文档更新及个性化应用，如将个人网站内容转化为知识库，助力 AI 生成履历或模仿文风。

以 DaisyUI 网站为例。

```json
{
  "mcpServers": {
    "daisy-ui": {
      "command": "npx",
      "args": [
        "-y",
        "sitemcp@latest",
        "https://daisyui.com",
        "-m",
        "/components/**"
      ]
    }
  }
}

```

8、[voideditor/void](https://github.com/voideditor/void) 。一款开源的 Cursor 替代工具，支持在代码库中使用 AI Agent、Checkpoint 和可视化更改，并可引入任何模型。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/MJpZ0l.png)
