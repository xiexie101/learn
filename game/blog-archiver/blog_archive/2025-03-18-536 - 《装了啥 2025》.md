# 536 - 《装了啥 2025》

> 上一次更新这个主题 [375 - 《装了啥 2023.11》](https://sorrycc.com/zhuang-2023-11) 已经是一年多了，最近感觉更新了很多 AI 相关的工具，抽空更新一版。

## AI

> 新增板块。

*   在线的 AI 服务按使用频次，依次是 [Google AI Studio](https://aistudio.google.com/prompts/new_chat?model=gemini-2.0-flash-thinking-exp-01-21)、[GROK](https://grok.com/)、[Claude](https://claude.ai/)、[ChatGPT](http://chat.openai.com/)、[Perplexity](https://www.perplexity.ai/) 和 [DeepSeek](https://chat.deepseek.com/) 。
*   本地客户端装了平台提供的 [ChatGPT](https://openai.com/chatgpt/desktop/) 和 [Claude](https://claude.ai/download)。API 调用方式的客户端主要用 [ChatWise](https://chatwise.app?atp=sorrycc)，备用 [Cherry Studio](https://github.com/CherryHQ/cherry-studio) 。
*   API 侧，除了各家官方的 API 外，还会有 Doubao、Aliyun、Sillicon Flow 提供的 DeepSeek（注：只有 Doubao 的 DeepSeek R1 支持 Function Call），主打快的 [Groq](https://groq.com/)，模型全的 [Open Router](https://openrouter.ai/) 等。
*   编辑器主要用 [Cursor](https://www.cursor.com/)，备用 VSCode Insider + [Github Copilot](https://github.com/features/copilot)、[WindSurf](https://codeium.com/windsurf) 和 [Cline](https://github.com/cline/cline)。准备试试命令行类型的 [Claude Code](https://github.com/anthropics/claude-code) 和 [Aider](https://github.com/Aider-AI/aider) 。
*   语音输入主要用 [Wispr Flow](wisprflow.ai/r/%E9%99%88%E6%88%902)，备用 [Super Whisper](https://superwhisper.com/) 。
*   Codebase Pack 用 [Repomix](https://repomix.com/) 。
*   跑本地模型用 [Ollama](https://ollama.com/) 。

更多信息，见 [530 - 《我怎么用 AI 辅助编程》](https://sorrycc.com/how-to-use-ai-to-assist-programming) 。

## 科学上网

> 和之前的没啥变化。

*   机场在用的有两个。主机场是 [YTOO](https://y-too.com/aff.php?aff=3277)，￥372/Y，用了一年多；备机场是 [IPLC.VIP](https://thirdislandchain.com/aff.php?aff=1717)，¥488/Y，用了三年多。
*   软件方面，Mac、iPhone 和 Apple TV 上用 [Surge](https://nssurge.com/)，Windows 上用 [Clash for Windows](https://github.com/Fndroid/clash_for_windows_pkg) 。

更多信息，见 [474 - 《科学上网》](https://sorrycc.com/pfd-01-how-to-gfw) 和 [529 - 《教普通人手机科学上网和 AI》](https://sorrycc.com/how-to-use-vpn-and-ai)。

## 编程工具

> 2025.3 版变更：字体新增 Sarasa Mono。

*   编程字体用 Monolisa、[Dank Mono](https://dank.sh/)、[Operator Mono](https://www.typography.com/fonts/operator/overview)、[Monospace](https://github.com/githubnext/monaspace) 和 [Sarasa Mono](https://picaq.github.io/sarasa/)，一个用久了看腻了切另一个。
*   Terminal 工具是 [iTerm2](https://iterm2.com/) + zsh + [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) + [starship](https://starship.rs/) 的组合，zsh 额外装了 [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)、[zsh-completions](https://github.com/zsh-users/zsh-completions) 和 [fast-syntax-highlighting](https://github.com/zdharma/fast-syntax-highlighting) 插件。
*   其他辅助工具包括，API 工具 [RapidAPI](https://paw.cloud/)、Hosts 管理工具 [Gas Mask](https://github.com/2ndalpha/gasmask)、GUI Git 工具 [SourceTree](https://www.sourcetreeapp.com/)、辅助工具集 [DevUtils](https://devutils.com/)、取⾊⼯具 [ColorSnapper2](https://colorsnapper.com/)、Docker 客户端 [OrbStack](https://orbstack.dev/) 。

## 知识管理

> 2025.3 版变更：新增 Screen Studio，删除 LICEcap 。

*   信息输入部分，用 [Reeder](https://reederapp.com/) 阅读 RSS 源的信息，用 [ReadWise Reader](https://read.readwise.io/) 读电子书和一些 Youtube 视频，用 [DeepL](https://www.deepl.com/) + [Bob](https://bobtranslate.com/) + [Immersive Translation](https://immersivetranslate.com/) 阅读非中文的文章、书、视频和播客，适合浅阅读的文章会用 AI 总结阅读，干货长播客会用[飞书妙记](https://www.feishu.cn/product/minutes)把音频转文本后慢慢消化。
*   信息输出部分，主 [Obsidian](https://obsidian.md/)，辅 [Flomo](https://flomoapp.com/)、[Tot](https://tot.rocks/) 和 Apple Notes，Obsidian 保存了我 90% 的内容，Flomo 保存非技术类的文章阅读笔记，Tot 保存临时想法，Apple Notes 保存生活类的，优点是可以设置文件夹和家人共享。
*   信息输出辅助工具包括，用于临时编辑单个的 Markdown 文件的 [IA Writer](https://ia.net/writer)、一键上传图片的 [upic](https://blog.svend.cc/upic/) 传到 Github 仓库然后通过 [cdn.jsdelivr.net](http://cdn.jsdelivr.net) 访问、视频和 GIF 录制用 [Screen Studio](https://screen.studio/)、截屏编辑工具 [Shottr](https://shottr.cc/) 和 [CleanShot X](https://cleanshot.com/)、用 Markdown 做 PPT 的 [iA Presenter](https://ia.net/presenter) 。
*   任务管理直接 Obsidian 进行管理，做年、周、日计划，做时间记录，参考 [237 - 《2023 年应该如何做事》](https://sorrycc.com/how-things-should-be-done-in-2023) 。

详见 [515 - 《我的信息流 2025.1》](https://sorrycc.com/information-flow-2025) 。

## 系统利器

> 2025.3 版变更：新增 PDFGear、Bitwarden，准备废弃 SetApp，废弃 1Password、HandBrake、Focus。

*   [SetApp](https://setapp.com/)，一个 App 平台。不过他应该是最后一次出现在这个列表里了，因为感觉并不划算，等今年开的车到期后，就不续了，相关 App 基本上都有开源或其他的代替方案。
*   [Bitwarden](https://bitwarden.com/)，密码管理，服务器用自部署，代替 1Password，挺好用。
*   [Alfred](https://www.alfredapp.com/) + Powerpack，用于应用启动和全局搜索，比如搜 Chrome 收藏夹和文件等，但感觉这工具没跟上 AI 时代，今年会尝试下 [Raycast](https://www.raycast.com/) 。
*   [Bartender](https://www.macbartender.com/)，菜单图标隐藏工具。
*   [Karabiner Element](https://karabiner-elements.pqrs.org/)，用于键盘自定义工具，我只用来做了一件事，修改右 Command 为 Hyper 键，为快捷键的定制增加了更多选择，比如 Hyper+D 用于切换 iTerm2 的显示隐藏。
*   [ZeroTier](https://www.zerotier.com/)，让所有的电脑、手机和 Nas 组成局域网。
*   [Thor Launcher](https://apps.apple.com/cn/app/thor/id1120999687?mt=12) 用于通过快捷键切换应用，切换包含的功能有启动、显示和隐藏。
*   [Eagle](https://eagle.cool/)，用于管理素材，我只用于保存网络图片。
*   [Espanso](https://espanso.org/)，用于替换文本，比如我想输入我的邮箱只要 ;mail 即可。
*   [Paste](https://pasteapp.io/) 用于粘贴板管理，SetApp 到期后考虑用 [PastePal](https://indiegoodies.com/pastepal) 或 Raycast 的粘贴板功能代替。
*   [KeepingYouAwake](https://keepingyouawake.app/)，禁用系统自动休眠。
*   [PDFGear](https://www.pdfgear.com/)，免费的 PDF 工具，用于代替付费的 [PDF Expert](https://pdfexpert.com/) 。
*   [OBS](https://obs.camera/) + [KeyCastr](https://github.com/keycastr/keycastr) + [剪映](https://www.capcut.cn/) + [RDM](https://github.com/avibrazil/RDM) + [Screen Studio](https://www.screen.studio/) + …，用于直播和视频录制。
*   [IINA](https://iina.io/)，用于视频播放。
*   [f.lux](https://justgetflux.com/)，调节⾊温的护眼⼯具。
*   [Battery Buddy](https://batterybuddy.app/)，给菜单栏的电池换个有趣的图标。
*   [Rectangle](https://rectangleapp.com/)，用于窗⼝管理，最常用的操作是缩放到全屏。
*   Google Chrome + [Microsoft Edge](https://www.microsoft.com/en-us/edge) + Google Chrome Canary，三个浏览器三个账号体系，方便随时切换，有些服务对使用有限制，多几个账号有备无患。
*   [Screens](https://edovia.com/en/screens-mac/) + [Microsoft Remote Desktop](https://apps.apple.com/us/app/microsoft-remote-desktop/id1295203466?mt=12)，分别用于远程连接 Mac 和 Windows 电脑。
*   [Downie](https://software.charliemonroe.net/downie/) + [yt-dlp](https://github.com/yt-dlp/yt-dlp)，视频下载⼯具，后者是命令行、开源且能力更强。

## 硬件

> 2025.3 版变更：新增 HHKB Studio x2、RODE VideoMic Pro+ Plus、富图宝空气三号三脚架，废弃 Apple Watch Series 8。

*   手机是 iPhone 15 Pro Max 512G。
*   主电脑是 MacBook Pro（16-inch M2 Pro Max，96G），当时顶配买的，用了几年，还能战。
*   家里备了一台 Windows 电脑和一个 Mac Mini M2，常开着，通过 ZeroTier 可以随时连回去。Windows 主要用于游戏，配置是 RTX 3070 TI + Intel i7 12700K + 华硕 TUF GAMING Z690-PLUS D5 + 威刚XPG 16Gx2 + 三星980 PRO 2T。
*   拓展坞用 [Caldigit TS4](https://www.caldigit.com/thunderbolt-station-4/)，接口多，稳定，速度快。
*   键盘鼠标分别是 [HHKB Studio](https://hhkeyboard.us/hhkb-studio/Product) 和 [Razer DeathAdder Chroma](http://www.razerzone.com/store/razer-deathadder-chroma)，x2，公司和家里用一套，保持体验一致。HHKB 系列的用了 10 多年了，每次迭代都会跟。
*   相机是 Sony A7M4 + 24-70 GM 镜头 + Elgato Cam Link 采集卡 + Blue Yeti 麦克风 + RODE VideoMic Pro+ Plus + 富图宝空气三号三脚架，直播和视频录制用。
*   NAS 是 DS920+ + 希捷企业级16T\*4 。
