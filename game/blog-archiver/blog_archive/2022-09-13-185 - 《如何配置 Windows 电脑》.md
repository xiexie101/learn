# 185 - 《如何配置 Windows 电脑》

2022.9.1 配了一台 Windows 电脑后，就开始断断续续地做 Windows 的电脑配置。考虑某些游戏的兼容性，系统用的 Win10。基本上有 10 年没用过 Windows 了，感觉很陌生。上个周末花时间研究了下，算是慢慢找到感觉了。

1、确保上网无忧，安装 Google Chrome 和 [Clash For Windows](https://github.com/Fndroid/clash_for_windows_pkg)。

2、安装 [winget](https://github.com/microsoft/winget-cli)，类似 Mac 系统的 brew，然后就可以用他来安装各种软件了，可以拿着关键词在 [winget.run](https://winget.run/) 上搜。

3、我用 winget 装了以下这些软件，遇到有些下载慢的切全局代理可解。

```bash
winget install -e --id Daum.PotPlayer
winget install -e --id Tonec.InternetDownloadManager
winget install -e --id Baidu.BaiduNetdisk
winget install -e --id Sogou.SogouInput
winget install -e --id Valve.Steam
winget install -e --id RandyRants.SharpKeys
winget install -e --id flux.flux
winget install -e --id Obsidian.Obsidian
winget install -e --id Alibaba.aDrive

# Dev 相关
winget install -e --id Microsoft.WindowsTerminal
winget install -e --id Git.Git
winget install -e --id Microsoft.VisualStudioCode
winget install -e --id CoreyButler.NVMForWindows
winget install -e --id Github.GithubDesktop
winget install -e --id Obsidian.Obsidian
winget install -e --id Atlassian.Sourcetree
```

4、Git 的配置。1）安装 Git 之后记得添加 Git 命令路径 `C:\Program Files\Git\cmd` 到 PATH 环境变量中，Git 安装器不会自动添加；2）配 Git SSH KEY，并添加到 Github。

```bash
ssh-keygen -t rsa -b 4096 -C "sorrycc@gmail.com"
cat ~/.ssh/id_ed25519.pub
```

5、Node 用 NVMForWindows 安装，而没有用官方安装包，因为不想污染全局，同时之后升级方便。

6、pnpm 推荐用 winget 安装，可以制定版本。

```bash
winget install -e --id pnpm.pnpm -v 8.11.0
```

7、介绍下 [SharpKeys](https://github.com/randyrants/sharpkeys)，刚搜到的软件。由于我用 HHKB 键盘，从 Mac 切换到 Windows 后 Ctrl 位置变成了之前 Capslock 键的位置，非常不习惯，先是找了 PowerToys 但是不够用，后来找到基于注册表实现的 SharpKeys，可以完美解决问题，我把 Command 键改成了 Ctrl 键。

8、在 Windows 上也安装了 Obsidian。为了更好的同步体验，花 600+/Year 开了 Obsidian Sync 服务，在 win、mac 和 ios 上同步，同步效果很不错，感觉值这个价。

9、配置 Rust 环境。

1）安装 Rustup。

```bash
winget install -e --id Rustlang.Rustup
```

2）Microsoft Store 上安装「Visual Studio Installer」

3）Visual Studio Installer 里安装 Visual Studio，记得选上「Desktop development with C++」，不然你在用 rust 编译时会缺库
