# 541 - 《AI Tips（5）》

> Whisper 闪念笔记、Flora、Gemini 2 支持图片、AI 视频制作、VSCode LM API 避坑、SVG AI 绘图等。

1、强烈推荐一下这个 **[Whisper 闪念笔记和键盘](https://onenewbite.gumroad.com/l/whisper)**，它解决了我在手机上输入效率的问题。我可以随时随地有想法的时候，就用手机快速地输入。之前我在用 iPhone 自带的语音输入，本地不联网的方案，不太准确。输完之后，总是需要去调整，有时候反而比自己输入要慢。这个方案基于 OpenAI 的 Whisper，准确性很高，同时可以用 Action Button 快速唤起，用起来也很顺手。（注：Mac下我也在用这一套方案，给 Shortcuts 绑定一个快捷键就能快速启动了。)

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/lDDWp9.jpg)

2、**[Flora](https://www.florafauna.ai/)** 这个网站的操作流还挺有意思的。它基于画布。你可以在画布上创建文本、图片、视频的节点。然后，通过画线的方式去创建下一个节点。下一个节点输入一些 prompt。然后，它会基于 prompt 去渲染新的节点。我喜欢这种交互方式。之前我也有想过自己去实现类似的，无奈还是行动力有限。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/0qLCD2.png)

3、**[tldr.tube](https://tldw.tube/)**，一个 YouTube 的总结服务，[开源](https://github.com/stong/tldw)，偶尔会用一下，但是用处并不大。

4、**Gemini 2 Flash Exp（Image Gen）** 值得一玩。有些场景已经可以代替 PhotoShop 了。一个场景是，你给他一张图片，然后让他按照这个风格生成你想要的东西。比如给一张照片，然后让 AI 给她换一身衣服。比如生成各种“梗图”。可以在 AI Studio 上使用，也可以生成一个 token 然后在 ChatWise 或其他的 chat app 上使用。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/1cTMlo.png)

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/2P0UjZ.png)

5、**一个 AI 视频制作流程**。之前没有接触过 AI 的视频制作，感觉还挺好玩的。思路是先用 Midjourney 生成图片，然后再用 Kling 1.5 或者 1.6 去生成视频，最后用剪映去把这些视频做合成、剪辑、音效等。Kling 的使用有几种方式：1）在 Poe 上充值，2）在 [Fal.ai](https://fal.ai/) 上用 API 的方式来调，3）用 [可灵 AI](https://klingai.kuaishou.com/)。

6、**[r.jina.ai](https://r.jina.ai/)** 可以为网站生成 AI 友好的格式，通常是 Markdown，[开源](https://github.com/jina-ai/reader)。这个工具在我的流程里面用得挺多。比如总结的任务，我会先用它去生成 Markdown 文本，再让大模型去总结。

7、**VSCode LM API 避坑**。这是我自己踩过的一个坑。VS Code 的 Copilot 服务，有提供 API 授权的隐藏接口，也就是说，你可以把 Copilot API 拿出来自己用，比如说 Cloud 3.7 Sonnet 那种。这个流程是能跑通的，但是会导致功能被禁或者封号。我试了两种方式，一种是开学生包，另外一种是绑卡开 $10/Month 的付费版本，都不行。最终 Copilot 功能都被禁用掉了。

8、**基于 SVG 格式的 AI 画图**。Claude 3.7 Sonnet 对于 SVG 的支持已经比较好了，所以有一种思路是让大模型用 SVG 去画图。先分析你文章的内容，然后用 SVG 格式去画图，用图的方式来展现你文章的思路，或者给你的文章配图。你可以让它用某种风格来画，比如说手绘的黑白漫画。我自己试过，效果还不错。

```
生成一幅手绘黑白漫画，用 SVG 的格式，可用于讲解「这篇文章的内容」。保存为同名的 .svg 文件。
```
