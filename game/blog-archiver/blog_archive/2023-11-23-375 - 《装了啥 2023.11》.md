# 375 - 《装了啥 2023.11》

> 看到 [https://defaults.rknight.me/](https://defaults.rknight.me/) 大家都在贴自己用的软件和硬件，手痒更新一版，感觉 1 年不到变化还是挺大的。

## 科学上网

> 2023.11 版变更：机场换了，软件从 ClashX 和 ShadowRocket 换成了 Surge。

*   机场是 [YTOO](https://y-too.com/aff.php?aff=3277)，￥372/Y，用了几个月，稳定。之前用的 [IPLC.VIP](https://thirdislandchain.com/aff.php?aff=1717)，￥488/Y，用了 3-4 年了，今年有段时间 ChatGPT 响应不太稳就换了
*   软件方面，Mac 和 iOS 上用 [Surge](https://nssurge.com/)，Windows 上用 [Clash for Windows](https://github.com/Fndroid/clash_for_windows_pkg)

## 编程工具

> 2023.11 版变更：主编辑器从 WebStorm 切换到 VSCode，废弃了 CotEditor、He3、DevToys、Docker，新增了 Github Copilot、Github Copilot Chat、Monospace 字体、zsh-completions、starship、OrbStack。

*   [ChatGPT](https://chat.openai.com/)、[Github Copilot](https://github.com/features/copilot) 和 [Github Copilot Chat](https://docs.github.com/en/copilot/github-copilot-chat)，提效神器，这个钱不能省
*   编辑器主 [VSCode](https://code.visualstudio.com/)，辅 [WebStorm](https://www.jetbrains.com/webstorm/)
*   WebStorm 插件只装了两个，[Github Copilot](https://github.com/features/copilot)、[Inspection Lens](https://plugins.jetbrains.com/plugin/19678-inspection-lens)，前者大幅提升编码效率，后者大幅提升编码体验
*   编程字体用 Monolisa、[Dank Mono](https://dank.sh/)、[Operator Mono](https://www.typography.com/fonts/operator/overview) 和 [Monospace](https://github.com/githubnext/monaspace)，一个用久了看腻了切另一个
*   Terminal 工具是 [iTerm2](https://iterm2.com/) + zsh + [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) + [starship](https://starship.rs/) 的组合，zsh 额外装了 [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)、[zsh-completions](https://github.com/zsh-users/zsh-completions) 和 [fast-syntax-highlighting](https://github.com/zdharma/fast-syntax-highlighting) 插件
*   其他辅助工具包括，API 工具 [RapidAPI](https://paw.cloud/)、Hosts 管理工具 [Gas Mask](https://github.com/2ndalpha/gasmask)、GUI Git 工具 [SourceTree](https://www.sourcetreeapp.com/)、辅助工具集 [DevUtils](https://devutils.com/)、取⾊⼯具 [ColorSnapper2](https://colorsnapper.com/)、Docker 客户端 [OrbStack](https://orbstack.dev/)

## 知识管理

> 2023.11 版变更：新增 Immersive Translation、飞书妙记。

*   信息输入部分，用 [Reeder](https://reederapp.com/) 阅读 RSS 源的信息，用 [ReadWise Reader](https://read.readwise.io/) 读电子书和一些 Youtube 视频，用 [DeepL](https://www.deepl.com/) + [Bob](https://bobtranslate.com/) + 自研中英对照脚本 + [Immersive Translation](https://immersivetranslate.com/) + [Whisper](https://github.com/openai/whisper) 阅读非中文的文章、书、视频和播客，干货长播客会用[飞书妙记](https://www.feishu.cn/product/minutes)把音频转文本后慢慢消化
*   信息输出部分，主 [Obsidian](https://obsidian.md/)，辅 [Flomo](https://flomoapp.com/)、[Tot](https://tot.rocks/) 和 Apple Notes，Obsidian 保存了我 90% 的内容，Flomo 保存非技术类的文章阅读笔记，Tot 保存临时想法，Apple Notes 保存生活类的，优点是可以设置文件夹和家人共享
*   信息输出辅助工具包括，用于临时编辑单个的 Markdown 文件的 [IA Writer](https://ia.net/writer)、一键上传图片的 [upic](https://blog.svend.cc/upic/)、GIF 录屏工具 [LICEcap](http://www.cockos.com/licecap/)、截屏编辑工具 [Shottr](https://shottr.cc/) 和 [CleanShot X](https://cleanshot.com/)、用 Markdown 做 PPT 的 [iA Presenter](https://ia.net/presenter)
*   任务管理直接 Obsidian 进行管理，做年、周、日计划，做时间记录，参考 [237 - 《2023 年应该如何做事》](https://sorrycc.com/how-things-should-be-done-in-2023)

## 系统利器

> 2023.11 版更新：废弃了 Parallels Desktop、Yoink、Shiftlt、RunCat、CleanMyMac X，新增 ZeroTier、Screen Studio、Microsoft Edge、Rectangle。  
> 注：按我的推荐度排列，靠前是非常推荐或者使用频率很高的。

*   [SetApp](https://setapp.com/)，包含大量优秀 App，相比单个买 App 能省很多钱，由于官方改了 Family 的配置，可能考虑弃坑
*   [Alfred](https://www.alfredapp.com/) + Powerpack，除了应用启动，我觉得最好用的是，1）搜索 Chrome 收藏夹，2）搜索文件，比如搜索 Markdown 文件，然后用 iA Writer 打开编辑
*   [Focus](https://heyfocus.com/)，强制自己保持专注
*   [Thor Launcher](https://apps.apple.com/cn/app/thor/id1120999687?mt=12) 用于通过快捷键切换应用，切换包含的功能有启动、显示和隐藏
*   [1Password](https://1password.com/) 用于密码管理，之前买的 Family 版，但是只有我一个人用，有点贵，准备换 Bitwarden
*   [Bartender](https://www.macbartender.com/)，菜单图标隐藏工具
*   [Karabiner Element](https://karabiner-elements.pqrs.org/)，键盘自定义工具，我只用来做了一件事，修改右 Command 为 Hyper 键，为快捷键的定制增加了更多选择，比如 Hyper+D 用于切换 iTerm2 的显示隐藏
*   [ZeroTier](https://www.zerotier.com/)，让所有的电脑、手机和 Nas 组成局域网
*   [Eagle](https://eagle.cool/)，素材管理工具，我只用于保存网络图片，去年新增了 200 多张
*   [Espanso](https://espanso.org/)，用于替换文本，比如我想输入我的邮箱只要 ;mail 即可
*   [Paste](https://pasteapp.io/) 用于粘贴板管理
*   [PDF Expert](https://pdfexpert.com/)，用于 PDF 标注、创建和管理的工具
*   [KeepingYouAwake](https://keepingyouawake.app/)，禁用系统自动休眠
*   [OBS](https://obs.camera/) + [KeyCastr](https://github.com/keycastr/keycastr) + [剪映](https://www.capcut.cn/) + [RDM](https://github.com/avibrazil/RDM) + [Screen Studio](https://www.screen.studio/) + …，用于直播和视频录制
*   [IINA](https://iina.io/)，用于视频播放
*   [CleanMyMac X](https://cleanmymac.com/)，系统清理工具，每次硬盘空间不足时就会想到他
*   [RunCat](https://apps.apple.com/cn/app/runcat/id1429033973?mt=12)，显示 CPU 状态的小猫，CPU 占用越高，他跑地越快
*   [f.lux](https://justgetflux.com/)，调节⾊温的护眼⼯具
*   [Battery Buddy](https://batterybuddy.app/)，给菜单栏的电池换个有趣的图标
*   [Rectangle](https://rectangleapp.com/)，用于窗⼝管理，最常用的操作是缩放到全屏
*   [Screens](https://edovia.com/en/screens-mac/) + [Microsoft Remote Desktop](https://apps.apple.com/us/app/microsoft-remote-desktop/id1295203466?mt=12)，分别用于远程连接 Mac 和 Windows 电脑
*   [HandBrake](https://handbrake.fr/)，用于视频格式转换
*   [Downie](https://software.charliemonroe.net/downie/) + [yt-dlp](https://github.com/yt-dlp/yt-dlp)，视频下载⼯具，后者是命令行、开源且能力更强
*   [Microsoft Edge](https://www.microsoft.com/en-us/edge)，偶尔用，这个浏览器好好配置下之后还是很简洁的
