# 573 - 《在用的 AI 工具》

> 整理下我在用的 AI 相关的工具。

通用助手：

*   [Claude](https://claude.ai/) 。免费用户，考虑升 Pro，主要是想用他的 Claude Code 额度，迟迟没动手因为担心被封。
*   [ChatGPT](https://chatgpt.com/) 。Plus + API 用户。Plus 很早开过，然后感觉没用就断了，然后出 4o 画图时续订，接着用习惯 o3 后就一直续着。不过近期 o3 API 降价 80%，可能考虑切 API 方案。
*   [Gemini](https://gemini.google.com/) 。Plus + GCP API 用户。Web 版主要用 Deep Research 和 NotebookLM 和 Gemini 2.5 Pro 。API 方式大量用 Gemini 2.5 的 Flash 和 Pro 。
*   [Grok](https://grok.com/) 。有点鸡肋，偶尔用用。
*   不用的有，1）[Perplexity](https://www.perplexity.ai/)，主打 Deep Research，但明显 Gemini 的更好用，2）国内的模型和厂商，因为感觉没啥优势，不过各家都开了点 API 额度，主要拿来做测试。

编程助手：

*   [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 。主力编程工具，一般不直接用官方的 API 方案，因为太贵。两个方案，1）升他的 Pro 或 Max 会员，会给一定的额度，2）用 [gemini-code](https://github.com/coffeegrind123/gemini-code) 或 [claude-bridge](https://github.com/badlogic/lemmy/tree/main/apps/claude-bridge) 代理 Gemini API， 然后用 Gemini Pro 。参考 [575 - 《Claude Code》](https://sorrycc.com/claude-code)
*   [Cursor](https://www.cursor.com/) 。年度会员，辅助编程工具，有 Claude Code 后比较少用。但每个月的额度只能用到 2/3，希望下个月能用完。
*   CodeX 有两个产品，[Cli](https://github.com/openai/codex) 和[平台](https://chatgpt.com/codex)。前者很少用，因为太轻量，主要拿来学习怎么写 Agent；后者偶尔心血来潮会对某些 github 的仓库跑个小任务。
*   不用的有，1）IDE 和插件类的 WindSurf、Cline、Augment Code、Github Copilot，2）平台类的 Bolt.new、V0.Dev、Lovable、Replit 等。因为精力有限。

API：

*   主力模型是 Gemini 2.5 的 Flash 和 Pro，因为 GCP 给的 $300 免费额度以及这两个模型确实给力，Flash 够快，Pro 虽然编码比 Claude 差一点但综合能力强。
*   OpenAI 付费用的 [WildCard](https://bewildcard.com/i/SORRYCC) 。
*   GCP 的开法参考 [564 - 《GCP $300 流程》](https://sorrycc.com/gcp-300)，不过似乎 WildCard 后来不支持了，注意验证。
*   除 Claude 之外，其他的提供商也都有开一些 API 额度。然后 [OpenRouter](https://openrouter.ai/) 和 [OAI](https://api.oaipro.com/) 作为 Backup，用不太容易在官方开到的模型，比如 Claude 家的。
*   客户端用 [ChatWise](https://chatwise.app?atp=sorrycc) 。
*   自研 Agent 基于 [@openai/agents](https://github.com/openai/openai-agents-js) + [ai](https://github.com/vercel/ai) 。
*   不用的有，1）逆向接口，Token 这么便宜，折腾这个的时间划不来，2）API 代理比如 [new-api](https://github.com/QuantumNous/new-api) 等，因为没有代理给非翻墙朋友用的需求，3）[open-webui](https://github.com/open-webui/open-webui)。

App 和 Chrome 插件：

*   [Wispr Flow](https://wisprflow.ai/) ，有 Mac 和 iOS 版，两个都很好用，免费，但应该需要翻墙，因为走的 OpenAI 的 Whisper 服务。
*   [RayCast](https://www.raycast.com/) 。其 AI 功能上周支持 BYOK（Bring Your Own Key），试用了下体验很好，准备从用了 10 多年的 Alfred 切过来了。
*   [SumBuddy](https://sumbuddy.app/)，Chrome 插件，经常用，用来做网页总结。
*   [Dia](https://www.diabrowser.com/)，偶尔用，用来做网页总结。
