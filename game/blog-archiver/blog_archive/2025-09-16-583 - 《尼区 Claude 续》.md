# 583 - 《尼区 Claude 续》

1、上个月买的 Claude Max 到期了，因为一些原因没续，换普通的 Pro 会员，消费降级下。

2、原因如下。

1）之前用一个 gmail 账号注册的 claude，但 gmail 是用接码平台注册的，gmail 要求二次认证，接码不能用直接的手机号，所以 gmail 账号等于挂了，而 claude api 的登录方式，有可能是会要求重新验证的，机制未知，出于保险起见，想着换个 claude 的号。

2）opus 没想象中那么神奇，也可能和我的用法有关。我目前主要在开发 code agent，这个类型的项目，我比较少让他进行长时间的大任务，还是以我自己为主，ai 为辅，ai 会写很多代码，60%+，但全都需要我进行 review 才会合入。所以，通常都拆了小任务交给 ai 做。

3）这段时间写文档更多，代码偏少。同时后面就是 10.1 长假。所以现在买 opus 不太划算。

4）想多用自己的 code agent 完成任务。

3、由于之前的 gmail 账号出问题，就不再走接码的方式注册了。但自己尝试了下，又都会弹手机，所以就上万能的闲鱼买了。¥9 买了 10 个… 好便宜。试了几个，登录时可以换辅助邮箱，然后进去后换密码就好。

4、然后用 gmail 账号注册 claude，并养号（时不时地聊几句）一天。

5、然后用尼区 apple store 下的 claude app 登录，再用尼区 app store 升级到 pro 。礼品卡是闲鱼买的，14900 = 89.9 。注：如果一个账号的 claude 已有 subscription，需要等 subscription 到期后才能切 claude 账号付新的。

6、最后，claude relay service 切新账号登录，DONE 。
