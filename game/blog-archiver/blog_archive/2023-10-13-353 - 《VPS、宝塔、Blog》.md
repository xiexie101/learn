# 353 - 《VPS、宝塔、Blog》

收到 cloudcone 的推广邮件，有「Pre-Black Friday」促销，2 CPU 2G 内存一年只要 $29，于是买了份。然后 google、chatgpt 一通问，最终用宝塔面板安装了 WordPress，并且把 [blog.sorrycc.com](http://blog.sorrycc.com) 指了过去，还有 https。全程不需要写代码，带摸索差不多 1 小时吧。记下流水账，方便有同样需求的朋友操作。

注：好久没玩 VPS 了，可能不是最佳实践，但部署个 Blog 应该是够用的。

1、买 VPS。（10m）

1）打开 [CloudCone](https://app.cloudcone.com/?ref=10409) 注册账号  
2）充值 $29（如果你买 2 CPU 1G 内存的可以冲 $16.5），支持支付宝  
3）打开 [https://hello.cloudcone.com/2023-pre-black-friday-vps-sale/](https://hello.cloudcone.com/2023-pre-black-friday-vps-sale/) ，选「PRE-BF VPS 23 - 2」，点「Deploy Server」，然后系统我选的「Debian 11 - x86\_64」，Host 随便填一个你期望的域名，点「Deploy」  
4）等几分钟，完成后会给你三样东西，IP 地址、用户名、密码，其中密码是通过邮件发给你的  
5）用 [https://ping.chinaz.com/](https://ping.chinaz.com/) 验证下 IP，确保没有被墙

2、部署宝塔面板。（10m）

1）ssh 登录，「ssh root@你的IP -p 22」，输入密码后登录  
2）安装宝塔面板，Debian 系统的命令是「wget -O [install.sh](http://install.sh) [https://download.bt.cn/install/install-ubuntu\_6.0.sh](https://download.bt.cn/install/install-ubuntu_6.0.sh) && bash [install.sh](http://install.sh) ed8484bec」，详见 [https://www.bt.cn/new/download.html#linux](https://www.bt.cn/new/download.html#linux)  
3）安装完成后会给三样东西，面板地址、用户名和密码

登录后长这样：

![](https://img.alicdn.com/imgextra/i1/O1CN012vRIOn24D6LUTA9vI_!!6000000007356-2-tps-2454-1324.png)

3、创建站点。（30m）

1）进入首页会弹出让你安装 LNMP 或 LAMP，建议 LNMP，用 Nginx。这个时间比较长，好像得 20 多分钟。  
2）选「网站 > PHP 项目 > 添加站点 > 创建站点 > 输入域名、选上创建数据库 > 提交」，会创建一个空的 PHP 站点。（这里有一键部署 WordPress 应用的选项，但是我总是失败，只能从空站点开始，然后手动装 WordPress）  
3）把域名指过来  
4）选「配置 > SSL > Let’s Encrypt」，开 https

此时已经可以通过 https 的域名访问你的站点了。

4、安装 WordPress。（10m）

1）在 [https://wordpress.org/download/](https://wordpress.org/download/) 下载最新的 wordpress zip 包  
2）回到宝塔面板，选「文件」，上传压缩包、解压到 /www/wwwroot/ 下前面对应的网站里  
3）访问你的域名，根据 wordpress 的引导完成操作  
4）配置 > 配置文件里加这一段，这样在调整了固定链接后也让非入口能不加 `/index.php` 直接访问

```nginx
location / {
  try_files $uri $uri/ /index.php?$args;
}
```

5）设置 WordPress  
设置 > 固定链接，选「文章名」  
6）安装插件。
