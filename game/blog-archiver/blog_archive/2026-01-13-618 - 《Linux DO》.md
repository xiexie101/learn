# 618 - 《Linux DO》

![cover](https://pic.sorrycc.com/proxy/1768804531763-308845557.png)

个人感觉 [linux.do](http://linux.do) 是目前国内内容质量和社区氛围都比较出众的程序员社区，应该和站长的风格也有关系，禁 ai 润色、禁 aff 等，同时加上等级系统和保级压力，以及最近上的 credit 货币系统，大家会有压力和动力去产出更好的内容。

所以感觉可以囤点号和有意地升下级。我注册应该比较早了，最早是为了免费的 deeplx api 接口注册的，后来没咋管。最近感觉有些想看的帖子需要等级，就特地关注了下，今天发了个[帖子](https://linux.do/t/topic/1438030)点赞数就够三级了。

![](https://pic.sorrycc.com/proxy/1768273656232-527807615.png)

整理下一些 Tips。

1、[https://connect.linux.do/](https://connect.linux.do/) 可以看自己的等级和升下一级的要求，对照着数据来，升起来还是很快的。

2、我注册了两个号，手写的申请稿，也忘记咋写的了，反正是一边过的。目前好像已经关注注册，需要 3 级号的用户手动邀请才行。

3、不少类别下都有 2 级和 3 级区，不同等级用户可以看到不同的内容。同时有些公益站对用户等级做了限制，我也是我之前想升 2 级的原因之一。3 级区个人感兴趣的主要是各种注册机的 topic，感谢上周末借我账号的群友，周末抽空搞定了一些网站的自动注册，同时买了一堆 google 账号，省了不少自己注册的时间。

4、升 3 级最短时间应该是 50 天。1）刷已读帖子的 Tip 是找火一点的话题，1000+ 回帖的那种，刷一下就有很大的量了，2）之前也有试一个保活脚本，跑过一段时间，但感觉太慢，不如刷或的话题更快一些，我还拿来完善了一把，代码见 [https://gist.github.com/sorrycc/f680de25d3fec8851c4e12c24a09981e](https://gist.github.com/sorrycc/f680de25d3fec8851c4e12c24a09981e) 。

5、跳蚤市场不需要三级，只要到 [https://linux.do/g](https://linux.do/g) 点击加入「交易分组」即可。
