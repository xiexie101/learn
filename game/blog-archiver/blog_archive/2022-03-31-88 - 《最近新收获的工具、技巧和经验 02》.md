# 88 - 《最近新收获的工具、技巧和经验 02》

1、自从豆瓣关闭 API 之后，好多 douban 的 alfred 插件都挂了，最近发现这个 [douban-workflow](https://github.com/h3l/douban-workflow) 能用，不用再打开豆瓣网站搜索评分了，见图。

![](https://img.alicdn.com/imgextra/i3/O1CN01UQPNhP1xVh9PPrNm7_!!6000000006449-0-tps-1246-1070.jpg)

2、启用 [Focus 软件](https://heyfocus.com/)，支持 SetApp，强制上午和其他需要专注的时间不开社交软件和社交网站，增加坏习惯的门槛。

3、在淘宝付费 23 元为阿里云盘新增 8T，用钱换时间的事最划算了。

4、万能的淘宝搜「Apple 教育优惠」，5 块钱可以买到优惠码，还没试过，有同事用这个买 mbp 省了几千块钱。

5、试了一段时间用「语音录入」提升写作速度，快速把想法记录下来，然后统一整理，Mac 下识别不太准，iPhone 下倒是还挺准确的，但还不够准，后续准备尝试其他语音转文字的服务。

6、多思考 10X 的做法，参考 [70 - 《如何成为 10X 工程师》](https://sorrycc.com/10x-engineer)

7、尝试了一天[「间歇性断食」](https://freewechat.com/a/MzA3MzE5MjM2Mw==/2672247475/1)，被老婆叫停，下个月再试试。操作方法很简单，中午 12 点吃饭、晚上 8 点前吃饭，保证 16 小时不进食。

8、尝试微信「关闭朋友圈」，并打开「青少年模式」，看下能坚持多久。

9、查 IP 或运营商数据可在命令行中输入「curl [cip.cc](http://cip.cc)」。
