## 本周（这段时间）我感兴趣的事

1、Mako 对外内测了！

Mako 已于 5 月初[正式对外内测](https://zhuanlan.zhihu.com/p/697853948)，Umi 用户可通过配置 `mako: {}` 一键开启，非 Umi 用户也可尝鲜。更新了下内测群二维码，感兴趣的同学可以加入讨论。

![](https://res.cloudinary.com/sorrycc/image/upload/v1717380719/blog/sstfm233.jpg)

可能有些同学还不了解，和大家再介绍下 Mako。Mako 是又一个 Rust 构建工具，出自蚂蚁，去年上半年开始研发，下半年对内正式发布，目前在蚂蚁上线了几百个应用，包含中后台、无线、搭建、Low Code 等场景。

官网地址： [https://makojs.dev/](https://makojs.dev/) 。

2、GIAC 和 SECon。

感觉疫情结束，AI 发展起来后，今年各种会议又多起来了。Mako 今年也有几场分享，5 月底在 GIAC 分享了 [《Rust 构建工具在蚂蚁的研发和落地》](https://giac.msup.com.cn/2024sz/course?id=17467)，6 月底还会再 SECon 换个角度再分享下 Mako。

5、Solid Start。

![](https://res.cloudinary.com/sorrycc/image/upload/v1716367426/blog/2nogai4e.png)

趁 SolidStart 1.0 发布的时机开了下源码。感觉 Server 组件是新框架的标配了，同时新看到 [Vinxi](https://github.com/nksaraf/vinxi) 这个库感觉还不错，他提供了用于搭建框架所需的常见 SDK，比如路由、服务端处理、RSC、SSR、SPA 等。有了 Vinxi，基于 Vite 搭建一个新的框架好简单，可惜我们不用 Vite。详见 [442 - 《Solid Start》](https://sorrycc.com/solid-start) 。
