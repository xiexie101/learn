# 358 - 《CSS 错误片段集锦》

以下均为有错的 CSS 片段，来自真实项目。验证 mako 在 Bigfish 项目的可用性时发现。大家可以看看能发现哪些错误，检查下自己是否有掌握各种 CSS 的奇怪知识点。

1、

```css
@media screen (max-width: 800px) {
    .jsonWrapper {
        width: 300px;
        overflow-y: auto;
    }
}
```

2、

```css

   left: 24px;
   height: calc(100vh - 60px);
   max-width: calc(100%-60px);
 }
```
