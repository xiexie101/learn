# 581 - 《在用的 Chrome 插件 2025》

> 更新一版。

*   [uBlock Origin Lite](https://chromewebstore.google.com/detail/ublock-origin-lite/ddkjiahejlhfcafbddmgiahcphecmpfh)，高效的内容拦截器，符合 Manifest V3 规范的广告屏蔽工具
*   [Immersive Translate](https://chromewebstore.google.com/detail/immersive-translate-trans/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)，沉浸式双语网页翻译扩展，支持 PDF 和视频字幕翻译
*   [爱贝壳内容同步助手](https://chromewebstore.google.com/detail/%E7%88%B1%E8%B4%9D%E5%A3%B3%E5%86%85%E5%AE%B9%E5%90%8C%E6%AD%A5%E5%8A%A9%E6%89%8B/jejejajkcbhejfiocemmddgbkdlhhngm)，一键全网分发内容到 50+ 社交平台的多平台内容管理工具
*   [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)，用户脚本管理器，让您运行自定义脚本增强网站功能
*   [Bitwarden Password Manager](https://chromewebstore.google.com/detail/bitwarden-password-manage/nngceckbapebfimnlniiiahkandclblb)，开源的密码管理器，安全生成和存储密码
*   [Extension Manager](https://chromewebstore.google.com/detail/extension-manager/gjldcdngmdknpinoemndlidpcabkggco)，扩展管理器，快速启用/禁用浏览器扩展并支持批量操作
*   [Enhanced GitHub](https://chromewebstore.google.com/detail/enhanced-github/anlikcnbgdeidpacdbdljnabclhahhmd)，增强 GitHub 功能，显示文件大小、下载链接和复制文件内容
*   [SumBuddy](https://chromewebstore.google.com/detail/sumbuddy-browser-ai-assis/knpckkifmkioijpoejgngbghdpacfajp)，浏览器 AI 助手，可基于网页内容对话并使用自己的 AI 模型
*   [Obsidian Web Clipper](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)，Obsidian 官方网页剪藏工具，将网页内容保存为 Markdown 文件
*   [Buster: Captcha Solver for Humans](https://chromewebstore.google.com/detail/buster-captcha-solver-for/mpbjkejclgfgadiemmefgebjfooflfhl)，自动解决音频验证码的辅助工具，帮助视力障碍用户
*   [I still don’t care about cookies](https://chromewebstore.google.com/detail/i-still-dont-care-about-c/edibdbjcniadpccecjdfdjjppcpchdlm)，自动处理 Cookie 同意弹窗，移除或自动接受网站 Cookie 政策
*   [小电视空降助手](https://chromewebstore.google.com/detail/%E5%B0%8F%E7%94%B5%E8%A7%86%E7%A9%BA%E9%99%8D%E5%8A%A9%E6%89%8B/eaoelafamejbnggahofapllmfhlhajdd)，自动跳过视频中的赞助广告片段，精准定位到高能时刻
*   [SingleFile](https://chromewebstore.google.com/detail/singlefile/mpiodijhokgodhhofbcjdecpffjipkle)，将完整网页（包含 CSS、图片、字体等）保存为单个 HTML 文件
