# 56 - 《App 推荐：Espanso》

[https://github.com/federico-terzi/espanso](https://github.com/federico-terzi/espanso)

Espanso 是文本替换工具，比如输入 :mobi 替换为手机号，输入 :date 替换为日期，输入 :weather 替换为当前天气等。

**💡 特点**

*   支持 Win、Mac 和 Linux
*   支持图片替换
*   支持全局搜索（有时候会忘了关键字）
*   支持正则触发（所以可以有参数，比如 :hello(成)）
*   支持交互式表格
*   支持 Shell 和自定义脚本（Python、Node 均可）
*   支持包管理，提供类似 npm 的 espanso hub
*   配置基于文件（所以方便组织，比如区分 life、code、emoji、chat 等）
*   支持部分配置针对应用开启（比如代码类的只开 WebStorm）

**💸 价格**

当前为免费的开源软件，推荐安装使用。

**🧙‍♂️ 使用体验**

之前我是用 Alfred 的 Snippets 功能做文本替换，Mac 系统其实也内置了文本替换功能，但这些都是静态的。Espanso 则是动态的，比如输入 :weather，可以通过请求拿到当前天气，然后输出，这对于程序员来说，有着非常大的想象空间。此外，Espanso 安装后的引导也做的很好，还有提供命令行，并且可通过 brew 安装，安装后 16.8M。

注：大家安装时记得安装 2.0.0-alpha 的版本。
