# 524 - 《cursor-tools》

1、cursor-tools 是一个命令行工具，它可以让 Cursor AI Agent 获得网页搜索、代码库理解、浏览器自动化、文档生成能功能，从而扩展 Cursor 的能力范围。cursor-tools 是针对 Cursor Composer Agent 优化的，但它可以通过命令行的方式被其他 AI Agent 所使用。

![](https://tcc.sorrycc.com/p/31536112-cb2c-465b-b1ef-bff845c072f1.png)

2、安装使用。

```
npx -y cursor-tools@latest install .
```

然后在 Composer 里用自然语言提问即可。

3、cursor-tools 实现的方式是这样。1）提供各种命令，2）在 cursor rules 里描述所提供命令的参数和结果。他所提供的功能如下（基于命令），当然你不用记命令，只需在 Composer 里用自然语言描述需求即可。

*   `cursor-tools web`, 使用 Perplexity AI 从网络获取答案
*   `cursor-tools repo`, 使用 Google Gemini 获取仓库相关的上下文感知答案
*   `cursor-tools doc`, 为仓库生成全面的文档
*   `cursor-tools github pr`, 获取最近 10 个 PR 或指定 PR 的信息
*   `cursor-tools github issue`, 获取最近 10 个 issue 或指定 issue 的信息
*   `cursor-tools browser open`, 打开 URL 并捕获页面内容、控制台日志和网络活动
*   `cursor-tools browser act`, 使用自然语言指令在网页上执行操作
*   `cursor-tools browser observe`, 观察网页上的交互元素并提供可能的操作建议
*   `cursor-tools browser extract`, 基于自然语言指令从网页提取数据

4、看了下实现。

1）web 基于 Perplexity，用 ai + @ai-sdk/perplexity 很好实现

2）repo 用 [repomix](https://github.com/yamadashy/repomix) 打包，然后用 Gemini 2 提问，因为 Gemini 2 有 2M 的 Context Token，可以问大项目

3）doc 和 repo 类似，用 repomix 打包，然后用 Gemini2 提问，prompt 如下。

```
Focus on:
1. Repository purpose and "what is it" summary
2. Quick start: How to install and use the basic core features of the project
3. Configuration options and how to configure the project for use (if applicable)
4. If a repository has multiple public packages perform all the following steps for every package:
5. package summary & how to install / import it 
6. Detailed documentation of every public feature / API / interface
7. Dependencies and requirements
8. Advanced usage examples
```

4）github pr 和 issue 和 [api.github.com](http://api.github.com)，配 `GITHUB_TOKEN` 可以去 rate limit 限制。

5）browser open 基于 playwright，获取 html、录视频和截图。

6）browser act、browser observe 和 browser extract 都基于 [stagehand](https://github.com/browserbase/stagehand)，一个 AI web 浏览框架。
