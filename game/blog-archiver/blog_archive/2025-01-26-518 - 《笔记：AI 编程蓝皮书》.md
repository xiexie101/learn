# 518 - 《笔记：AI 编程蓝皮书》

昨天看到 [https://superhuang.feishu.cn/wiki/CBBPwvgEuicVhFkx0s7cPmhpn4e](https://superhuang.feishu.cn/wiki/CBBPwvgEuicVhFkx0s7cPmhpn4e) ，之前没用过 Coze，实操了下，做下记录。

1、表情包生成器。

1）注册 [Coze](https://www.coze.cn/)，花 1 快钱升级专业版（会话 50 QPS）  
2）创建智能体、工作流，先用「大模型」基于你的输入生成 title 和 content，然后喂给「图像生成」，返回结果

![](https://tcc.sorrycc.com/p/cf204375-337f-4213-8fa6-9f561f3a6249.png)

3）发布工作流  
4）设定「人设与回复逻辑」

```
当用户发送内容时，调用 biaoqingbao 工作流，将用户发送的内容作为 input 字段传入
```

5）发布智能体（取消扣子商店，选上 API）。  
6）[Coze 授权](https://www.coze.cn/open/oauth/pats) 里创建 Token，然后通过 curl 或者 fetch 的方式验证（user\_id 随便填）。

```
curl --location --request POST 'https://api.coze.cn/v3/chat' \
--header 'Authorization: Bearer pat_xxxx' \
--header 'Content-Type: application/json' \
--data-raw '{
    "bot_id": "746***",
    "user_id": "9527",
    "stream": true,
    "auto_save_history":true,
    "additional_messages":[
        {
            "role":"user",
            "content":"早上好",
            "content_type":"text"
        }
    ]
}'
```

2、部署到 Cloudflare。

1）用 Cursor 创建 index.html 和 worker.js，我的需求描述如下。

```
生成一个表情包应用。

- 基于简单的 html、css、js 实现，所有代码写在一个 html 文件里。
- 整体风格要简洁，不要花里胡哨。
- 用户输入后，点击按钮，有 loading 效果，然后生成表情包。
- 配色用马里奥游戏的配色，鲜艳一些。
- html 将会部署到 cloudflare pages 上。
- api 部分会部署到 cloudflare workers 上，请生成单独的 worker 文件。

api 部分的参考代码如下。
{{复制 curl 的那部分}}
```

2）Workers & Pages > 创建 Worker > 部署。  
3）创建 Page > 部署
