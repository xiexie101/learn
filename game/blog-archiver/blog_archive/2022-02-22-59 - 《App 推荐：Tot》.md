# 59 - 《App 推荐：Tot》

Tot 是极简的笔记软件，开发团队还有开发过 Twitterrific 等。极简到什么程度？有且只有 7 个 Tab，存放 7 篇笔记。所以这注定不是用来存放「普通」内容的笔记软件。

💡 特点

*   支持 Mac 和 iOS
*   支持 iCloud Sync
*   支持纯文本、富文本和 Markdown
*   快捷键丰富，比如 ⌃A、⌃E、⌃F、⌃B 的光标切换、⌘/ 隐藏状态栏、⌘1-7 切换 Tab，也可以用 ⌃⇧⇥ 或 ⌃⇧⇤ 左右切换等
*   支持 schema 协议和 applescript，比如 open tot://1/append?text=foo

💸 价格

macOS 免费，iOS $19.99。可以先用 macOS 版，iOS 版等有特价时再入。

🧙‍♂️ 使用体验

用来存什么？我的想法是用来存「暂存」的内容，比如简报、日更、会议记录，见图1。这一类发完即删，我不会将其存于本地。之前是记在 Flomo 中，有个问题是 Flomo 每天会记很多，Flomo 又没置顶功能，所以每次添加简报内容时都需要往下翻一下，就很麻烦。而基于 iA Writer 用 Markdown 记又太麻烦，这类内容用 PlainText 足以。
