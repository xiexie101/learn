# 170 - 《教授与年轻人阅读笔记：认知思辨篇》

拿到书后，先挑感兴趣的最后一部分「认知思辨篇」看了，以下是笔记。

1、我们为什么一定要读书？为了「将人置于抽象的因果背景之中，并给予大尺度根本性的指向」。说人话，就是「能做出更明智的选择」。而要做出选择，就得理解因果关系，知道做 A 会导致 B。这就需要有推导的能力，而推导又需要人们能理解抽象概念，比如社会主义、一国两制、节日促销、春节档期都是抽象概念。作者举了个不识字的农民的例子，「鲁利亚问这位农民，在遥远且下雪的北方，所有的熊都是白色 的，新地岛位于遥远的北方并总是下雪，所以那里的熊是什么颜色 的?」农民由于只见过黑色的熊，是推导不出「白色」的答案，而读书能让你见过白色的熊。

2、进步的「迷信」。不是说在学习的过程中承受痛苦就能取得进步，痛苦和进步之间并没有因果关系。取得进步的过程中确实可能承受痛苦，但承受痛苦不一定会带来进步。那怎么才能进步？科学正确的训练，用两位美国心理学家安德斯·艾利克森和 罗伯特·普尔的话说，要取得进步，得做到 3 个 F。Focus（专注）、Feedback（反馈）和 Fix It（修正）。在练习中解决一个个具体切实的小问题，不断地积跬步为千里，积小胜为大胜，才能最终脱胎换骨。

3、逻辑的力量。什么是逻辑？「正确地将感知到的事实串联起来，就可以用已知预言未知，而这种正确串联事实的方式，就是逻辑。」逻辑有什么用？对外可以扩宽认知边界，对内可以主宰自己。对外，人眼只能看到380~780纳米的可见光，人耳只能听到20~20000赫兹的声音，这对于大千世界来说，几乎又聋又哑。借助逻辑，可以推断出未知领域，比如海王星、微观粒子等，比如「在夏至日用直尺就能知道地球的周长」。对内，通过逻辑仔细检查吸纳进来的观点，可以避免成为蛋白质傀儡，被别人的想法肆意玩弄和利用。具体怎么检查？分两个层面。1）事实是否可靠、全面，2）事实串联起来的方式是否正确且符合逻辑。

4、无聊的本质。无聊是痛苦的，有个实验里，不少人宁愿选择电击自己的痛苦，也不愿忍受无聊。那无聊的本质是什么？「消遣太少或太多都会促成无聊的情绪。要想摆脱无聊，就需要确保消遣的数量在不多不少之间处于微妙的平衡。」消遣太少很好理解，缺乏刺激会让人不知如何安放自己的注意力，有句老话叫「没事就要找事」。很多年轻人没事找事就是因为无聊，因为他们自制力更差一些，负责自制力的前额叶大概20岁左右才会发育成熟。那消遣太多为啥也会无聊？因为其中隐含着放弃，人们感觉无聊，是因为机会成本的上升。如果游戏机硬盘里只有一款游戏，那你能玩得很投入，但如果有 150款游戏，那你玩其中1款就意味着你放弃了另外149款，机会成本太高，你就会玩得心神不宁。那怎么不无聊？文中教授的方法是，1）回归现实生活，2）把注意力放在科研和教学上，3）删除多余的手机应用，4）把 PS5 送人。。

5、幽默的内核是悲剧。陈佩斯也说过「喜剧的内核是悲情」。先来个类比，比如辣椒，在吃了辣椒之后，身体就会感到灼伤似的疼痛，大脑为了压制 这种痛苦就会分泌一种叫内啡肽的东西，它能镇痛，同时让你感到愉悦。再说痛苦，当精神上处于煎熬之中时，人格为了宽慰和保护自己，就会营造出幽默，以带来镇痛的效果。幽默的本质是内心痛苦的溢出。一个人要想成为幽默的人，往往需要饱经痛苦。我真有点为脱口秀演员担心。。同时，要区分搞笑和幽默，这两者是不同的。

6、“远方”在此刻的意义。也是两层含义，精神的远和肉体的远，比如读远的书和去远的地方。先看近有什么问题，近的知识更多是对固有经验和直觉的修正，这意味着局限性。随着交通的发达，现代人很少甘愿守在一个小地方度过一生。这就需要突破经验和直觉的局限，需要更加深刻的规律带来更具普适性的指导。不仅看到蚂蚁搬家就去田里开沟，还要深究其原理。但越是深刻的规律，就越是需要遥远的知识来孕育。不仅看到蚂蚁搬家就去田里开沟，还要深究其原理。所以，让精神和肉体都抵达远方，这对个人的成长很有价值。「远方除了远，一无所有？」「远，就够了。」

7、脑内繁星。「要把知识记得牢靠很简单，那就是你要尽可能地把它们用一个故事联系起来。人类这个物种就是这样，凭空记忆很难，联想却很容易。」
