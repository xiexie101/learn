# 417 - 《我如何用翻译》

之前多少也提过，这里汇总下。

1、我应该是比较重依赖翻译工具和服务的。简单说，就是两个软件 [Bob](https://bobtranslate.com/) 和 [Immersive Translate](https://chromewebstore.google.com/detail/immersive-translate-web-p/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)，以及两个服务 Deepl 和 OpenAI。

2、Bob 用于翻译片段。Bob 这个软件一开始是开源的，后面转成付费 App。如果你想用免费的，也可以考虑开源的 [Easydist](https://github.com/tisfeng/Easydict)，个人没试过，但看着也不错。Bob 需要安装两个插件，bob-plugin-openai-translator 和 bob-plugin-deeplx，分别用于 openai 和 deepl 的服务集成。

![](https://img.alicdn.com/imgextra/i3/O1CN01IEReBD1N3uyEhSMEI_!!6000000001515-2-tps-892-1084.png)

3、Immersive Translate 用于文章和书的翻译。需要注意的是，默认 deeplx 没有开启，需要先在「左下角 Developer settting」里开启「Enable Beta Testing Features」，然后「Translation Service」里才会有「DeepLX（Alpha）」。

![](https://img.alicdn.com/imgextra/i3/O1CN01zDz2EG1tYCrvYwrbh_!!6000000005913-2-tps-2134-1286.png)

注意可以开下「Hover + Ctrl 翻译当前段落」的功能。

![](https://img.alicdn.com/imgextra/i4/O1CN01P2ssrP1pZLOAjvhxk_!!6000000005374-2-tps-620-886.png)

他还有个「连按三次空格」会把当前输入框里的内容做翻译的功能，藏的很深但很好用。

4、服务选择上。非技术类的我会用 Deepl，免费，够用；技术类的我用 OpenAI 的 GPT 4 加 Turbo，并且加上如下 prompt，效果会好很多，不会乱翻。

> You are a professional, authentic translation engine, with professional programming experience, keep the link as much as possible, add space between chinese charactor and others, only returns translations, don’t answer questions.

4、Deepl 找了很多渠道想付费付不了，只能用免费的了。Deepl 的 API 里加了一些坑，通过逆向其官方的 Chrome 插件可以找出逻辑。Deeplx 可以直接用别人搭好的服务是 [https://api.deeplx.org/translate](https://api.deeplx.org/translate) ，也可以直接[部署一个](https://github.com/OwO-Network/DeepLX)，推荐[云函数版](https://github.com/LegendLeo/deeplx-serverless)以避免「429 请求太频繁」报错。

5、OpenAI 的 API 有几个选择，官方、Azure 和第三方。Azure 可参考 [362 - 《用 Azure 申请 OpenAI API》](https://sorrycc.com/azure-openai) 申请一个。第三方胜在便宜，价格是官方的 1/8，缺点是偶尔不稳定和相对官方的慢，因为有一次中转。日常我会用官方，翻译书时会切换到第三方。

6、。
