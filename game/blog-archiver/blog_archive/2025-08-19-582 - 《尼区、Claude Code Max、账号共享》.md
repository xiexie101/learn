# 582 - 《尼区、Claude Code Max、账号共享》

以下是我走完一遍之后梳理的 SOP。

1、参考 [579 - 《美国家宽》](https://sorrycc.com/usa-static-proxy) 搞几个美国家宽代理，确保 Claude Code 不会被封。

2、注册 gmail、claude 账号和养号。

1）先下载「比特浏览器」，10 窗口免费够用，你也可以选其他指纹浏览器。作用是让某个账号始终走某个 IP。

![](https://pic.sorrycc.com/proxy/1755570373737-782017759.png)

2）到 [https://mail.google.com](https://mail.google.com) 注册 gmail 账号，接码我在用 [DaisySMS](https://daisysms.com/) 你也可以用其他的。（注：可以多搞几个，万一一个被封了换另一个。）

3）用这个 gmail 注册 claude 账号。

4）养号，每天和 claude 聊聊，过几天看看有没有封。（我之前在手机上用尼区账号直接注册了一个 claude 账号，没多久就被封了。）

3、注册尼区 Apple 账号。机场切尼区（可能也不用切），到 [https://account.apple.com/account](https://account.apple.com/account) 注册苹果账号，国家选「Nigeria」，邮箱用 gmail 账号，手机用国区手机号。在手机上登录后，要补充收款地址，地址用一个随机的公寓地址就好了。

注：1）别用接码平台，后续两步验证的时候可能还需要校验码，2）一个手机可绑十几个 Apple 账号的，具体数字未知。

4、闲鱼买礼品卡给尼区账号充钱。我买 100$ 的 5x max，是买了 7 张 14900 的卡。

5、尝试付款。付 20$ 的 pro 可能会成功，但付 100$ 的大概率会遇到风控，会提示付款失败。

> your purchase could not be completed  
> for assistance,contact iTunes Support at [http://www.apple.com/support/itunes/ww/](http://www.apple.com/support/itunes/ww/)

![](https://pic.sorrycc.com/proxy/1755571318799-132973842.png)

6、找客服。去 [http://www.apple.com/support/itunes/ww/](http://www.apple.com/support/itunes/ww/) 找人工客服，描述问题，然后他会和你说已处理，等待 48 小时后再试。（这个 48 小时是常量不是变量，不用提前试，没用）

![](https://pic.sorrycc.com/proxy/1755571298188-876054962.png)

7、48 小时后再次尝试付款，成功！

![](https://pic.sorrycc.com/proxy/1755571421826-521374189.png)

8、共享给朋友使用。基于 [https://github.com/Wei-Shaw/claude-relay-service](https://github.com/Wei-Shaw/claude-relay-service) 搭一套就好，注意添加账号时填入前面买的美国家宽代理。好处是，1）多人共享，2）不用翻墙，3）IP 稳定不会被封（或者被封概率少）。
