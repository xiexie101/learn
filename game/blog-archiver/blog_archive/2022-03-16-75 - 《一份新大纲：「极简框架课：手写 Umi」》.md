# 75 - 《一份新大纲：「极简框架课：手写 Umi」》

见图，计划先在星球里试更一部分。

![](https://img.alicdn.com/imgextra/i3/O1CN01dR9XWD1OZo9eyt4Ds_!!6000000001720-0-tps-1588-2052.jpg)

之前工程化那个试写了几篇草稿，虽然熟悉，但真正上手写起来还有点费精力，一些点写之前还得先翻阅资料，等 Umi 4 发布之后时间充裕了再深入。然后在想有啥是写起来无压力的，就想到 Umi。

定位是面向新手的入门课，写起来没有压力，所以不容易烂尾。计划是从 Demo 开始，一步步引导开发者完成一个极简框架的开发，过程中穿插前端的前沿技术、框架和工程化的知识，最终发布到 github 和 npm。

虽然极简，但五脏俱全，该有的功能都会有。

极简是我学新领域内容时经常看的书籍种类。比如七堂极简物理课、极简宇宙史、极简音乐史等，读起来也轻松无压。
