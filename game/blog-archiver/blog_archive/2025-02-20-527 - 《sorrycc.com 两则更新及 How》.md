# 527 - 《sorrycc.com 两则更新及 How》

> 有没有觉得你的网站用户登录过期时间比你在 TikTok 上刷短视频的耐心还短？ 嗯，[sorrycc.com](http://sorrycc.com) 听到了你的无声尖叫（或者只是收到了一些用户反馈），并且刚刚发布了两项更新，让一切都变得更顺滑了，而且幕后功臣竟然是……没错，AI！

两则更新。

1、登录失效时间改成 3 个月。  
2、默认的 RSS Feed 包含会员文章。

下面说下如何实现的。

1、我对 php 和 wordpress 都不熟，所以就得借助 AI 。

2、Wordpress 失效时间的修改。

我先用 [Perplexity.ai](http://Perplexity.ai) 查了一下 WordPress 默认的登录过期时间，确实有点短。没有勾「Remember Me」48 小时，勾了「Remember Me」14 天。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/gHun9v.png)

然后问了下 AI，推荐用插件修改。于是我找到 [Remember Me Controls](https://wordpress.org/plugins/remember-me-controls/)，安装、启用、设置时间，三步搞定！

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/RveyIL.png)

3、RSS Feed 包含会员文章，这个稍微复杂点。

由于我用了 [Ultimate Member](https://wordpress.org/plugins/ultimate-member/) 插件做会员权限管理，然后发现 RSS Feed 默认情况下并不会包含会员文章，这怎么行。。我找了下相应的配置，没有找到针对 Feed 的管理。于是只能从代码角度入手。由于不会 php，只能让 AI 来帮忙了。

我先在官网下载代码。（注：这里踩了个坑。我先用最新版 2.10.0 走了遍流程，但发现无效，结果我用的是 2.8.2）。下下来后用 repomix 打包代码成单独的文件，然后复制后问 AI。

```
repomix --include "includes/**/*.php"
cat repomix-output.txt| pbcopy
```

PROMPT 如下。

```
代码。

wordpress 加 Ultimate Member 之后，有些文章设置了隐藏。问题是 rss 输出里也没有这些文章了，我希望输出这些文章的摘要部分，应该怎么处理，应该告诉我一步步的步骤。in chinese.
```

结果 AI 非常给力，直接给出了解决方案。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/DKsIev.png)

搞定。
