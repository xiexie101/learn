# 411 - 《我的小技巧集合 07》

> 每凑够 10 条发一篇。

1、推荐个 App「Whisper Notes」，用于语音转文字，可离线使用，断句感觉比 iOS 17 的还好用，支持 Mac。很好用，售价 ￥8。

2、Mac 上如何用「小红书」？1）App Store 上下载 iOS 版的 App，2）用 Safari 打开后存成 Web App。推荐后者。

![](https://img.alicdn.com/imgextra/i4/O1CN01hOldBF1MzL0Cyxoxt_!!6000000001505-2-tps-2386-1690.png_1200x1200.jpg)

3、iOS 上切美区安装「ReadWise Reader」，然后 iOS 上看到文章时就可以分享到 Reader 里了。

![](https://img.alicdn.com/imgextra/i3/O1CN01EizMbU1lsLKkvH34h_!!6000000004874-2-tps-804-1742.png)

4、如何阅读 Medium 的文章？1）[Medium Parser Chrome 插件](https://chromewebstore.google.com/detail/medium-parser/egejbknaophaadmhijkepokfchkbnelc)，2） [https://readmedium.com/](https://readmedium.com/) 或基于此的[油猴脚本](https://greasyfork.org/zh-CN/scripts/481493-medium-unlocker-2023-12-06-updated)。

![](https://img.alicdn.com/imgextra/i4/O1CN01Ji1qOT1FrpBVUiLTC_!!6000000000541-2-tps-2226-1370.png)

5、Mac 可以用流的方式玩 Windows 上的游戏，我试过玩动作游戏 SIFU 都很顺畅。

6、APTV 支持 Mac，最近用来在 Mac 上看亚洲杯和 CBA。

7、如何在一个项目里使用一个依赖的多个版本？用 npm: 前缀声明依赖，比如 jquery2: npm:jquery@2。[来源](https://twitter.com/housecor/status/1738212361406730320)。

8、如何找机场？ [这里](https://t.me/jieniguijichang)的评测还不错。

9、iCloud 中的文件夹只要在末尾添加 `.nosync` 就不会同步了，实用。[来源](https://twitter.com/mzwlevi/status/1747196713516024231?s=20)。

![](https://img.alicdn.com/imgextra/i1/O1CN01OkIGYt1ysQPu5XY2A_!!6000000006634-2-tps-514-180.png)

10、写文档时推荐用 VSCode，因为有 Copilot 辅助。

11、一个 VSCode 技巧，保存时自动添加缺失的依赖。先「⌘+⇧+P」，输入「open keyboard shortcuts (json)」，回车后会打开「keybindings.json」，添加内容如下（window 系统可以把 cmd 换成 ctrl）。[参考](https://dev.to/krisplatis/auto-add-missing-imports-on-file-save-in-vs-code-1b89)。

```json
{
  "key": "cmd+s",
  "command": "editor.action.sourceAction",
  "args": {
    "kind": "source.addMissingImports",
    "apply": "first"
  }
}
```

效果如下。

![](https://img.alicdn.com/imgextra/i4/O1CN01wHnqXn20cyI1tqqRh_!!6000000006871-1-tps-606-342.gif)
