# 1 - 《前端流量密码》

早上在想一件事，互联网的流量密码是「美女、大长腿、豪宅、豪车、八卦、吃瓜、…」（金钱、异常、捷径、性暗示、暴力、死亡、民族主义），sspai 的流量密码是「笔记、写作、App、…」，前端的流量密码是啥？

拍脑袋列了一些，

*   趋势
*   发布
*   Low Code
*   Bundless
*   Rust
*   Vite
*   提速
*   名人
*   手写 xxx
*   心路历程
*   资源整理（awesome-xxx）
*   …

这些「因子」取其一二，加上「免费、下载、笔记、开源」等通用因子的组合，再配上「提升主题的图、有争议的观点、适当的标题党」，很多人就会莫名其妙被戳中。

然后相关的文章就比较火，比如每年都会有人问「202x 前端有啥新变化」、「强如死月」。
