# 296 - 《CSS 颜色表示法》

1、最早的颜色表示是这两种。1）预定义的颜色值，2）十六进制颜色表示法（CSS 2 里引入）。预定义的颜色值比如 red、blue、green 等；十六进制颜色表示法以 `#RRGGBB` 的格式，其中 `RR`、`GG` 和 `BB` 分别表示红、绿、蓝三个通道的值，比如 `#FF0000` 表示红色。

```css
color: red;
color: #FFF000;
```

十六进制表示法允许在 0-255 的范围内为每个通道指定一个值，从而提供了约 1670 万种颜色的选择。然而，这种表示方法并不直观，难以从颜色代码中直接推断出颜色的外观。

2、然后随着 CSS 3 的推出，引入了 `rgb()`、`rgba()`、`hsl()` 和 `hsla()` 四个函数。`rgb()` 函数接受三个参数，分别代表红、绿、蓝三个通道的值（范围为 0-255），比如 `rgb(255, 0, 0)` 表示红色，这相比十六进制更直观，但还不够直观；`hsl()` 函数接受三个参数：色相（0-360）、饱和度（0-100%）和亮度（0-100%），使颜色的选择和调整更加直观和容易，比如 `hsl(0, 100%, 50%)` 表示红色，这种表示法可以在保持相同的饱和度和亮度的情况下，通过改变色相值，可以轻松地获取不同的颜色。

```css
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, .5);
color: hsl(0, 100%, 50%);
color: hsla(0, 100%, 50%, .5);
```

3、随着 CSS4 和 CSS Color Module Level 4 的推出，一些基于更先进的色彩空间的颜色表示方法被引入，包括 `hwb()`、`lab()`、`lch()`、`oklab()` 和 `oklch()`。

![](https://img.alicdn.com/imgextra/i3/O1CN01bMJDra20qFgyQMh6z_!!6000000006900-2-tps-1754-366.png)

其中最吸睛的是 `oklch()` ，他基于人类对色彩的感知。`oklch()` 函数接受四个参数：亮度（L，0-100）、色度（C，鲜艳度，0-0.37）、色相（H，彩虹的颜色，红橙黄绿青蓝紫，0-360）和透明度（0-1，可选），比如 `oklch(50% 0.37 30deg)` 表示红色。

```css
color: oklch(50% 0.37 30deg)；
```

其中色相里有一些关键的颜色点还是需要记的，

*   红色：30
*   黄色的：90
*   绿色的：140
*   青色/青色: 195
*   蓝色: 260
*   品红：330
