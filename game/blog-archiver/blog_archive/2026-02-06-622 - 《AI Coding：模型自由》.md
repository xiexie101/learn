# 622 - 《AI Coding：模型自由》

AI 编程最重要的是：模型自由。

前一篇 [621 - 《AI Coding FAQ》](https://sorrycc.com/ai-coding-faq) 也说了，模型相较于工具会更重要一些。这个时代，大家对于工具的忠诚度很低，谁有便宜好用的模型，就会用谁，除非工具太 la，典型的就是 Antigravity，有点浪费这么好的模型资源。

## 先随便写点

1、模型直接决定产出。举个例子，一个人有用不完的 Opus 4.5，一个人只有 Claude Code $20 餐套，一个人只能用国产模型。他们的产出效率和质量肯定是不同的。

2、关于御三家 Anthropic、OpenAI 和 Gemini 的模型。个人体感以及社区大多数人的体感肯定是 Opus 4.5 最佳，然后是 Codex 5.2 Codex，最后是 Gemini 3 Pro 。Codex 的有点玄学，可能和用法以及工具和提示词关系比较大，好坏评论两级分化比较大，OpenClawd 作者也比较推这个。

3、国产模型看社区风评，目前是 Kimi K2.5 > Minimax M2.1 > GLM 4.6 > 其他。个人用的比较少，只是作为辅助模型用。不过说回辅助模型，我个人用 Gemini 3 Flash 偏多一些。

4、虽然同是 Opus 4.5，各个渠道还是有差异的。比如做了 context limit 限制，有些甚至没有 think 能力，比如有些没有 search 能力（这个无关紧要，有需要加 search skill 就好），比如中转的更恶心拿其他模型代替。个人用过官方、Antigravity、Cursor 的，都不错，其他用的较少。

5、要有备份，一个渠道挂了可以用另一个。

## 那么，怎么做到模型自由

要么投钱，要么投精力。

1、如果你有钱，直接无脑 Claude Code Max + Google AI Pro Ultra 即可。我有些朋友就是 Claude Code Max 100$ + Google AI Pro Ultra + Cursor Plus 的组合，每个月 200$ 左右，非常省心。

2、如果想多花精力省点钱，那么可以关注各种薅羊毛的机会和活动。

比如。

Google Ultra 之前便宜的时候每个月 $40 还是 $60  
Google 的学生羊毛  
OpenAI 的老兵、教师、Team 羊毛（Team 有 Pro 模型额度）  
Lennysproductpass 最早上的时候是包含 Cursor 包年会员的，不过现在没了也还是很值，有 Amp、Factory、Manus、Perplexity、Warp 等  
Kiro 不定期会有送 Credit 活动

以及开源模型相关的。

iflow 有各种免费模型  
qwen oauth 可以用 qwen 3 coder plus  
cerebras 搞一些账号可以免费用 glm 4.7（注：context limit 免费版有阉割，但胜在速度快）  
nvidia 和 canopywave 也有免费额度

3、我目前的搭配是。

主模型：  
Claude Code 20$ 订阅（基本闲置，浪费）  
Antigravity 账号 x N（主力，因为我用 Neovate Code 开发，需要 API 形态的 Opus 4.5，用法见 [621 - 《AI Coding FAQ》](https://sorrycc.com/ai-coding-faq)）  
Cursor 年付会员（来自 Lennysproductpass）

辅模型（主打快，比如用来提 commit 和生成 summary 等）：  
Cerebras 账号 x N  
iFlow 账号 x1，用 glm 4.7 和 qwen 3 coder，[kimi k2.5 在路上](https://vibex.iflow.cn/t/topic/2664)

还有些备用的：  
ZenMux + OpenRouter 里都有不少 Credit  
Warp + Amp + Factory 年付会员（来自 Lennysproductpass）  
Github Copilot 额度（做开源送的）  
Kiro、Kilocode 等都有不少余额  
公司内部的模型渠道  
Google 送的 10$/月 x N

应该还有其他的，想到了再补。。
