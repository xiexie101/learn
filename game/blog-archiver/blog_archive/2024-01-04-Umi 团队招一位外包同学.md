# Umi 团队招一位外包同学

![](https://img.alicdn.com/imgextra/i1/O1CN01eafMQE1DgFwzgd5Ed_!!6000000000245-0-tps-1920-1280.jpg_1200x1200.jpg)

\> 2024.02.20 更新：已找到合适的同学。

找了挺久没找到合适的同学，之前拿到的简历质量都不太好。我试着自己找下简历，看能不能给 Bigfish/Umi 团队找到合适的外包同学。

**基本情况：**

1、Bigfish 团队有两名外包，由于之前给其中一位外包同学申请转正成功，目前要补位一位外包同学，我来带  
2、外包公司是佰钧成，薪资是外包岗 W3-W4 的水平  
3、工作地点在杭州飞天园区  
4、工作内容是负责 Bigfish 框架的答疑和日常维护  
5、不需要写 CRUD，不做业务，每周仅有周四上午 1 小时的周会，「通常」不需要加班  
6、工作挑战性看你个人的上限

**要求：**

1、熟悉 React  
2、熟悉 Node、Webpack、Babel 等  
3、编码能力过关，面试过程中肯定会有现场写代码的环节  
4、沟通能力过关，应该要做答疑的工作，需要有点耐心，同时能讲清楚事情  
5、学习能力过关，因为很多分配的任务可能都是你不了解的

**个人感觉：**

1、比较适合工资本身就是这个 Level 的同学，或者做腻了 CRUD 业务愿意稍微降点薪过来的同学  
2、沉下心来做几年，又比较努力的话，技术提升肯定会很快的，来源有几点，1）每天要面对不同的 Bigfish 问题，2）我这边也会有框架、组件、工具侧的各种需求。
