# 545 - 《mdcopilot》

> 群里有同学需要，就把 mdcopilot 也开源了。

1、这是解决我自己需求的一个工具，因为我有收集文章的习惯，看到好的文章就会把它记录下来，并且希望把这些文章分享给别人。但是只有 URL 会略显单薄，所以要加上标题和总结的信息，让大家阅读的时候，可以大致了解本章的内容，然后决定是否深入去阅读。mdcopilot 是一个 Markdown 的辅助工具。它可以把链接列表转换成包含总结的格式，效果先下图。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/mdcopilot2.gif)

2、背后的实现比较简单。先解析 Markdown，分析出要做转换的 URL 列表，然后把 URL 交给 [https://r.jina.ai/](https://r.jina.ai/) 去获取内容，然后提取标题，并且让 AI 去做总结，最后再写回到 Markdown 里。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/mdcopilot-flow.jpg)

3、地址是 [https://github.com/sorrycc/mdcopilot](https://github.com/sorrycc/mdcopilot) 。

4、如何使用。

选个好点的模型来做总结，比如 Claude 3.5 Sonnet、Open AI 的 ChatGPT 4。省钱的话，可以用 Groq 的 Qwen QwQ 32B。

```bash
$ OPEN_ROUTER_API_KEY=your_key mdcopilot foo.md --model OpenRouter/anthropic/claude-3.5-sonnet
```
