# 330 - 《星球 Next》

脑暴了下星球下一步的想法，计划如下。

1、日更目前平均每周 3 篇，期望是提升回每周 4 篇。

2、日报。YingCi 不更后我继续吧，佛系更新，比如一天没凑够条数或者没时间看资讯，就延后后一天更新。

3、新增一个 TOKEN，基于「成员编号」和「用户昵称」计算得出。和星球到期时间同步，为了避免退费期的麻烦，加入星球 3 天后可用。基于这个 TOKEN，可以享受一些额外在线服务。

额外在线服务可能包括，

1）日更文章的 Web 访问方式，提供列表和搜索功能，支持 RSS 的方式订阅。  
2）某 GPT 3 和 4 免费用，现在 GPT 4 通过 Telegram 群也能直接访问用，量不大，费用完全可以接受  
3）深度好文 RSS  
4）深度好文中英对照翻译存档  
5）几个我在用的 Twitter List 信息源的 RSS 服务，加点赞超两百后的过滤，带中文翻译  
6）基于 Deepl 的翻译服务和自用很久的双语翻译插件（社区好像已经有更好的方案了？）  
7）社区有价值新书分享，新的书和在线资料我基本都会无脑入（不喜勿入）

4、同时提供一些免费的额外服务。

1）简历评估，会告诉你简历中存在的问题和建议的方向，总数每周 <= 3 次  
2）应邀分享，可线上也可线下，线下通常得周末，和个人时间不冲突即可，异地报销路费和酒店费用，总数每周 <= 1 次  
3）问题的解，对于描述清楚的问题，提供我对此的解题思路，超出我知识面的会无法回答，但可能会帮你转问合适的同事或朋友，总数每周 <= 3 次

5、定期组织星球范围的分享交流会。还没想好怎么组织，改天找裕波咨询下方案。

6、适当运营。

1）每周从 3 - 4 篇里挑一篇发公众号，用以拉新。  
2）把 MDH 周刊捡回来，周刊下方加广告。  
3）合作和广告。

7、微信朋友位。每个来自星球的我都会备注成「星球 - xxx」，可以日常交流，但回复可能不及时，我的手机常年开勿扰的。
