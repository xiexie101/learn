# 49 - 《新入的几个 App》

过年时间有整理 App，有退役的也有新增的。以下是一些新加入工具箱的，以及我仅有几天的使用经验。

1、滴答清单 | 任务管理  
2、iPic | 图片上传  
3、OmniOutliner | 大纲工具  
4、Logseq | 大纲工具  
5、Eagle | 图片管理  
6、Shottr | 截图工具  
7、Anki | 卡片工具  
8、IA Writer | Markdown 编辑器

滴答清单有好多大佬推荐，我也跟风试试，用来代替之前的任务工具 NotePlan。Pro 版本 ￥167/年。端支持很全，我用 iOS + Mac 端。有清单、标签、过滤器、日历回顾等，感觉要配成 OmniFocus 的功能都没问题。

iPic 用于图片上传，免费版的 weibo cdn 够用，有其他上传源需求的可以花 ￥60/年 解锁 Pro 功能。加入之后，写作中的图片处理流程变成了，1）复制图片到粘贴版 2）执行 iPic 快捷键 Command + Shift + U 3）在文章内粘贴其 Markdown 格式。

图片上传还探索了另一个方案，基于 upgit 封装了一个 shortcuts，upgit 在 [https://mp.weixin.qq.com/s/gFDJDf1I8T9Ko3qP3g8EFQ](https://mp.weixin.qq.com/s/gFDJDf1I8T9Ko3qP3g8EFQ) 中有介绍，Typora 中的图片上传工具，支持上传到 github 和 [sm.ms](http://sm.ms)。封装成 shortcuts 后，和 iPic 的上传流程差不多，只不过上传源换成了 github，仓库见 [https://github.com/sorrycc/images](https://github.com/sorrycc/images) 。

OmniOutliner 属于大纲工具。最近发现大纲对于结构化的信息归纳非常有用，他比表格更灵活，比思维导图又更具表达力。比如图 1 是我新整理的在用的工具。

原本计划用 OmniOutliner 整理完整的信息结构，谁知遇到了 LogSeq。OmniOutliner 虽然很好，但有个缺点是文件级的，比如 A 大纲和 B 大纲之间不能互通。所以在还没有完全深入 OmniOutliner 时，就断然转向到 LogSeq 了。

LogSeq 是开源的免费软件。我对他的理解是无限大纲工具，基于 Markdown 格式，并且任意节点可断开编辑，还支持双向链接等。我基于 P.A.R.A 分类法整理了自顶向下的信息结构，然后纳入 LogSeq 的大纲中，遇到新的信息，找到自己的坑位填进去就好。我的分类见图 2。

Eagle 是图片收集软件。买断制，$29.95。专业的事交给专业的软件来做，尤其是从网页收集的体验非常好，安装 Chrome 插件后，用鼠标拖拽，一个步骤完成收集任务。

Shottr 是最近的截图软件的黑马。免费，尺寸仅 1.3M。建议看下官网视频介绍。CleanShot X 我暂时还留着，过几个月熟悉 Shottr 后估计会删。

Anki 是卡片工具。还没具体用，只是看过介绍知道如何用了。可以拿来背单词、温书等，做一切加深记忆的事。

IA Writer 我用来当做 Markdown 编辑器。比如编辑 MDH 周刊，见图 3。
