# 560 - 《AI Tips（9）》

> 上一篇 [552 - 《AI Tips（8）》](https://sorrycc.com/ai-tips-08)。

1、 [AIHubMix](https://aihubmix.com/) 有 gpt-image-1 接口，试了下图生图，一张 $0.25，还挺贵的。所以，画的多的话，还是买 ChatGPT 会员划算。

2、Notebook LM 支持生成中文等 50 国语言的播客，早上看到时顺手为今天内部的 NeoCoder Cli（Takumi 内部版）生成了一份播客，见 [https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/Introducing NeoCoder CLI.mp3](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/Introducing%20NeoCoder%20CLI.mp3) 。如果要做音频可视化，可以尝试 [Specterr Music Visualizer](https://specterr.com/music-visualizer/)、[Musicvid](https://musicvid.org/)、[echowave](https://echowave.io/) 或 [Veed](https://www.veed.io/) 。

3、[16x prompt](https://prompt.16x.engineer/) 和 [16x eval](https://eval.16x.engineer/) 。前者是 prompt 管理，后者是 prompt 评估。还没具体用过，但要开发 code agent，prompt 评估应该是绕不过的，准备找时间试试。[paradite/eval-data](https://github.com/paradite/eval-data) 下有相关的示例数据。

4、[kortix-ai/suna](https://github.com/kortix-ai/suna)，开源的通用 Agent，类 Manus 。由后端 API、前端界面、Agent Docker 和 Supabase 数据库四部分组成。

5、[VoltAgent/voltagent](https://github.com/VoltAgent/voltagent)，开源的 AI Agent 框架，用于构建和编排 AI Agent，基于 TypeScript 。看着有点类似 [mastra-ai/mastra](https://github.com/mastra-ai/mastra) 。

6、[ChatGPT Degrade Checker](https://chromewebstore.google.com/detail/chatgpt-degrade-checker-%E9%99%8D/inidgeckbobnafenlmlgfbeoijiamepm?hl=en) 据说可用于检查 ChatGPT 是否降智，效果未知。也可以试试 [KoriIku/chatgpt-degrade-checker](https://github.com/KoriIku/chatgpt-degrade-checker) 这个油猴插件，通过拦截 fetch 解析 `data.proofofwork.difficulty` 来判断。

7、几个 mcp 相关的库。1）[open-webui/mcpo](https://github.com/open-webui/mcpo)，用于把 mcp server 转成标准 Restful 的 OpenAPI server，2）[f/mcptools](https://github.com/f/mcptools)，一个 cli 形式的 mcp inspector 。

8、Google 发布了[《Agent 白皮书》](https://www.kaggle.com/whitepaper-agent-companion)，正在用 notebooklm 辅助阅读，感觉一般般，内容对我吸引力不强。大家可以听我转的[音频版](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/Agents_Companion.mp3)。

9、针对 Github 仓库产品，1）[DeepWiki](https://deepwiki.com/) 把仓库转成文档应该不用多介绍了，2）[The-Pocket/PocketFlow-Tutorial-Codebase-Knowledge](https://github.com/The-Pocket/PocketFlow-Tutorial-Codebase-Knowledge) 可以把仓库转成教程。

10、近期的新模型，1）[Qwen 3](https://qwenlm.github.io/blog/qwen3/)，MOE 模型，内部服务已经用起来了，2）[Xiaomi Mimo](https://github.com/XiaomiMiMo/MiMo)，推理模型，还没有试过，3）[Vidu Q1](https://www.vidu.cn/)，高性价比的视频模型，已接入飞书多维表格，可以批量生成视频，4）[DeepSeek-Prover-V2-671B](https://huggingface.co/deepseek-ai/DeepSeek-Prover-V2-671B)，5）即梦 3.0，6）[Dia](https://github.com/nari-labs/dia/)，开源 TTS 模型，将文字脚本转换成对话音频，但不支持中文。

11、[前端网页生成优化 Prompt](https://xiangyangqiaomu.feishu.cn/wiki/QX49wa9FAiwIPjkuslSc13EznDh)，收藏好久了，还没有试过。一直看到有人用这种方式生成一些资料的 Html，都有些审美疲劳了。

12、[QuivrHQ/MegaParse](https://github.com/QuivrHQ/MegaParse)，可以解析 PDF、PPT、Word 等，转成 LLM-ready 的格式。

```python
from megaparse import MegaParse
from megaparse.parser.unstructured_parser import UnstructuredParser

parser = UnstructuredParser()
megaparse = MegaParse(parser)
response = megaparse.load("./example.pdf")
megaparse.save("./output.md")
```
