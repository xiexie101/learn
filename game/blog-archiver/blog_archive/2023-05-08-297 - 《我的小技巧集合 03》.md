# 297 - 《我的小技巧集合 03》

> 每凑够 10 条发一篇。

*   Benchmark 方法，用 brew 装个 hyperfine，然后 `hyperfine --runs 10 'umi -v'` 就可以看到 umi -v 跑 10 次的性能结果。
*   如何用 Mid Jounery 生成 3D 头像？先传自己的照片上去，然后基于此加 prompt「simple avatar, pixar, 3d rendering, flat blue gradient background --s 500 --v 5」，接着就是体力活了，多生成一些，比如 100 个，总能挑到有点像的。
*   用 Reeder 订阅 RSS 的源通常有两种选择，1）远程的比如 Feedly，2）本地。建议前者，因为后者有个问题是，当一个 Feed 有数量限制而本地一段时间没更新时，中间的那部分会漏掉。
*   手机微信切换账号或退出账号时如何让电脑上登录的微信不退出？关闭手机网络点击退出账号，等一分钟左右跳转到登陆页面登陆大号就可以。亲测有效。
*   入手 Surge iOS，刷推速度能明显感觉比用 ShadowRocket 时快。
*   App Store 美区[可以用支付宝购买礼品卡](https://zhuanlan.zhihu.com/p/591446073)，[操作非常简单](https://www.youtube.com/watch?v=W3chc223K-w)。目前只有在美国地区提供礼品卡购买服务，由服务商 [Pockyt](https://pockyt.io/) 提供，通过支付宝付款。
    *   直接打开这个地址购买？支持支付宝和微信。 [https://shop.pockyt.io/product?brand=App Store %26 iTunes US](https://shop.pockyt.io/product?brand=App%20Store%20%26%20iTunes%20US)
*   如何一键识别屏幕中的二维码？先安装依赖 `brew install zbar`，然后执行命令 `osascript -e 'tell application "System Events" to set visible of process "iTerm2" to false' && screencapture -i /tmp/abc.jpg && zbarimg /tmp/abc.jpg | sed -e 's/scanned.*//g' -e 's/QR-Code://g' | pbcopy && rm /tmp/abc.jpg`，来源 [https://utgd.net/article/20140/](https://utgd.net/article/20140/) 。
*   重要的群，置顶，免打扰，定期看看消息；其他群，全部免打扰，甚至退出。
*   悲观者往往正确，乐观者往往成功。
*   一个[时间管理的技巧](https://world.hey.com/dhh/how-to-have-buckets-of-time-38693993)。即将相关任务放入一个桶中，让这个桶积累直到满，然后一次性处理。这与试图在数不清的打断管道中捕捉每个任务形成对比，如果你让它们滴答滴答地流逝，它们会将你的一天淹没。作者强调，时间并不是稀缺资源，注意力才是，找到你的桶，培养耐心让它们填满，然后逐个清空。这是 10 倍生产力的窍门。
