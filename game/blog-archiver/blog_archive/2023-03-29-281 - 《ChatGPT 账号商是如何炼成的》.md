# 281 - 《ChatGPT 账号商是如何炼成的》

不说太细，大家可以自己理解下，感兴趣的可以自行尝试下。有人做这个赚了几百万，比如选印度尼西亚地区，一个账号成本 22.5 卢布（非批发），相当于人民币 2.01 元。目前虽然红利期已过，但感觉账号长期来看还是强需求，因为 TOKEN 有过期时间，过段时间就得刷一遍新账号。

基本流程是这样。

1、跑通账号注册 SOP（标准作业流程）  
2、利用「RPA」把上述 SOP 自动化，基本上 1 分钟一个（程序员自己写程序应该也算在 RPA 之内吧）

RPA 的流程是这样。

1、得先有临时邮箱，临时邮箱有大量方案，试了下之后我选择的方法是利用 cloudflare 的 email router worker 实现自动提取（晚上写了下已跑通，发送邮件后可通过 API 拿到邮件内容）  
2、接码平台 API 化，sms-activate 支持 API 访问的，传 token 即可  
3、利用 Puppeteer 把以上流程串起来

一些注意点。

1、ChatGPT 账号本身不支持修改密码，所以如果你如果要卖账号，客户的感受度可能不太好。但是，ChatGPT 注册时是可以选择用 Google 账户、微软账户绑定的方式，这种就可以改密码了，比如微软账户，改微软账户的密码即可  
2、如果选微软账号这条路，RPA 步骤会复杂不少，有个校验码的环节我试过，暂时没找到通过机器绕过的方法  
3、之前 ChatGPT 会给 18$ 的免费 TOKEN 额度，现在只给 5$ 了  
4、有个 120$ 的配额，可以通过填写配额申请表来增加配额，官方收到申请后，会在 1-2 个工作日回复  
5、如果你要卖账号，自动发货可以用「阿奇索」  
6、如何引流卖账号，就不展开了…
