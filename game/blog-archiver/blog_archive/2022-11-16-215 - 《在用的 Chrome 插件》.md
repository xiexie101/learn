# 215 - 《在用的 Chrome 插件》

前几天有人问，整理下，以下是目前还在用的。链接就不贴了，google 下第一个就是。

*   [Bypass Paywalls](https://github.com/iamadamdev/bypass-paywalls-chrome)，绕过一些站点的付费墙，比如 medium
*   TWP Fork 的 Chrome 版，用于产出中英对照翻译，翻译基于 Google，但我自研脚本后就没用这个了，地址是 [https://github.com/immersive-translate/immersive-translate](https://github.com/immersive-translate/immersive-translate)
*   1Password
*   Deepl Translate，官方 Chrome 插件
*   Eagle for Chrome，用于把图片保存到本地的 Eagle 软件中，顺便推荐下 Eagle，功能和使用体验感觉都是做到了极致
*   Enhanced GitHub，显示文件尺寸并提供下载链接
*   GitHub Issue Link Status，给 pr、issue 的链接按不同的状态增加颜色区分
*   Modern for Wikipedia，让 Wikipedia 好看点
*   Neat URL，移除链接后面的无效参数，但感觉做地不够好，因为有不少场景漏掉的
*   NopeCHA: CAPTCHA Solver，校验码输入辅助，想法很好但感觉用处不大
*   Notion Web Clipper，收集文章到 Notion，好久没用了
*   React Developer Tools
*   Redirector，配置规则让访问 A 页面的自动 redirect 到 B 页面
*   Redux DevTools
*   RSSHub Radar，检测当前访问的网站是否支持通过 RSSHub 订阅
*   Tampermonkey，油猴，神器
*   Twemex: Sidebar for Twitter，Twitter 扩展，神器
*   XSwitch，开发工具，用于代理请求
*   YouTube™ dual subtitles，Youtube 双语字幕，神器
*   壹伴，转载别人文章时临时开一下使用，通常用于发广告文章
