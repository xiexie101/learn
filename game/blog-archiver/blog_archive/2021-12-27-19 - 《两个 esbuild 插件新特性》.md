# 19 - 《两个 esbuild 插件新特性》

esbuild 重要性不必多说，插件开发肯定少不了。最近的两个插件特性记录下：

1、0.14.3 - Pass the current build to JS Plugins  
[https://github.com/evanw/esbuild/releases/tag/v0.14.3](https://github.com/evanw/esbuild/releases/tag/v0.14.3)  
插件代码中可以直接访问 esbuild 实例对象。

```ts
let p = { name, setup(build) { build.esbuild.transformSync('1+2') } }
```

2、0.14.8 - Add a resolve API for plugins  
[https://github.com/evanw/esbuild/releases/tag/v0.14.8](https://github.com/evanw/esbuild/releases/tag/v0.14.8)  
插件中可以解析文件路径。

```ts
let p = { name, setup(build) { build.resolve('./foo') } }
```
