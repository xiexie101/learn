# 549 - 《AI Tips（7）》

> 总结提示词、Dia 和 SumBuddy、MacWhisper、代理抓提示词、Lennysnewsletter、新模型等。

1、**总结提示词**。AI 一个很大的用途是提效。当你阅读一篇长文的时候，AI 可以帮你缩减获取到其中关键内容的时间。但是，怎么写这个提示词呢？一个简单的「用中文总结」是否够用？以下是我目前在用的几套总结提示词，欢迎评论补充你的。

1）当你要详细了解一篇文章的时候，可以用这个，它会给出非常详细的总结。

```
你是一位资深新闻编辑，请对这篇文章进行专业的综述。请遵循以下要求：

# 综述格式
1. 使用Markdown格式输出
2. 注明原文发布时间

# 内容要求
1. 准确提炼每篇文章的核心论点和关键信息
2. 突出重要的数据、引用和具体事实
3. 保持客观中立的叙述语气
4. 按照文章在原文中的顺序进行综述
5. 确保对每篇文章都进行完整的总结

# 注意事项
1. 直接输出综述内容，不要加入任何与综述无关的回应性语句
2. 保持专业的编辑视角，注重新闻价值的提炼
3. 适当保留原文的叙事结构和重要细节
```

2）当你只想知道一篇文章大概内容的时候。

```
你是美国一线的资深开发者，在不遗漏文章重要信息的前提下，请将文中的所有内容，整合成一篇简报形式的摘要。简报的格式为：

第 1 自然段：总结，但不要展开细节。
第 2 自然段：根据这篇文章，提供面向开发者的决策建议。

注：请直接输出正文，不要在开头输出响应我服务的回应。
```

或。

```
你是美国一线的资深开发者，请总结这篇文章。

- 用中文总结。
- 100 - 300 字，不要超过 300 字。
- 不要换行，如有列表项需要换行，请以 1), 2) 的方式让他归为一类，总结里不要出现多行代码。
- 不要用太官方的口吻。
- 注意盘古之白（中文和字母，中文和数字之间需要空格）
```

2、**Dia 和 [SumBuddy](https://chromewebstore.google.com/detail/sumbuddy-ai-assistant/knpckkifmkioijpoejgngbghdpacfajp?pli=1)** 。至于怎么把总结提示词有效的利用起来，我这里有用到两个工具。Dia 是新出的浏览器，应该不少人都有邀请码，如果大家没有可以问一下，或者闲鱼上买一个，他对我来说最大的功能不是浏览器，而是一个总结工具。SumBuddy 这是一个 Chrome 的插件，配置上大模型和总结提示词之后可以达到和 Dia 类似的效果。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/4gA8jz.png)

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/jqoo0E.png)

3、**[MacWhisper](https://goodsnooze.gumroad.com/l/macwhisper)**。最近比较沉迷在语音录入这一块，所以遇到合适的工具都会尝试一下，MacWhisper 是新看到的这个类别的工具。它的特点是，1）功能比较强大，支持语音文件的 Transpile、翻译等各种功能，2）Transpile 用的是本地的 whisper 模型，好处是安全，缺点是对电脑性能会有一定的要求。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/macwhisper-2.gif)

4、**用代理的方式抓取提示词**。大家应该都知道，提示词通常是一个工具的核心之一，这也是为啥 Cursor 把部分提示词隐藏到服务端的原因。当你想知道一个客户端工具的提示词时，你可以通过抓包的方式，如果他支持配置大模型，你也可以在本地搭一个 OpenAI 的代理服务器（AI 可以帮你写），通过日志的方式抓取到提示词。我抓取 ChatWise 这个 App MCP 功能提示词的时候，就用到了后者。

5、**Branch from here**。AI Studio 新上了个「Branch from here」的功能，可以让你从聊天的某一条记录开始「Fork」一份出去接着聊。挺实用的，感觉后续应该会有很多 Chat App 抄这个功能。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/KQmpVt.png)

6、**「元宝」联系人**。感觉微信在大力元宝上真实不遗余力啊。现在已经可以在微信里把元宝添加为联系人了，你可以随时随地在微信上跟元宝聊天。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/PgS87z.png)

7、**Veo 2**。Google 的 Veo 2 可以做 text to video 或 image to video。可以[通过 API 的方式](https://www.xiaohu.ai/c/xiaohu-ai/google-veo-2-api-gemini-api)调用，但是收费比较贵。现在也可以在 Google 的 AI Studio 上免费使用。我试了几个帐号，有些有，有些没有，应该是灰度开启了。

8、**Lennysnewsletter 的 AI Bundle**。这个包太划算，我昨天看到就买了，单一个 Cursor 就值回票价。1）单买 $200，2）组团买每人 $170，买完还可以加人，3）教育优惠之前 $100，现在也是 $170，4）应该是拉新活动，需要未注册过会员的账号或者新账号。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/uycN7O.png)

9、**新模型**。近期值得一试的有 GPT 4.1、O3、O4-Mini、可灵 2.0、智谱 GLM-Z1-Air-32B 。感觉开发 AI Agent 的一大乐趣就是去接入、体验和测试新的大模型。
