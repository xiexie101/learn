# 469 - 《VSCode 快捷键技巧》

> 通过 , 前缀的方式定义额外的快捷键，比如在侧栏打开定义代码、在 github 打开当前文件、打开当前项目的 package.json 等。

1、上午学到的一个技巧，感觉有用。方法是通过 keybindings.json 定义一些额外的常用快捷键，具体定义啥看个人喜好了，我的全部配置如下，可以直接抄作业。

```json
// keybindings.json
// e = edit
// o = open
[
  { "key": ", ,", "command": "type", "args": { "text": "," } },
  { "key": ", .", "command": "editor.action.revealDefinition" },
  { "key": ", e c", "command": "editor.action.transformToCamelcase" },
  { "key": ", e m", "command": "editor.action.sourceAction", "args": { "kind": "source.addMissingImports", "apply": "first" } },
  { "key": ", e r", "command": "editor.action.sourceAction", "args": { "kind": "source.removeUnusedImports", "apply": "first" } },
  { "key": ", e s", "command": "editor.action.sortLinesAscending" },
  { "key": ", o g", "command": "gitlens.openFileInRemote" },
  { "key": ", o k", "command": "workbench.action.openGlobalKeybindingsFile" },
  { "key": ", o t", "command": "workbench.action.terminal.toggleTerminal" },
  { "key": ", o p", "command": "workbench.action.tasks.runTask", "args": "Open package.json" },
  { "key": ", r", "command": "workbench.action.tasks.runTask" },
  { "key": ", shift+,", "command": "type", "args": { "text": "，" } },
  { "key": ", shift+.", "command": "editor.action.revealDefinitionAside" },
  { "key": ", shift+space", "command": "type", "args": { "text": "，" } },
  { "key": ", space", "command": "type", "args": { "text": "," } },
  { "key": ", w", "command": "editor.action.toggleWordWrap" },
  { "key": "d", "command": "deleteFile", "when": "filesExplorerFocus && !inputFocus" },
  { "key": "n", "command": "explorer.newFile", "when": "filesExplorerFocus && !inputFocus" },
  { "key": "r", "command": "renameFile", "when": "filesExplorerFocus && !inputFocus" },
]
```

```json
// tasks.json
{
  "version": "2.0.0",
  "tasks": [
    { "label": "Open package.json", "type": "shell", "command": "code-insiders ${workspaceFolder}/package.json", "problemMatcher": [], "group": { "kind": "none", "isDefault": true } }
  ]
}
```

2、有几招挺实用的。1）按 `, .` 跳转到定义，然后按 `, shift+.` 把定义项目在侧边栏打开，2）按 `, e m` 添加 missing 的 import，比如在代码里写 assert，然后按这个快捷键，可自动添加 `import assert from 'assert'` 的 import，然后按 `, e r` 会删除没有用到的 import，3）按 `, o g` 把文件在 github 打开，对于做开源的同学会有点用。

3、这个方案的缺点是会影响 `,` 的输入，你可以换其他的，也可以定义额外输入 `,` 的快捷键，比如 `, space`。

4、需要注意的是，keybindings.json 里不能用 `${workspaceFolder}` 这种预定义变量，比如我要一键打开根目录的 package.json 就不行。变通的方法是在 tasks.json 里定义任务，然后在 keybindings.json 里执行任务。
