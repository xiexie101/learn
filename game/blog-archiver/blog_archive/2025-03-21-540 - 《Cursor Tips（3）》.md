# 540 - 《Cursor Tips（3）》

> 这个系列我换一种更新方式。因为每次如果没有即时更新，这个系列拖得比较长，很多 tips 感觉会过时，并且不能及时地推荐给大家。新的方式是：我遇到一个好的 tips，就会更新上来，然后持续更新这篇文章。等凑够九条之后，再新开一篇。

> WIP。

1、推荐一个 [Cursor Stats](https://marketplace.visualstudio.com/items?itemName=Dwtexe.cursor-stats) 插件，可以让你在状态栏看到当前账户剩余可以调用的次数，感觉有点焦虑，因为我用得太少了，但我这个用的量一个月都用不完 500 个调用。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/snhtDE.png)

2、如果你发现 Markdown 没有 Code Complete 提示了，鼠标移到状态栏上的「Cursor Tab」，关掉「Disable for Markdown」，我被这个坑了好几天，还以为是 Code Complete 在 Markdown 里不能用了。

3、用 Claude Code 的 init 命令可以帮你生成项目的基本信息。这个信息包含代码风格、常用代码、依赖信息等。可以在后续过程中让大模型更好地理解你的项目。见 [https://x.com/al3rez/status/1899777132148170902](https://x.com/al3rez/status/1899777132148170902) 。另外，[539 - 《takumi》](https://sorrycc.com/takumi) 也有提供 init 命令，通过 `OPEN_ROUTER_API_KEY=YOUR_KEY npx -y takumi init --model OpenRouter/anthropic/claude-3.5-sonnet` 执行即可。

4、Cursor 0.49 如何下载？

Cursor 是灰度逐步更新的，想尝鲜的需要手动下载，见 [https://ucnxwkbe679c.feishu.cn/wiki/FOJIwct17i6Y7FkMhh1cihyDnJb](https://ucnxwkbe679c.feishu.cn/wiki/FOJIwct17i6Y7FkMhh1cihyDnJb) 。

同时你可以在 Cursor Setting 里更改「Update frequency」配置项为「Early Access」，以便之后更早收到更新。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/yzuQok.png)

5、Cursor 0.49 新特性。

*   新增 ‎`/Generate Cursor Rules` 命令，用户可直接在对话中生成规则，便于定义 Agent 的行为。
*   新增 `Show Chat History`，可以快速访问历史记录。
*   MCP 支持图片作为上下文输入。
*   Agent 模式下的 Terminal 增强，可以在执行之前「编辑或跳过命令」。
*   允许配置全局的 Cursor Ignore，Settings 里搜「Global Cursor Ignore List」，已经默认配置里不少，包括 `**/.env` 等安全相关的文件。

6、修改「Generate in Terminal」快捷键，默认是「⌘+K」，和 iTerm2 下用于「清理屏幕」的快捷键冲突，会很不习惯，我改成了「⌥+K」。
