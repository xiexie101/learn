# 427 - 《云谦月报 2024.03》

> 列下流水账。

1、Mako 本月工作。1）收尾 2 月做的 build 重构，2）Smallfish 组件库支持，实现 style-loader 支持和 bugfix，3）从无玄那学习 XCode Instrument 后，找到一个 Mako 设计上的问题，4）Rolldown 发布后，学习源码，整理我们可以改进的点，空闲时间慢慢写 Toy Rolldown，5）写了 Mako 支持 MFSU 的 RFC，进度 10%，6）整理 Mako Analyze 等新手任务给 Mako 新同学，7）重构 binding 和 mako 入口模块结构，进度 80%，8）用 napi 3 alpha 重构 mako 和 binding 层的插件机制，进度 30%，9）例行发版、小 Feature、PR Review 和 Bugfix。

2、Bigfish 本月工作。1）梳理 Node 性能调试方法，整理 Umi/Bigfish Cli 提速方案的 RFC，细化了一些 Action，逐步实施中，2）应国际同学要求，他们会有其他国家同学入职，所以借助 AI 翻译了 Umi 和 Bigfish 文档的英文版，翻译质量不错，且支持增量更新，3）探索 Bigfish 项目如何写 CURD 更好的最佳实践，4）整理 Umi 5 的设计，进度 5%，只调研了一些路由层，5）外包同学杜杰来了 1 个多月，已平稳落地，在日常答疑之外，能负责一些 Bigfish 的开发工作。

3、报名了 GIAC 2024.5 深圳的一个分享，主题是《Rust 构建工具在蚂蚁的研发和落地》，已迭代了几个版本，在内部分享了 3 次，感谢数科、会员和马力线团队的邀请。

4、其他。1）读完一个作者的两本 Rust 书，《Code Like a Pro in Rust》和《Rust Design Patterns》，查漏补缺，2）写了个 Program Agent，日常可以帮我完善一些小功能，3）用便宜的 GPT-4 和 Deepl 翻译了好多本书，4）分享部门招聘广告，收到简历不少，过滤出 2 个内推，都没过，5）调整了下阅读文章的流程，存 markdown、翻译、阅读和高亮、补充个人想法、分享，6）一个新习惯，if 不下雨，早起在公司篮球 45 分钟，7）下个月公众号接了 [6 个广告](https://sorrycc.com/ad/)，得多发点文章对冲下，8）11 篇星球更新（太少）、3 篇 MDH Weekly 更新、16 篇译文更新。
