# 522 - 《AI Tips（3）》

522 - 《AI Tips（3）》

1、Cursor Rules 更新。1）全局规则在 Settings 配置，2）项目规则存储在 .cursor/rules 下，基于 description 和 glob 匹配文件自动附加，支持文件夹，3）.cursorrules 被弃用，之后会移除。我给 TNF 的 cursor rules 调整了下，见 [commit](https://github.com/umijs/tnf/commit/6f3f65da16bea8d94a3c97d630bfd38f120a5dc2) 。

2、意外发现 [ChatWise](https://chatwise.app?atp=sorrycc) 还挺好用，对我来说有两点。1）可以用 Github Copilot 的 API，如果你本身就有 Github Copilot 的服务，那等于好的 AI 服务免费用了，2）⌘+K 功能快捷输入 Prompt，挺方便的，而且还会自动贴入粘贴版里的内容。注：我没付费，感觉免费版够用，想要 web search 功能时我会用 ChatGPT Search 或 DeepSeek Chat。

![](https://tcc.sorrycc.com/p/20a9a1dd-b331-4066-a442-3338ba539bca.png)

4、飞书的 DeepSeek R1 + 多维表格还挺实用的。但是不想用飞书，所以有点想写个本地版，数据基于 JSON，通过 CLI 启动并自动编辑。
