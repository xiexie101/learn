# 410 - 《云谦月报 2024.01》

关于工作。

1、主线依旧是 Mako。发布了 0.4.x，包含系列功能更新和 Bugfix。比如，Tree Shaking Skip Modules 问题修复并重新开启、SWC 升级到最新、支持 Styled Components、完善 Node 支持、支持类 raw-loader 能力、支持 HMR 时调试 link 的 npm 包等；业务方面，对接了两个新的场景；工程化方面，建了数据大盘和 Benchmark 脚本；人员方面，本月投入度略有降低。

2、Bigfish SSR 和各方聊了下，会把之前内部试点项目的方案做成标准化的方案，整理出最佳实践，然后推给更多对性能有要求的业务。至于 RSC 和 PPR 的是否支持，还需要进一步考虑投入产出比。

3、外包补人投了不少时间招聘，陆陆续续也有几个月了，还是挺难的。对外也发布了 [Umi 团队招一位外包同学](https://sorrycc.com/umi-hire)。但发现大部分同学写不出「Todo List」或「倒计时组件」，有两位比较合适的，一位拿了其他 Offer，一位还在犹豫薪资。
