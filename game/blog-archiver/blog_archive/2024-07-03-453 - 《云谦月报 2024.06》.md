# 453 - 《云谦月报 2024.06》

## 工作

1、Mako 对外开源。本月大部分时间在[准备 Mako 的对外开源](https://github.com/umijs/mako/issues/1268)，包括文章预热、Feature 补全、补充文档、准备双语、准备官网、准备发布稿等，然后在 2024.6.28 [正式对外开源](https://makojs.dev/blog/mako-open-sourced)，以及在当日晚 9 点做了直播答疑。

![](https://res.cloudinary.com/sorrycc/image/upload/v1719929915/blog/6y5ndhtn.png)

2、Mako 业务应用。Mako 对外接入 Dumi 2.4.x、Father 4.5.0-rc 和 Umi 4.2.x；Mako 对内接入 Smallfish（RSC）并已公测，提速效果非常明显，冷启和热更都有 10 倍提升，见下图；Mako 中后台 Bigfish 应用数截止 2024.7.1 为 307 个；其他有接入 Ant Design、AntV、画眉等，yuyanAssets 和闪蝶的接入也在进展中。

![](https://res.cloudinary.com/sorrycc/image/upload/v1719930324/blog/30zy0vdn.png)

![](https://res.cloudinary.com/sorrycc/image/upload/v1719976250/blog/gf6sibgs.png)

3、本月值得关注的 Mako 功能。

1）Code Splitting 支持 Granular 策略  
2）Concatenate Modules 重构  
3）调整 Mako API  
4）多线程 Less Loader 支持 Node 低版本（16）
