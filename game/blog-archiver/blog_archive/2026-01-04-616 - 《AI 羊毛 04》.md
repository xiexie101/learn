# 616 - 《AI 羊毛 04》

1、随时跑路公益站。

地址是 [https://runanytime.hxi.me/](https://runanytime.hxi.me/) 。需要 [linux.do](http://linux.do) 2 级用户。基于 kiro 逆向的公益站，rpm 5分钟35次请求。每天可以到 [https://runanytime.hxi.me/console/personal](https://runanytime.hxi.me/console/personal) 签到和到 [https://fuli.hxi.me/wheel](https://fuli.hxi.me/wheel) 幸运转盘。

如果你用 @neovate/code，在 `~/.neovate/config.json` 里这么配。

```json
{"provider": {
    "runanytime": {
      "options": {
        "baseURL": "https://runanytime.hxi.me/v1/",
        "apiKey": "YOU_API_KEY"
      },
      "models": {
        "claude-sonnet-4-5-20250929-thinking": "claude-4-5-sonnet",
        "claude-haiku-4-5-20251001-thinking": "claude-haiku-4-5",
        "claude-opus-4-5-20251101-thinking": "claude-opus-4-5"
      },
      "createModelType": "anthropic"
    }
}}
```

然后执行。

```
neovate -m runanytime/claude-opus-4-5-20251101-thinking
```

参考：  
[https://linux.do/t/topic/1154353](https://linux.do/t/topic/1154353)

2、Canopywave 。

据说是英国的公司，但也有在小红书做推广。模型都是开源的模型。

访问 [https://cloud.canopywave.io/](https://cloud.canopywave.io/) 用 google 或 github 注册，新用户有 $20，然后用 XRSJR4Q5DD code 会有额外 $20，然后访问 [https://cloud.canopywave.io/model-api-key](https://cloud.canopywave.io/model-api-key) 生成一个 key 。

如果你用 @neovate/code，在 `~/.neovate/config.json` 里这么配。

```
{"provider": {
    "canopywave": {
      "api": "https://inference.canopywave.io/v1",
      "options": {
        "apiKey": "YOUR_KEY"
      },
      "models": {
        "minimax/minimax-m2.1": "minimax-m2.1",
        "zai/glm-4.7": "glm-4.7",
        "moonshotai/kimi-k2-thinking": "kimi-k2-thinking",
        "deepseek/deepseek-chat-v3.2": "deepseek-v3-2-exp"
      }
    },
}}
```

然后执行。

```
neovate -m canopywave/zai/glm-4.7
```

3、Nvidia。

开始免费提供 glm 4.7 和 minimax m2.1 。

地址是 [https://build.nvidia.com/models](https://build.nvidia.com/models) ，注册后要验手机号才能用 api key。glm 4.7 太慢，可以用 minimax m2.1，比较快。

如果你用 @neovate/code，在 `~/.neovate/config.json` 里这么配。

```
{"provider": {
    "canopywave": {
      "api": "https://integrate.api.nvidia.com/v1/",
      "options": {
        "apiKey": "YOUR_KEY"
      },
      "models": {
        "z-ai/glm4.7": "glm-4.7",
        "minimaxai/minimax-m2.1": "minimax-m2.1",
        "moonshotai/kimi-k2-thinking": "kimi-k2-thinking",
        "openai/gpt-oss-120b": "gpt-oss-120b",
        "qwen/qwen3-coder-480b-a35b-instruct": "qwen3-coder-480b-a35b-instruct"
      }
    },
}}
```

然后执行。

```
neovate -m canopywave/minimaxai/minimax-m2.1
```
