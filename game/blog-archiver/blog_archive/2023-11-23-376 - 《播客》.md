5、遇到干货播客想慢慢消化怎么办？

先按前一个步骤下载音频，然后传到「飞书秒记」里。见图：

![](https://img.alicdn.com/imgextra/i3/O1CN01FiuTOT1my7vyAvEun_!!6000000005022-2-tps-3834-1764.png)

如果遇到音频超过 4h 的，可以用 ffmpeg 拆下，因为飞书妙记限了 4h。

```
ffmpeg -ss 0 -t 03:59:59 -i input.m4a -acodec copy first.m4a
ffmpeg -ss 03:59:59 -i input.m4a -acodec copy left.m4a
```

其中 `-ss` 表示开始时间，单位为秒；`-t` 表示截取的时间长度，不传表示剩下所有的。
