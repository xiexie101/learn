# 528 - 《新买了个 VPS》

看了下，发现情人节促销还在，之前 CloudCone 机器到期没续，于是买了台 [CloudCone](https://app.cloudcone.com/?ref=10409) 的机器，3 Core 2G RAM 240G DISK，$24.99/YEAR。CloudCone 的优势是便宜，感兴趣的可以去看一眼 。

1、ssh。生成密钥对，上传公钥到服务器，添加 alias，方便后续登录。

```bash
# generate
ssh-keygen -t rsa -b 4096

# upload to server
ssh-copy-id -i ~/.ssh/key_file_name username@ip

# add alias
vi ~/.ssh/config
```

一个人用，非主力 vps，安全性要求没那么搞。加非 root 账户、启用防火墙、加日志监控之类的事就算了。

2、用 [https://ping.chinaz.com/148.135.122.9](https://ping.chinaz.com/148.135.122.9) 测速，没墙，速度感觉还不错。虽然相对于[搬瓦工的 VPS](https://bandwagonhost.com/aff.php?aff=74975)，速度肯定是没法比的，搬瓦工提供了 CN2 GIA 线路，[我的博客](https://sorrycc.com/)和 [Mako 官网](https://makojs.dev/)都部署在搬瓦工。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/sEvq9F.png)

3、安装 nezha 探针。Nezha 探针可以监控服务器的 CPU、内存、带宽、磁盘等资源使用情况。在管理界面，新增服务器，然后在「一键安装」下复制命令去机器上执行。稍等一会后即可生效。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/A4r2HZ.png)

4、安装 [1Panel 面板](https://1panel.cn/)。

```bash
curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && bash quick_start.sh
```

1）网站里安装 OpenResty（默认选项，OpenResty 通常用于搭建高性能 Web 服务器和反向代理）  
2）证书里创建 Acme 账户（Acme 协议是用于自动化证书申请）

5、安装 [Bitwarden](https://bitwarden.com/) 服务器。

1）1Panel 应用商店里搜「Bitwarden」点安装。背后是 bitwardenrs/server（现在叫 Vaultwarden），这是个非官方、轻量级的 Bitwarden 服务器实现，主打资源占用少。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/HMioPa.png)

2）配个域名指向这个机器，比如 [bitwarden.xxxx.com](http://bitwarden.xxxx.com) 。

3）网站建个反向代理，地址配上 127.0.0.1 + 第一步安装的端口号。

4）新建证书，为了可以通过 https 访问（试过 http 不行，有些字体加载不出来，会导致网站渲染失败）。

5）定期备份数据。执行一下命令搜「Mount」，会发现数据被保存在 /opt/1panel/apps/bitwarden/bitwarden/data 目录，定期备份就好了。

```bash
docker ls
docker inspect 1Panel-bitwarden-fkjA
```

DONE（注：搞完后洁癖犯了，花好几个小时整理了密码、银行卡和各种私密信息。）
