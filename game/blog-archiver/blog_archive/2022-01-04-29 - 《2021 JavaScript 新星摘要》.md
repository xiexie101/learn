# 29 - 《2021 JavaScript 新星摘要》

1、[zx](https://github.com/google/zx)，全局新增 Star TOP 1，实至名归，我最爱的脚本组合就是 zx + esno。  
2、[Tauri](https://tauri.studio/)，electron 竞品，也是用 web 写 desktop application，基于 rust，但不包含 node runtime，所以小。  
3、[Astro](https://astro.build/) 虽不在 TOP 10 内，但今年非常瞩目，0JS 方案让页面加载更快，和 SSG 类似，但关键的不同是 astro 按需可以引入动态交互的部分，这个「需」可以是多种策略的，比如页面载入、页面闲置或者基于 Intersection Observer 的页面可见。  
4、[Svelte](https://svelte.dev/) 今年新增 Star 数超过 Angular。  
5、[Solid](https://solidjs.com/) 也值得关注，和 React 一样用 JSX 写组件，但不依赖 Virtual DOM；同类的有 [Mitosis](https://mitosis.builder.io/)，可以把组件编译后兼容四大主流框架  
6、[Remix](https://remix.run/) 框架同样不得不提，slogon 是 “Web Fundamentals, Modern UX”，其 API 大量遵从 web 标准  
7、框架部分的还有，[Nest](https://nestjs.com/) 引领服务端 Node 框架，而 [Strapi](https://strapi.io/) 则引领 Headless CMS  
8、构建部分两大变化，1 是 Native ES Module，以 [Vite](http://vitejs.dev/) 为代表的，2 是大量非 JS 写的 [JS 工具](https://github.com/RobinCsl/awesome-js-tooling-not-in-js)，包含 SWC、esbuild、Bun、Turborepo、Parcel 2 等  
9、CSS in JS 部分，vanilla-extract 增长最快，也是我非常喜欢的一个方案

详见 [https://risingstars.js.org/2021/en](https://risingstars.js.org/2021/en) 。
