# 361 - 《读书笔记：习惯的力量》

1、什么是习惯？

习惯是机体核控制的一个自动运行的一个程序，大脑的其余部分已经不参与决策了。举一些我的例子。比如周四经常去去吃 KFC，周四 + 每周四上午开会来不及去食堂就是 cue，垃圾食品的糖和盐就是 reward；比如每天早上阅读资讯，感觉已经有获取新鲜事物的瘾（craving），每到那个时间，就想着了解发生了什么，然后不自主的打开 Reeder。

那习惯是怎么形成的？它是通过重复进行了组快化，组快化一旦完成习惯就形成，形成之后大脑就会保留这个回路。所以习惯有两个特点，第一，坏习惯的形成特别的轻易，比想象轻易得多。第二，一旦你这个坏习惯形成了以后，就永久存在，无法消除。好消息是好习惯也这样，所以我们要尽量培养好的习惯，占据所有的cue，把所有的 cue 给占住了之后，我们的坏习惯就比较不容易占领这个领地了。

那在这个意义上来说，记住一句话，勿以恶小而为之，勿以善小而不为。大家一定要看好自己的北极星，无论有没有人看见，无论现在的代价有多大，你都一定要坚持做正确的事情，因为习惯的养成是非常的重要的。

2、习惯的圈圈。

习惯的圈圈有三个元素，cue（暗示）、routing（行动）、reward（奖赏）。找到这三个元素分别是什么？尤其是找到你的 cue，找到你的 reward，有助于我们管理和改变习惯。

![](https://img.alicdn.com/imgextra/i4/O1CN01VKjM112ABANDD8Re2_!!6000000008164-2-tps-1262-684.png)

而在这三个元素中，最最最关键的那个核心要素其实是 craving（渴望）， craving 就是瘾，上瘾的那个瘾。craving 形成的原因是，当习惯的折腾在你脑子中越来越深，刺激点不再是在 reward 的那一刻，而是在 cue 的那一刻，出现 cue 的时候就兴奋，对 reward 有 craving，就有了瘾。从中我们会看到习惯的力量是多么的巨大，当你有了 craving 的时候，你就很难抗拒它了。但是我们可以通过两个方法来改善自己坏习惯对我们的影响，第一就是避开 cue，第二就是找到 reward 之后换一个 routing，达到同样的 reward。

记住，我们一旦被 craving 所控制，比如书中那个被 craving 所控制的猴子，它不会出实验室，它也不会去吃别的东西，它就一遍一遍的拉那个拉杆，记住你食物已经被放了毒，杠杆已拉杆已经被通了电，在这种情况下它还是会不断的去被电，吃这个食物，这是多么可怕的一件事情。记住这件事情，记住这种被夺权被掌控的极端的情况就是习惯真的已经非常根深蒂固。

craving 在控制你的时候，这个时候其实我们已经不再是我们自己了，整个人是否还有自主行为能力都要打个问号。我觉得这种情况下已经非常接近寄生虫控制的情况下，完全失去自己身体控制权的宿主。我们知道梦游的人在梦游中如果犯了罪，只要证明事实如此，就可以被判无罪。因为梦游的时候，我们之所以我们认为自己之所以为人的那个人的部分，也就是能够自主行为的部分是在沉睡，他是没有意识到自己在干这件事的，那犯罪行为是大脑的一种，就是病变，它导致你没有你的那个沉睡的自主行为的部分，没有办法意识到和控制住。

3、习惯的作用。

习惯可用于培养好习惯，抑制坏习惯，还可用于营销。

如何培养好习惯？设定 cue，设定 reward，然后让 routing 足够简单，如果能形成 craving 就更好了。比如想要培养小娃每晚阅读的习惯，可以设定一个时间的 cue「每晚 8 点」或者地点的 cue「晚上上床后睡觉前」；然后提前选好书并把书放在床头让 routing 简单，这样就不需要每次想着要读书还要先选书和去书房找书；设定 reward 比如读完一本书奖励积分卡，通过积分卡可以兑换提前准备的玩具或食物。而是否能形成 craving 这件事得分人，多尝试不同的 reawrd。要有 craving 比如产生多巴胺（还有其他的），通常有三类可以产生多巴胺的 reward，比如食物、新鲜的事物、多变的奖赏。感觉很多学霸应该都是对学习后的 reward 有 craving 的那种人。

如何避免在无意识的情况下养成坏习惯？勿以恶小而不为，把他们掐灭在萌芽状态，不要形成习惯。比如闯红灯、不良饮食习惯、刷抖音、说谎、负面思考、不锻炼、久坐、急躁和冲动、晚睡、过度消费、过度使用社交媒体、吸烟、酗酒、抱怨、乱丢垃圾、害怕困难和挑战等等，当这些形成习惯，在脑子里的折痕越来越深，那以后就可能会不动脑子无意识触发，有些可能还挺危险的，比如闯红灯。

在坏习惯已经养成的情况下，如何抑制他？找到 cue 并避免他，让 routing 变得麻烦，或者找到 reward 之后换一个能达到同样 reward 的 routing。比如要减少获取资讯的隐，可以通过「设定计划」减少 cue，因为有了计划之后，每个时间点就应该做另一件事而不是获取资讯；比如通过「取消 Reeder、微信、钉钉切换的快捷键」「把手机放抽屉里」「用 Focus 软件强制不让特定软件和网站开启」等方法，增加 routing 的复杂度；比如通过「减少频次」「定时每晚获取」等方法，换一种能达到相同 reward 的不同的 routing 方式。

4、核心习惯。

什么叫核心习惯？核心习惯是能带动其他习惯或引起连锁效应的习惯。对我来说，早睡、早起、运动、和家人吃饭、制定计划，都是核心习惯。比如说早睡，他会影响到第二天的早起，进而影响第二天开始工作的时间，接着影响第二天一天的工作安排。所以我考虑每天 11:00 睡觉。

核心习惯产生的涟漪效应会连带所有其他好习惯的建立，所以我们应该先达成一个小目标，还是那个道理，勿以善小而不为。我们先完成一个小善，它就会产生涟漪效应，一层一层的连带出去，产生好多个核心习惯的好处。

> 2024.06.03 更新，我的核心习惯有哪些？1）早睡，所以第二天能早起、运动、有个好状态等，2）年月周计划，3）选择难的，4）写作。

5、自律是不存在的。

自律是不存在的，自律是你对意志力聪明的使用损耗比较少的结果，同时双管齐下，你应该去进行体育锻炼，把自己的意志力建池子扩建，建得更大，同时对池子里面的这个水用的比较少，这个时候你每天的意志力很充足，那你就会觉得自己是一个自律的人。在这种情况下，其实我们就能够反复的重复一些健康的行为，那我们的好习惯就能够更好地建立起来，就会进入一个良性的循环。
