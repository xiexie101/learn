# 246 - 《装了啥 2023》

> 2023 版。本文耗时 2h+，写整理类的日更最花时间…

## 科学上网

> 和 2022 版没啥区别。

*   机场是 [IPLC.VIP](https://thirdislandchain.com/aff.php?aff=1717)，￥488/Y，用了 3 年了，优点是稳定
*   软件方面，Mac 用 [ClashX Pro](https://github.com/yichengchen/clashX)，Windows 用 [Clash for Windows](https://github.com/Fndroid/clash_for_windows_pkg)，iOS 用 ShadowRocket

## 编程工具

> 2023 版新增了 ChatGPT 和 DevUtils。

*   编辑器主 [WebStorm](https://www.jetbrains.com/webstorm/) 开发项目，辅 [VSCode](https://code.visualstudio.com/) 临时编辑项目，再辅 [CotEditor](https://coteditor.com/) 临时打开文件（注：如果是 Rust 项目，会用 VSCode）
*   WebStorm 插件只装了两个，[Github Copilot](https://github.com/features/copilot)、[Inspection Lens](https://plugins.jetbrains.com/plugin/19678-inspection-lens)，前者大幅提升编码效率，后者大幅提升编码体验
*   编程字体用 Monolisa、[Dank Mono](https://dank.sh/) 和 [Operator Mono](https://www.typography.com/fonts/operator/overview)，一个用久了看腻了切另一个
*   Terminal 工具是 [iTerm2](https://iterm2.com/) + zsh + [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) 的组合，zsh 额外装了 [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) 和 [fast-syntax-highlighting](https://github.com/zdharma/fast-syntax-highlighting) 插件
*   [ChatGPT](https://chat.openai.com/) 放这里不知是否合适，但今年遇到不会的确实大概率会先问他一下，感觉是以为虚拟的编程老师
*   其他辅助工具包括，API 工具 [Paw](https://paw.cloud/)（in-setapp）、Hosts 管理工具 [Gas Mask](https://github.com/2ndalpha/gasmask)、GUI Git 工具 [SourceTree](https://www.sourcetreeapp.com/)、辅助工具集 [DevUtils](https://devutils.com/)（in-setapp，备选 [He3](https://he3.app/) 和 [DevToys](https://devtoys.app/)）、取⾊⼯具 [ColorSnapper2](https://colorsnapper.com/)

## 知识管理

> 2023 版废弃了 LogSeq、NotePlan、滴答清单、Bear、Notion、OmniOutliner，新增了 Obsidian、RWR（ReadWise Reader）、iA Presenter 和 Whisper，详见 [241 - 《我的信息流 2023.1》](https://sorrycc.com/information-flow-2023)。

*   信息消费部分，用 [Reeder](https://reederapp.com/) 阅读 RSS 源的信息，用 [ReadWise Reader](https://read.readwise.io/) 读电子书和一些 Youtube 视频，用 [DeepL](https://www.deepl.com/) + [Bob](https://bobtranslate.com/) + 自研中英对照脚本 + [Whisper](https://github.com/openai/whisper) 阅读非中文的文章、书、视频和播客
*   信息输出部分，主 [Obsidian](https://obsidian.md/)，辅 [Flomo](https://flomoapp.com/)、[Tot](https://tot.rocks/) 和 Apple Notes，Obsidian 保存了我 90% 的内容，Flomo 保存非技术类的文章阅读笔记，Tot 保存临时想法，Apple Notes 保存生活类的，优点是可以设置文件夹和家人共享
*   信息输出辅助工具包括，用于临时编辑单个的 Markdown 文件的 [IA Writer](https://ia.net/writer)、一键上传图片的 [upic](https://blog.svend.cc/upic/)、GIF 录屏工具 [LICEcap](http://www.cockos.com/licecap/)、截屏编辑工具 [Shottr](https://shottr.cc/) 和 [CleanShot X](https://cleanshot.com/)（in-setapp）、用 Markdown 做 PPT 的 [iA Presenter](https://ia.net/presenter)
*   任务管理废弃了滴答清单，直接 Obsidian 进行管理，做年、周、日计划，做时间记录，参考 [237 - 《2023 年应该如何做事》](https://sorrycc.com/how-things-should-be-done-in-2023)

## 系统利器

> 2023 版废弃了 VLC、Short Menu，新增了 Focus、Eagle、yt-dlp、Espanso、Battery Buddy。  
> 按我的推荐度排列，靠前是非常推荐或者使用频率很高的。

*   [SetApp](https://setapp.com/) 是省钱利器，包含大量优秀 App，相比单个买 App 能省很多钱，需要注意的是目前 Family Plan 已绝版（注：本文涉及的软件如在 SetApp 中包含的我会带上 `in-setapp` 标记）
*   [Alfred](https://www.alfredapp.com/) + Powerpack，除了应用启动，我觉得最好用的是，1）搜索 Chrome 收藏夹，2）搜索文件，比如搜索 Markdown 文件，然后用 iA Writer 打开编辑
*   [Focus](https://heyfocus.com/)，强制自己保持专注，每天用（in-setapp）
*   [Thor Launcher](https://apps.apple.com/cn/app/thor/id1120999687?mt=12) 用于通过快捷键切换应用，切换包含的功能有启动、显示和隐藏
*   [1Password](https://1password.com/) 用于密码管理，之前买的 Family 版，但是只有我一个人用，有点贵，今年可能换其他工具
*   [Bartender](https://www.macbartender.com/)，菜单图标隐藏工具（in-setapp）
*   [Karabiner Element](https://karabiner-elements.pqrs.org/)，键盘自定义工具，我只用来做了一件事，修改右 Command 为 Hyper 键，为快捷键的定制增加了更多选择，比如 Hyper+D 用于切换 iTerm2 的显示隐藏
*   [Eagle](https://eagle.cool/)，素材管理工具，我只用于保存网络图片，去年新增了 200 多张
*   [Espanso](https://espanso.org/)，用于替换文本，比如我想输入我的邮箱只要 ;mail 即可
*   [Paste](https://pasteapp.io/) 用于粘贴板管理（in-setapp）
*   [PDF Expert](https://pdfexpert.com/)，用于 PDF 标注、创建和管理的工具
*   [KeepingYouAwake](https://keepingyouawake.app/)，禁用系统自动休眠
*   [OBS](https://obs.camera/) + [KeyCastr](https://github.com/keycastr/keycastr) + [剪映](https://www.capcut.cn/) + [RDM](https://github.com/avibrazil/RDM) + …，用于直播和视频录制
*   [IINA](https://iina.io/)，用于视频播放
*   [CleanMyMac X](https://cleanmymac.com/)，系统清理工具，每次硬盘空间不足时就会想到他（in-setapp）
*   [RunCat](https://apps.apple.com/cn/app/runcat/id1429033973?mt=12)，显示 CPU 状态的小猫，CPU 占用越高，他跑地越快
*   [f.lux](https://justgetflux.com/)，调节⾊温的护眼⼯具
*   [Battery Buddy](https://batterybuddy.app/)，给菜单栏的电池换个有趣的图标
*   [Shiftlt](https://github.com/fikovnik/ShiftIt)，用于窗⼝管理，最常用的操作是缩放到全屏
*   [Yoink](https://eternalstorms.at/yoink/)，文件暂存（in-setapp）
*   [Screens](https://edovia.com/en/screens-mac/)，远程 Mac 连接⼯具，用来远程操作我的 Windows 电脑（in-setapp）
*   [HandBrake](https://handbrake.fr/)，用于视频格式转换
*   [Downie](https://software.charliemonroe.net/downie/)（in-setapp） + [yt-dlp](https://github.com/yt-dlp/yt-dlp)，视频下载⼯具，后者是命令行、开源且能力更强
*   [Parallels Desktop](https://www.parallels.com/)，虚拟机⼯具，使用频率低，又贵，还老大版本升级，并且升级完一直提醒你更新，准备弃坑

## 付费服务

> 今年这部分变化太多，就不写变更记录了。

*   视频类，腾讯视频 + 爱奇艺 + 芒果 TV + Youku + [Netflix](https://netflix.com/) + [Youtube Premium](https://www.youtube.com/premium) + Bilibili 大会员全订了，但是今年开始看地少了，应该会减少一些。有一些省钱攻略，Netflix 是和朋友合订，每个月 11 块多；Youtube Premium 换了个便宜的区，100+ 一年
*   游戏类，一直订着 [PlayStation Plus](https://www.playstation.com/en-hk/ps-plus/) 会员，玩的少，之前开是因为要和小朋友 MineCraft 联机，到期不准备续了
*   软件类，长期付费的较少，能买断的尽量买断，目前长期订的主要是几个，1）[Setapp](https://setapp.com/)，$17.48/M，5 个人合租，2）[1Password Family](https://1password.com/families/)，有点贵，1 个人用太浪费，准备弃坑，3）[Obsidian Sync](https://obsidian.md/sync)，官方的同步服务体验就是好，4）ReasWise Reader，$4/M，申请了发展中国家 5 折
*   网盘类，百度云盘 + 阿里云盘 + 115 + iCloud 基础版 + 电信天翼云，天翼云到期后会弃坑，其他保留
*   DevOps 类，主要想到的有 [Vercel](https://vercel.com/)（静态站点） + [Name.com](https://www.name.com/)（买域名） + [Cloudinary](https://cloudinary.com/)（存图片） + [Replit](https://replit.com/)（简单服务） + [Render](https://render.com/)，预计今年会在这块增加投入
*   AI 类，持续订的只有 Text2Ismage 服务 [MidJourney](https://www.midjourney.com/)，为我打开了新的世界的大门
*   知识付费类，今年大部分都没续了，保留了[少数派会员](https://sspai.com/prime)、崇旭的价值阅读群、生财有术，新增了 [Untag 会员](https://utgd.net/member)，还付费购买了一些课程，包括 [CSS for JavaScript Developers](https://courses.joshwcomeau.com/css-for-js)、[React Query 官方课](https://ui.dev/react-query)、[Threejs Jounery](https://threejs-journey.com/)、Dan 的 [Just JavaScript](https://justjavascript.com/) 等，当然这些都是一次性付费的

## 硬件

> 2023 版新增了好多，只列和工作配置搭边的部分。

*   iPhone 12 Pro Max，没有换新
*   MacBook Pro (13-inch, M1, 2020)，公司配的，感觉内存不够用了，因为 WebStorm + Arc 真的很吃内存
*   [Caldigit TS4](https://www.caldigit.com/thunderbolt-station-4/)，让我从切换的 Mac 无线到有线，网速拉满，相见恨晚
*   Apple Watch Series 8，对我来说主要是两个用途，1）解锁 MacBook Pro 用，让我少输了无数次 1Password 密码，2）运动记录和打卡
*   Windows 电脑，配置是 RTX 3070 TI + Intel i7 12700K + 华硕 TUF GAMING Z690-PLUS D5 + 威刚XPG 16Gx2 + 三星980 PRO 2T，游戏机
*   NAS 新加了一台 DS920+ + 希捷企业级16T\*4，最大的问题是硬盘响，准备给他搬个位置
*   Sony A7M4 相机 + 24-70 GM 镜头 + Elgato Cam Link 采集卡 + Blue Yeti 麦克风，直播和视频录制用
*   [U28E590D](https://detail.tmall.com/item.htm?id=523282229383)，家里用的三星显示器，应该是 4K 中最便宜的了
*   (HHKB Pro 2 无刻 + [彩色键帽](https://item.taobao.com/item.htm?id=522721338431&_u=41h6urte838)) x2，[Razer DeathAdder Chroma](http://www.razerzone.com/store/razer-deathadder-chroma) x2，公司和家里用一样的键盘和鼠标，保持体验一致
