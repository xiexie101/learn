# 62 - 《高效使用 Chrome 收藏夹》

快周末了，分享个轻松点的话题，推荐下我的 Chrome 收藏夹使用方法给大家，10 分钟写完，😄。收纳时清楚放哪，使用时精确访问。

1、分类

见图1。我是把笔记的分类直接复刻到收藏夹中，基于 P.A.R.A 分类法做了一些扩展，加入 Inbox，然后把重要 Area（Programming、Family）往外提。这也是过年那段时间整理的，花了不少时间。

![](https://img.alicdn.com/imgextra/i2/O1CN01wyUdku26QVCbg8QFi_!!6000000007656-0-tps-1774-1036.jpg)

几个 Tips。1）条目用「标题 | 描述的形式」，可以加一些拼音简写作为关键词，方便搜索；2）Inbox 作为收集箱，不清楚放哪或临时的收藏先放这，参考图 2。

![](https://img.alicdn.com/imgextra/i1/O1CN01uzWh2O1LPmdleml2C_!!6000000001292-0-tps-748-1084.jpg)

2、快速访问

基于 Alfred。参考图 3 设置开启「Google Chrome Bookmarks」，然后在 Alfred 里输入关键词即可过滤出结果，参考 图 4。
