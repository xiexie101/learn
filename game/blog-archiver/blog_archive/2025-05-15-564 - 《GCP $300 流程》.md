# 564 - 《GCP $300 流程》

> 由于近期 Google 的 Pro 模型不再允许免费的 API Key 使用，因此我想着用一些渠道来继续使用。然后发现 GCP 的 90 天 300 刀的流程，试着走了一遍，没想到成功了。在此分享给大家，但不保证成功，只是我自己试过可以。

1、注册 Google 账号。

访问 [https://mail.google.com/](https://mail.google.com/) ，注册，要接码时找个接码平台，我用的 [DaisySMS](https://daisysms.com/r/amWiNfCW) （需要虚拟货币充值，你可以不用这个，[这里](https://linux.do/t/topic/497804)和[这里](https://linux.do/t/topic/642300)有很多别人整理的接码平台列表）。

2、注册 [WildCard](https://bewildcard.com/i/SORRYCC)、开卡、预存一点钱，比如 $2，GCP 会有扣款校验。（注：一张卡可以开多个 Google 账号，我同时开了两个。）

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/I9sk85.png)

3、获取 GCP $300 。

访问 [https://cloud.google.com/](https://cloud.google.com/) ，点「Get started for free」，填信息和卡（信息和卡在 WildCard 里选「Google」填就好。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/2zrr1s.png)

4、获取 API Token 。

选「APIs & Services」，选「Enable APIs and services」，搜「Gemini API」，确保其「API Enabled」。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/A2W2Mw.png)

选到「APIs & Services」，然后选「Credentials」。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/l9VsZp.png)

然后「Create credentials」选「API key」。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/asPemC.png)

拿到 key 就可以在 Chat App 或编程里使用了。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/image-2025-04@main/uPic/f7Dhaz.png)
