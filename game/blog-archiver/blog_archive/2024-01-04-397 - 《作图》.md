# 397 - 《作图》

> 向同事请教了下如何作图，记录下方法。

先看效果。

![](https://img.alicdn.com/imgextra/i2/O1CN01NC0ghT1ZMUP93mFob_!!6000000003180-2-tps-2042-598.png)

准备工作，三部分，图 + 字体 + 设计软件。

1、下载 Zoommy，找图用，上面都是无版权的图，集合了包括 unsplash 在内的数十个图片源，支持搜索  
2、下载字体，注意版权问题，得用可免费商用的，可以去小红书找，然后去阿里云盘或 github 下  
3、下载 Photoshop 或 Sketch，感觉后者够用

作图，以 Sketch 做图片+文字的方法为例：

1、新建固定宽高的 Artboard，比如公众号封面图是 900x383  
2、zoommy 找符合场景和氛围的底图  
3、加文字，并调整文字的字体和样式
