# 232 - 《2022 年度产品、游戏、硬件和付费服务》

以下只记录今年新接触的内容，如果大家之前关注我的内容，大部分都有提到过。

## 年度产品

1、ChatGPT + Mid Journey  
2、DeepL  
3、Obsidian  
4、z-library  
5、知识星球

ChatGPT + Mid Journey 带我入门 AI 应用，感受到 AI 带来的各种便利，同时也借机开始关注此领域，日新月异，每天都有新应用产生；DeepL 结合我自研的脚本，可以将任意网页转成双语格式，让我可以用母语的速度阅读英语文章，效率 ++；自从被 LogSeq 坑了一把后就切到 Obsidian，完全满足了我的需求，不满足的也有插件提供支持，这几年应该不会切了笔记软件了；z-library 让我电子书自由；知识星球让我一年写了 230 篇文章，同时给我带来额外的社区和收入。

## 年度游戏

1、SIFU  
2、战神 5：诸神黄昏  
3、艾尔登法环

全部通关。SIFU 配个叶问的 Mod 手感极佳。

## 年度硬件

1、新配的 PC 电脑  
2、RFID 迷你复卡机  
3、CalDigit TS4  
4、Apple Watch  
5、DS920 + 希捷企业级16Tx4  
6、Sony A7M4  
7、Philips S7786 刮胡刀

年中花 1.5W 配了台 PC，让我 PC 游戏自由，额外的收获是后来接触 AI 后可以用来跑重 GPU 任务；RFID 迷你复卡机让我把小区门禁卡复制到圆片上然后贴在手机后面，给全家人带来了生活上的遍历；CalDigit TS4 附加的好处是让我从 wifi 切换到有线网络，网速 ++ 且稳定；Apple Watch 让我少输入了无数次密码（其中大部分来自 1Password）。

## 年度付费服务

1、ReadWise Reader  
2、CSS for JavaScript Developers  
3、9DM（PC 游戏自由）  
4、知识资产  
5、少数派 Prime 会员第二季  
6、untag  
7、replit  
8、阿里云盘（淘宝花 23 快买 8T 空间）

ReadWise Reader 还没开始重度用，第一眼看到挺惊艳的，已付费并替换 Instapaper 的功能；Josh W Comeau 的 CSS for JavaScript Developers 课程让自认为 CSS 技能纯熟的我也学到不少，时间原因，还没有翻完；100 不到 9DM 付费群让我 PC 游戏下载自由。

## 年度软件

1、iA Presenter  
2、Arc Browser  
3、Focus

iA Presenter 用来做过一次分享，快速完成 PPT 而且视觉效果极佳；Arc Browser 有很多创新，很想接着用，但是内存占用太大到影响使用，只能先观望。
