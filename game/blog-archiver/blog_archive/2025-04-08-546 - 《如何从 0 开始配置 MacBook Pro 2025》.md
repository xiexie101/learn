# 546 - 《如何从 0 开始配置 MacBook Pro 2025》

> 昨天（2025.04.07）拿到了公司的一台新的电脑，MacBook Pro 14寸 M4 Pro 48G 1T，花了一些时间从零开始做配置，基于 [257 - 《如何从 0 开始配置 MacBook Pro》](https://sorrycc.com/mac) 做了些删减和更新。

1、Surge。

下载 [https://nssurge.com/](https://nssurge.com/) ，激活 LICENSE，安装 Network Extension，拷贝配置文件，开启并启用「Enhanced Mode」。

2、安装 App 和 Cli 。

1）先安装 [brew](https://brew.sh/) ，完成后记得执行一些结尾给出的命令，把 brew 添加到 PATH 里。

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2）然后用 brew 安装 App 和 Cli 。

安装 App 。

```bash
brew install --cask alfred battery-buddy coteditor espanso figma google-chrome gas-mask iina iterm2 karabiner-elements keepingyouawake keka keycastr licecap microsoft-edge obsidian shottr sourcetree thor visual-studio-code visual-studio-code@insiders cursor orbstack chatwise ghostty rectangle
```

安装 Cli 。

```bash
brew install autojump ffmpeg gh git tig jq tig tree wget hub
```

Mac App Store 安装 App 。

*   Bob，翻译软件
*   Tot，极简笔记
*   PDFgear，代替 PDF Expert
*   PastePal，粘贴板工具

再手动补一些 App 。

*   [微信输入法](https://z.weixin.qq.com/)，开启设备间同步后数据就都有了。
*   [Bartender](https://www.macbartender.com/)，隐藏菜单栏。
*   [Wispr Flow](https://wisprflow.ai/)，语音录入。
*   [uPic](https://github.com/gee1k/uPic)，图片上传。
*   [Flomo x Pake](https://github.com/tw93/Pake)，更简单的方式访问 Flomo。
*   [豆包](https://www.doubao.com/)，主要用他的语音输入功能，感觉比 Wispr Flow 速度更快，而且是流式的。
*   [Screen Studio](https://screen.studio/)，屏幕录制。

再补一些 Cli 。

*   [Bun](https://bun.sh/) 。

```bash
curl -fsSL https://bun.sh/install | bash
```

*   [Volta](https://volta.sh/)，用于管理 Node 版本。

```bash
curl https://get.volta.sh | bash
volta install node@22
```

4、配置。

1）系统配置，让这个电脑使用起来更顺手一点。

*   Language & Region
    *   把 First day of week 改成「Monday」
*   Trackpad
    *   去掉「Natural scrolling」
    *   选上「Tap to click」
*   Keyboard
    *   把 Key Repeat 调到「Fast」，把 Delay Util Repeat 调到「Short」
    *   Shortcuts 里，Mission Control 用「⌥A」,Application windows 用「⌥S」，Show Desktop 用「⌥D」，Input Sources 的 Select Previous 用 「⌘Space」，Screenshots 里 Save picture of selected area as a file 用「F19 + 3」，Copy picture of selected area to the clipboard 用「F19 + 4」
    *   输入法删除默认的拼音改用微信输入法
*   Spotlight
    *   只开 Applications、Bookmarks & History、Documents、Folders、System Preferences。
*   Mission Control
    *   把 Hot Corners 里的全部关掉，不需要，因为有 Thor 了，可以更快切除应用。
*   Sharing
    *   改下 computer name。
*   Notification
    *   不必要的全关掉，我只开了 Calendar、Find By 和 Reminders。
*   Desktop & Dock
    *   禁用「Close windows when quitting an application」。
*   Siri
    *   禁用。

2）Karabiner-Elements 。

参考 [Karabiner-Element 配置 F19 键 - HackMD](https://hackmd.io/@UXqYDTxCTie91Shvsppqyw/rk4u9i-pG?type=view) 。在 [Karabiner-Elements complex\_modifications rules](https://pqrs.org/osx/karabiner/complex_modifications/) 搜「Change caps\_lock key」，import 后只保留一条和 F19 相关的，然后在命令行里编辑「~/.config/karabiner/karabiner.json」，把刚才那条规则的「caps\_lock」换成「right\_command」（两处）。这样你就把基本不会用到的「右⌘」废物利用变成了「F19」键，然后你的快捷键组合会多很多。

3）Ghostty。

```
clipboard-read = allow
clipboard-write = allow
copy-on-select = clipboard
font-size = 18
font-family = MonoLisa
theme = iTerm2 Tango Dark
```

4）Git 。

```
git config --global user.name "Your Name"
git config --global user.email "you@your-domain.com"
git config --global --add push.default current
git config --global --add push.autoSetupRemote true
```

5）SSH 。

```bash
mkdir ~/.ssh
# file name 用 github，passphrase 随意
ssh-keygen -t ed25519 -C "github"
# 编辑配置，内容如下
touch ~/.ssh/config
Host *
  AddKeysToAgent yes
  UseKeychain yes
  IdentityFile ~/.ssh/github
# 添加到系统 keychain
ssh-add --apple-use-keychain ~/.ssh/github
# 添加 public key 到 github
gh auth login
gh ssh-key add ~/.ssh/github.pub -t github
```

6）ZSH。

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
omz update
source ~/.zshrc
# starship 是 rust 写的 prompt 工具，极快
brew install starship
echo 'eval "$(starship init zsh)"' >> ~/.zshrc
# plugins
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-completions ${ZSH_CUSTOM:-${ZSH:-~/.oh-my-zsh}/custom}/plugins/zsh-completions
git clone https://github.com/zdharma-continuum/fast-syntax-highlighting.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/plugins/fast-syntax-highlighting
```
