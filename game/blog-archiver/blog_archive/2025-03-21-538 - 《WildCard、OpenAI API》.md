# 538 - 《WildCard、OpenAI API》

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/lMyWaz.png)

之前找朋友冲的 OpenAI 的 API 已经用完了。然后想着重新冲一点，但又不想再麻烦别人了。就想着有没有一些渠道可以自己去操作。

早上在群里问了一下，大家推荐了 [WildCard](https://bewildcard.com/i/SORRYCC)。中午试了一下，发现还挺顺利的，应该十几二十来分钟就操作完了。我录了一个视频，整体非常流畅。

视频见 [https://g1i4rxq8hc.feishu.cn/docx/Vghxd5aVcoqLLix3G2ecavoNnH8](https://g1i4rxq8hc.feishu.cn/docx/Vghxd5aVcoqLLix3G2ecavoNnH8) 。

缺点是有一个开卡的费用，是 $11.99，加邀请码后减 $1，大家可以用我的邀请码「SORRYCC」；另外，还有百分之 3.5 的服务费。网站上给的指引也很完善。比如你去操作卡的时候，只要复制粘贴就可以。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/IsjCIM.png)

有一个要注意的是，OpenAI 在首次绑卡的时候会有一个 $5 的预授权费用。这个费用会在 14 个工作日左右退回。

然后我试了一下 Groq，也绑定成功了。 我之前试过其他的方式，都没有成功。

![](https://cdn.jsdelivr.net/gh/sorrycc-bot/images@main/uPic/D1UqPu.png)
