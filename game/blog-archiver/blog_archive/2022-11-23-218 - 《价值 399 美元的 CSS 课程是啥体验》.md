# 218 - 《价值 399 美元的 CSS 课程是啥体验》

趁着黑五打 5 折和香港区域 7 折，花 139 美元入手了原价 399 美元的 [CSS 课程](https://css-for-js.dev)。平时较多看作者的文章，每篇都写地深入浅出，早就想体验体验他的 CSS 课程，据说卖地特别好。下面说下体验。

1、我自认为 CSS 水平还是可以的，从 IE6 时代开始追求像素级完美，然后好多年没写过 CSS，前几年参与双 12 也还能把 Sketch 的设计稿像素级还原出来。昨天翻完课程的前几章，整理了 20 来条不会或不深入的知识点。作者能把 CSS 知识中有用的点整理出来，同时又讲得很深入浅出，这是他厉害的地方。所以我理解这门课 CSS 不管是啥水平的同学，看完都会有收获。

2、课程运营。1）结合黑五优惠，五折卖一波大的，2）课程分 3 个 Package，不同价格，面向有不同需要和不同承受能力的同学，3）写博客辅助卖课，比如他这周写的[《Flex 交互式指南》](https://www.joshwcomeau.com/css/interactive-guide-to-flexbox/)就是结合黑五优惠的文章。

3、整体分 0-9 个模块。第 0 个模块是其他模块的介绍和引导，通过一些知识点勾起读者对其他章节的兴趣。

4、课程整体是文字形式，但也穿插了很多视频。这些视频和文字是互补的，视频内容目前看到有两类。1）经验类的讲述，跳过不妨碍对课程的理解，这部分内容用视频是比较合适的，转换成文字就比较啰嗦了，2）操作类的，比如如何调试 CSS，用视频看更容易理解。缺点是，对于英文听力比较弱的同学不太友好。

5、即时反馈做的非常好。1）每一篇看完了可以标记为「已完成」，章节里有「完成百分比」，然后整体也有「完成百分比」，让读者能知道目前学完多少了，2）课程内穿插了大量的「Playground」，用户可直接修改代码进行验证，3）课程中还有少量交互式的习题和小游戏，选择后会给与反馈和解释，4）章节结束会有小练习，5）整体课程完成 60% 可收到「课程证书」。

6、周边。1）附带一个 Discord 社区，2）附带一个百宝箱，是作者原创的 CSS 相关的工具库，3）附带一个笔记系统，读者在课程中可以直接划词做笔记。

7、目前的感受是值这个价格的，就是贵了不止一点，能感受到作者花了非常多的心思。他的 React 课程也在准备中，预计明年推出，准备入手，哈哈。
