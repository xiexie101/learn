# 587 - 《我怎么用 AI》

> 6 月有写 [573 - 《在用的 AI 工具》](https://sorrycc.com/ai-tools)，更新一版。

1、模型。

国外三家的主流最新模型都在用，包括 Claude 的 Sonnet 4.5 和 Opus 4.1、GPT 5 和 GPT 5 Thinking、Gemini 的 2.5 Pro 和 2.5 Flash 。国内开源模型作为补充，包括 GLM 4.6、Qwen 3 Coder 和 Qwen 3 Max 。

我获取以上模型的方式如下：

*   **Claude。** 之前尼区账号被封，临时改用美区账号续费的，$20/M，可参考 [583 - 《尼区 Claude 续》](https://sorrycc.com/nigeria-claude-2) 。
*   **ChatGPT。** Plus + API 用户。Plus 在闲鱼搜 codex 买的，¥8 - ¥10 左右（但可能会被 deactive，需要找老板解决，有点麻烦）。
*   **Gemini。** 今年 7 月有啥活动获取了一年的 [Google AI Pro](https://one.google.com/about/ai-premium/) 的会员，Web 上的基本可随便用。同时 Google 的免费用户就可以用 Pro 和 Flash 的 API，但需花精力解 503 overload 和 429 报错。
*   **GLM 和 Qwen。** GLM 我买了 ¥20/M 的会员，可以敞开了用，但实际用量很有限，一是因为有 Claude 了，二是官方的 GLM 速度不太好。Qwen 模型可以用 [iflow](https://iflow.cn/) 或 [modelscope](https://modelscope.cn/) 平台的。

再介绍下邪教玩法 2api。比如我有 github copilot、warp 的会员，每个月都有一些用量，放着也是浪费，通过 2api 的方式就可以把其中的 claude 等模型的调用通过 api 的方式暴露出来，给其他场景用。但需注意潜在封号风险。

*   [claude-relay-service](https://github.com/Wei-Shaw/claude-relay-service) 支持 Claude、Gemini CLI、CodeX 和 Factory 的 2api，目前我只用其搭了 Claude 的，配了美区静态家宽 IP，方便自己多台电脑和朋友一起用。
*   用 [Warp2Api](https://github.com/libaxuan/Warp2Api) 和 [factory-go-api](https://github.com/libaxuan/factory-go-api) 在本机分别做了这两家的 2api，这样基本上主流模型的调用量就不愁了。其中 [factory](https://app.factory.ai/r/DUNS8B8K) 目前搞活动，可以免费薅其中的 claude 模型量，注：用[邀请链接](https://app.factory.ai/r/DUNS8B8K) 注册送 3800w 的 token，不用应该是送 2000w。
*   我写的 [neovate-code](https://github.com/neovateai/neovate-code) 默认支持 github-copilot 的 provider，不需要 2api 。
*   cursor、raycast 等也有 2api，但没用，感觉 claude 模型的量够用了。

最后的备选是 [OpenRouter](https://openrouter.ai/)，包含所有主流模型。

2、Chat 提问。

*   [Claude](https://claude.ai/)、[ChatGPT](https://chatgpt.com/) 和 [Gemini](https://gemini.google.com/) 的官网、App 和 Mobile App 都在用。日常提问用 ChatGPT，部分编程方向的会问 Claude + Opus 模型，Deep Research 用 Gemini。
*   偶尔用 [ChatWise](https://chatwise.app?atp=sorrycc)，用 API 的方式提问。

3、编程。

*   编辑器是 [Cursor](https://www.cursor.com/)，有买年付会员，但目前只用他的 Tab Tab 补全功能。
*   Agent 用 CLI 形式的，包括 Claude Code 和自研的 [Neovate Code](https://github.com/neovateai/neovate-code)，备选 Gemini CLI 和 [Codex CLI](https://github.com/openai/codex)。
*   编程模型主要是 Sonnet 4.5 和 Opus 4.1，有时也用 Qwen 3 Coder 和 GLM 4.6 。
*   Claude Code 会搭配 [Conductor](https://conductor.build/) 一起用。
*   CLI Agent 的 完整 Workflow 是 git worktree、brain storm、plan、execute，简单任务也会直接 execute 。
*   偶尔用 [Warp](https://www.warp.dev/) 。
*   很偶尔用 [CodeX 平台](https://chatgpt.com/codex)，解一些琐碎的任务。

7、其他。

*   [Suno](https://suno.com/)，音乐生成，闲鱼 ¥30 一个月买的共享账号，V5 的效果挺不错。
