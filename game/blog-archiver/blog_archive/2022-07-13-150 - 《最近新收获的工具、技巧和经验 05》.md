# 150 - 《最近新收获的工具、技巧和经验 05》

1、切换「无线」到「有线」，让网速拉满，相见恨晚啊。家里是新入了「CalDigit TS4」，好处是，1）接了网线之后，下载速度快，2）和 NAS 也是一个交换机下的有线连接，速度快，3）线材整洁了很多；缺点是1）贵、2）电源大。公司由于显示器自带 hub 能力，加根网线就解决了。

2、写[更「美味」的任务](https://blog.amazingmarvin.com/three-simple-rules-writing-great-tasks-to-do-list/)，实践了一段时间，感觉有效。图 1 是我今天的任务列表。三个原则，1）动词开头，2）最多 1 小时完成（如果不能，拆子任务），同时标上时间，3）任务明确，几周后看依旧知道怎么做。

![](https://img.alicdn.com/imgextra/i4/O1CN01OvIKNv1skk5KPCfce_!!6000000005805-0-tps-972-1034.jpg)

3、换了个搜狗皮肤，我用的[「Matrix 矩阵」](https://github.com/xiaochunjimmy/Sogou-Input-Skin)。

4、开始用「图怪兽、创可贴、稿定」做图，水印问题可以在 [greasyfork.org](http://greasyfork.org) 里搜「Kill Watermark」。

5、一个 Terminal 技巧，给自定义命令加了「,」前缀。好处是 1）和系统命令不冲突，2）巧「,」可以提示所有自定义命令，见图2，来自[一篇 2009 年的老文](https://rhodesmill.org/brandon/2009/commands-with-comma/)。

![](https://img.alicdn.com/imgextra/i3/O1CN01XUDvg01ZHRwduLZSo_!!6000000003169-0-tps-1422-194.jpg)

6、[又一个 Terminal 技巧](https://twitter.com/stefanjudis/status/1544034153871572994)，配置 `alias -s {html,css,js}=code` 和 `alias -s git="git clone --depth 1"`，然后比如直接输入 `git@github.com:umijs/umi.git` 就会做 clone，无需输入 git clone。

7、软件部分，新入了「大拍档剪辑助手」，「HandBrake」 用于视频格式转换，「To Text Converter」用于把 PDF 等格式转为文本
