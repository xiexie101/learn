# 124 - 《最近新收获的工具、技巧和经验 03》

1、眼馋 DeepL 的翻译效果很久，一直想方设法地想用上，但由于 DeepL 不接受国内信用卡，淘宝上买的又太贵而且容易被封，同时不想用官网上难用的 Web 服务，做了一些尝试终于找到完美解。1）用 Chrome 插件 [Deepl Translate](https://chrome.google.com/webstore/detail/deepl-translate-beta-vers/cofdbpoegempjloogbagkncekinflcnj)，还算方便，用了一段时间；2）最近看到 [https://github.com/clubxdev/bob-plugin-deeplx](https://github.com/clubxdev/bob-plugin-deeplx) 搭配 zu1k/deepl docker 镜像使用，直接通过 API 的方式使用，已经是能找到的完美解了。

![](https://img.alicdn.com/imgextra/i2/O1CN01KncOuc1IgK62h17Nu_!!6000000000922-2-tps-1943-1080.png)

2、我的干眼症护理经验（仅个人经验，不构成用药建议）。之前眼干，去浙二眼科看做各种治理，不仅不舒服而且效果不佳。一度以为可能不能再做电脑工作了。个人尝试有效的是用电脑时戴 Blutech 湿房镜（水槽里我滴的是海露） + 艾明可聚乙烯醇滴眼液，每天 3 次，日抛型，可长期使用。目前已恢复，不再有干涩的感觉。

3、执行 defaults write -g NSWindowShouldDragOnGesture -bool true 后可按住 `⌘+⌃` + 鼠标点击窗口任意地方拖动窗口，来自 [https://mmazzarolo.com/blog/2022-04-16-drag-window-by-clicking-anywhere-on-macos/](https://mmazzarolo.com/blog/2022-04-16-drag-window-by-clicking-anywhere-on-macos/)

4、新增 App [NeatDownloadManager](https://www.neatdownloadmanager.com/)，mac 下的 IDM，多线程下载器，尺寸仅 1M，搭配 [https://github.com/syhyz1990/baiduyun](https://github.com/syhyz1990/baiduyun) 可满速下百度盘

5、花了 ￥358 更新 Parallels Desktop 到 17，据说这个版本 M1 下会更快

6、删除 Apple 内置的 iMovie 和 Garage Band 应用，从来不用，未来也不会用，直接省出 2G+

7、购买了 [mdhweekly.com](http://mdhweekly.com) 域名，未来会把 MDH 周刊的内容和订阅都迁到这

8、看完[「Up and Running with Remix」](https://egghead.io/courses/up-and-running-with-remix-b82b6bb6)后，准备真实上手试试这个新框架，目前正在写一个有趣的小应用，基于 Remix + TailwindCSS + Prisma + [Fly.io](http://Fly.io) 等
