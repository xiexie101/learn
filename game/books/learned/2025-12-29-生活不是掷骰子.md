# 摘要

刘雪峰通过《生活不是掷骰子》展示了如何用贝叶斯定理应对生活的不确定性，将决策从运气的赌博转化为基于证据的理性迭代，以此建立科学的世界观。

# 想法

- 我们的直觉往往是错的，只有通过数学思维的矫正，才能接近世界的真相。
- 贝叶斯定理不仅是数学公式，更是人类大脑认知的底层逻辑。
- 所谓“情商高”，往往是因为大脑内部存储了更准确的先验概率模型。
- “两小儿辩日”的本质是使用了不同的观测模型（视觉大小 vs 触觉温度）得出了矛盾结论。
- 墨菲定律不是诅咒，而是因为我们对坏结果的记忆权重远高于好结果。
- 阴谋论之所以流行，是因为它们构建了一个“似然度”极高但“先验概率”极低的故事。
- 许多人把“小概率事件”当作“不可能事件”，或者把“有可能”当作“事实”。
- 索罗斯做空英镑并非仅仅是冒险，而是计算了不对称风险后的理性下注。
- 赵本山在《卖拐》中之所以能忽悠成功，是因为他通过心理暗示篡改了范伟的“先验概率”。
- 科学地扑点球需要混合策略纳什均衡，而不是单纯靠反应。
- 以前的经验（先验概率）加上当下的证据（似然度），等于对未来的判断（后验概率）。
- 我们对他人的刻板印象，本质上是大脑为了节省能量而使用的粗糙先验概率。
- 固执的人就像把先验概率设为 1 的模型，任何新证据都无法更新他的看法。
- 在信息不完备的情况下，贝叶斯思维教我们如何做出期望值最高的决策。
- 只有不断输入新信息并更新认知，一个人才能拥有真正的“成长型思维”。
- 很多“大师”利用幸存者偏差，把随机性的成功包装成必然的方法论。
- 垃圾邮件过滤器是贝叶斯定理最著名的应用之一，它不断学习从未见过的垃圾词汇。
- “奥卡姆剃刀”原理在数学上对应着对模型复杂度的惩罚项。
- 真正的理性不是没有情感，而是不被情感绑架了对概率的判断。
- 无论你怎么掷骰子，如果你知道骰子的重心偏了，你就能赢。

# 洞见

- 确定性是一种幻觉；成熟的标志是能够舒适地栖息在概率和不确定性之中，并依此行动。
- 我们的一生就是一个不断用新数据（经历）去更新旧模型（世界观）的大型贝叶斯推理过程。
- 许多所谓的“命运”或“运气”，其实是隐形概率分布在长周期中的必然显现；智者通过改变分布来改变命运。
- 批判性思维的核心不在于怀疑一切，而在于精准评估证据的权重，并据此调整信念的强弱。
- 真正的智慧不是一开始就知道正确答案，而是拥有一套能让错误答案快速收敛到正确答案的纠错机制。
- 阴谋论和伪科学利用了人类喜欢“连贯故事”胜过“离散数据”的认知漏洞；贝叶斯是抵抗这种诱惑的数学疫苗。
- 所有的决策本质上都是对未来的预测，而最准确的预测往往来自对过去数据最无情的客观分析。
- “听人劝，吃饱饭”的数学解释是：如果不信任他人的数据源，你的模型更新速度将慢得像在黑暗中摸索。
- 成功=（先验优势 + 持续迭代）× 运气；我们无法控制运气，但可以无限优化迭代过程。
- 世界不是黑白分明的二进制代码，而是一个即使在 99% 确定时仍保留 1% 怀疑的连续光谱。

# 习惯

- 在做决定前，先问自己：“这件事发生的基础概率（Base Rate）是多少？”
- 遇到违背常识的观点时，不要急着反驳，先评估证据的可靠性。
- 定期复盘过去的错误决策，分析是“运气不好”还是“模型不对”。
- 当新证据出现时，强迫自己量化地调整对某事的信心指数。
- 避免使用“一定”、“绝对”等词汇，改用“很有可能”、“大概率”。
- 阅读不同领域的书籍，为自己的大脑输入更多样化的先验知识。
- 在情绪激动时暂停决策，因为此时大脑的似然度评估功能已失效。
- 像训练AI一样训练直觉：通过大量高质量的案例反馈来校准判断。
- 面对复杂问题，通过画决策树来拆解概率分支。
- 主动寻找反驳自己观点的证据，以此来测试自己信念的稳固度。
- 记录下你的预测，事后验证准确率，以此修正你的“自信偏差”。
- 不要因为一次失败就全盘否定，要看样本量是否足够大。
- 学习基础的统计学知识，理解均值回归和标准差。
- 对“太好以至于不像是真的”的事情保持极高的贝叶斯怀疑。
- 把每一次对话都看作是获取新信息、优化模型的机会。

# 事实

- 贝叶斯定理通过 P(A|B) = P(B|A)*P(A) / P(B) 来计算条件概率。
- 即使检测结果是阳性，如果该病的患病率极低，你实际得病的概率可能依然很低（假阳性悖论）。
- 垃圾邮件过滤器不需要理解邮件内容，只需计算关键词出现的概率就能精准拦截。
- 第二次世界大战中，盟军通过统计德军坦克序列号（贝叶斯推断），比间谍更准确地估算了德军产量。
- 许多认知偏差（如确认偏误）本质上是大脑在处理概率信息时的偷懒行为。
- 大脑在处理视觉信息时，实际上是在进行无意识的贝叶斯推断（脑补画面）。
- 索罗斯在1992年狙击英镑时，并非确信英镑必跌，而是看到下跌的期望值巨大。
- 所谓的“第六感”，在神经科学上往往归结为无意识的模式识别和概率预测。
- 无论多小的先验概率，只有有足够强的证据（似然度），后验概率都能被逆转。
- 机器学习中的许多核心算法（如朴素贝叶斯分类器）直接源于这一数学原理。
- 只有不断犯错并从错误中更新信息，人工智能才能变聪明；人也一样。
- 墨菲定律的“如果坏事可能发生，它就会发生”其实是对大数定律的通俗误读。
- 我们的记忆并不可靠，每次回忆都是一次带有当下偏见的“重构”。
- 所有的科学理论都只是“暂时正确”的假设，等待着新证据的证伪或修正。
- 赌场是利用概率法则（庄家优势）最严格的商业机构。

# 建议

- 下次听到新闻头条时，先停下来想一想它的先验概率，别被标题党带节奏。
- 在工作中评估两份方案时，列出各自的优缺点并赋予权重概率，算一下期望值。
- 如果你觉得某人“针对你”，先算算他针对你的成本和概率，多半是你“自作多情”。
- 想要提升判断力，就多去接触你“不相信”的信息，看看能否动摇你的先验。
- 买彩票可以作为娱乐，但别把它作为致富策略，因为数学期望值是负的。
- 找对象时，不要只看当下的表现（似然度），也要考察他的过去（先验概率）。
- 别总想着“一战成名”，追求“大概率正确”的重复博弈才是长久之道。
- 遇到无法解释的巧合，先别急着迷信，计算一下巧合发生的统计学概率。
- 培养对“黑天鹅”事件的敬畏，给自己的生活留一点安全边际。
- 学会用数字说话，把“我觉得”改成“根据数据，我认为有X%的可能性”。
- 永远不要停止更新你的认知，在这个指数级变化的时代，停滞就是退步。
- 记住，好的决策不一定有好的结果（运气影响），但长期坚持好的决策过程一定会赢。
- 偶尔做一些反直觉的尝试（探索），可能会帮你跳出局部最优解。
- 承认无知是开启贝叶斯更新的第一步。
- 像投资一样投资你的时间，计算每小时投入的预期回报率。
