你可以直接将这份文档发给设计师，或者按照这个结构自己动手制作。这将确保 7 张角色卡在视觉上既统一又有区分度，且具备极强的实用性。

# 🎨 设计排版规范：CityWalk 角色任务卡

**Design Specs for CityWalk Mission Cards**

## 1. 物理规格 (Physical Specs)

* **推荐尺寸**：**A6 (105 x 148 mm)** 或 **塔罗牌尺寸 (70 x 120 mm)**。
* *理由：A6 适合放在手账本里；塔罗牌尺寸适合手持拍照。*


* **材质建议**：300g 铜版纸或特种纸（如大地纸），建议**圆角**处理（防止放在口袋磨损）。
* **色彩模式**：CMYK（印刷用）。

---

## 2. 正面排版：行动区 (Front Side: The Bingo)

**核心功能**：快速浏览任务，打钩确认。

### **A. 顶部信息栏 (Header) - 占高度 15%**

* **左上角**：角色图标（Icon）。
* *例：👁️ (感官) / 🍱 (美食)*


* **主标题**：角色名称（中文 + 英文）。
* *字体建议：粗黑体 / 艺术标题字。*


* **副标题/Slogan**：一句话定位（如：*“调动五感，捕捉细节”*）。

### **B. 核心任务区 (The Grid) - 占高度 70%**

* **结构**：标准的 **3x3 九宫格**。
* **格子样式**：
* 建议使用**线框**或**淡色色块**区分 9 个格子。
* 每个格子右下角或左上角，必须留出 **☐ 复选框** 或 **⭕️ 空心圆**，方便用户用笔打钩。


* **文字排版**：
* **主任务**（加粗）：如 **“找到2种暖色系”**。
* *辅助说明*（小字）：如 *“如砖红、橙黄”*。


* **视觉中心**：
* 九宫格正中间的那个任务（★星标任务），建议加一个**高亮底色**或**边框**，作为核心挑战。



### **C. 底部规则栏 (Footer) - 占高度 15%**

* **文案**：*“规则：完成任意 3 点连线即为通关 ✅”*
* **装饰**：可以放一个简单的二维码（指向你的社群或完整电子版手册）。

---

## 3. 背面排版：记录区 (Back Side: The Journal)

**核心功能**：沉浸式书写、贴票据、盖章。

### **A. 引导栏 (The Prompt) - 占高度 20%**

* **内容**：放置该角色的 **“灵感留白 (Deep Dive)”** 问题。
* **排版**：使用手写体或衬线体，营造文艺感。
* *例：“今天最让你意外的一种声音是什么？闭上眼，试着在心里形容它。”*



### **B. 留白创作区 (Canvas) - 占高度 65%**

* **样式选项**：
* **点阵 (Dotted)**：适合画图也适合写字，最灵活。
* **空白框 (Blank)**：适合粘贴照片或票据。
* **宽横线 (Lined)**：适合纯文字记录。


* **底纹**：可以在角落放置淡色的角色 Icon 水印，增加质感。

### **C. 信息记录栏 (Log Data) - 占高度 15%**

* **预设填空项**：
* `📅 日期：______`
* `📍 地点：______`
* `🌤 天气：______`
* `🚶 步数/时长：______`
* `心情指数：⭐️⭐️⭐️⭐️⭐️`



---

## 4. 视觉风格指南 (Visual Style Guide)

为了区分 7 个角色，建议为每张卡片设定一个**主色调**：

| 角色 (Role) | 建议主色 (Color) | 视觉元素/纹理 (Key Elements) |
| --- | --- | --- |
| **👁️ 感官收藏家** | **薄荷绿 / 天空蓝** | 眼睛、波纹、自然渐变色 |
| **🧘🏻‍♂️ 时间漫游者** | **静谧灰 / 米白** | 沙漏、时钟刻度、极简线条 |
| **🤝 连接实验员** | **暖橙色 / 珊瑚红** | 绳结、拼图块、对话气泡 |
| **🕵️ 城市侦探** | **深蓝 / 墨色** | 放大镜、脚印、指纹、网格纸 |
| **🎨 街头创作者** | **明黄 / 波普彩** | 相机取景框、画笔、胶卷 |
| **🍱 美食猎人** | **赤红 / 奶油黄** | 刀叉、热气、格子桌布纹理 |
| **📚 城市考据迷** | **牛皮纸色 / 砖红** | 邮戳、旧报纸肌理、古建筑剪影 |

---

### 💡 给设计师的特别提示

1. **易读性优先**：CityWalk 是在行走中阅读，字号**不能太小**（正文建议不小于 9pt）。
2. **书写友好**：卡片背面请**避免使用深色底色**或**覆亮膜**（UV），否则圆珠笔很难写上去。建议使用**哑光油**或**吸墨纸张**。
