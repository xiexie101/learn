# 掌心彩虹·焰之苍兰 ✨ (Hand-Yan)

这是一份为明天的福州花海公园面基专门准备的“程序员式浪漫”礼物。基于 MediaPipe 手势追踪与 Three.js 3D 粒子系统，将虚拟的魔法与真实的满目小苍兰花海完美融合。

![Three.js](https://img.shields.io/badge/Three.js-0.160-black?logo=three.js)
![MediaPipe](https://img.shields.io/badge/MediaPipe-Hands-blue)
![Romantic](https://img.shields.io/badge/Vibe-Romantic-pink)

## 🌸 专属设计元素

- 🌈 **雨后彩虹**：动态色彩流转的 3D 拱门，致敬她的专属昵称。（**默认开场**）
- 🌼 **小苍兰 (Freesia)**：奶油香槟金色的粒子汇聚，呼应福州花海公园的盛景。
- 🔥 **焰**：动态升腾的 3D 极光火焰图标，代表她的名字，温暖而灵动。
- 🎁 **魔法盲盒**：还内置了心动💖、繁星✨、霸王龙🦖等 10+ 种趣味 3D 形态。

## ✨ 交互指南

| 交互动作 | 视觉效果 | 浪漫寓意 |
|------|------|------|
| **翻转手掌** | 平摊 -> 侧立 -> 平摊，粒子会在 **10+ 种** 专属形态间循环切换 | 掌心里的万千世界，只为你变幻 |
| **握紧拳头 (石头)** | 粒子会向中心汇聚，发出**樱花粉色**的高频蓄力光芒 | 把所有的心意紧紧握住，蓄势待发 |
| **比个✌️ (剪刀)** | 触发“咔嚓”全屏快门特效，瞬间定格 AR 画面 | 记录下这瞬间的快乐与专属回忆 |
| **切换摄像头** | 点击 UI 上的按钮，在“自拍”与“拍风景”模式间一键切换 | 记录下花海，也定格下你 |

## 🎨 视觉风格：温柔奶油风
采用了低饱和度的马卡龙色系与奶油渐变，相比传统的赛博朋克风，更加柔和、高级，完美适配户外自然光环境。

## 📸 拍立得纪念功能

当触发拍照后，系统会自动生成一张带有：
1. **全彩实景背景**（真实的福州花海公园）
2. **AR 魔法粒子**（当前的 3D 唯美形态）
3. **拍立得专属边框**
4. **纪念落款**：“2026.03.14 · 福州花海公园 · 定格专属魔法”

生成的卡片支持在手机上**长按直接保存**。

## 🚀 快速启动

### 1. 本地运行 (用于测试)
```bash
# 在 hand-yan 目录下启动本地服务器
npx http-server -p 8080
```
访问：`http://localhost:8080/hand-yan/index.html`

### 2. 公网部署 (面基推荐)
建议部署至 **Vercel**, **Netlify** 或 **GitHub Pages**。
> **注意**：由于需要调用摄像头，必须通过 **HTTPS** 协议访问才能正常开启权限。

## 🛠️ 技术栈

- **3D 引擎**: Three.js 0.160 (WebGL)
- **AI 追踪**: MediaPipe Hands (实时手势骨骼算法)
- **交互逻辑**: 
  - `Rock-Paper-Scissors` 状态机触发快门
  - `Smart Mirroring` 智能镜像反转修正
  - `Canvas 2D` 高级拍立得渲染

## 📝 面基小贴士
1. **光照条件**：尽量在光线充足的地方展示，手势识别会更精准。
2. **镜头切换**：想拍花海背景下的她时切换到**后置**；想合影时切换到**前置**。系统已自动处理好镜像翻转。
3. **互动引导**：你可以说：“来，先出个石头蓄蓄力……一、二、三，比个耶！”
