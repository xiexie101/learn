# 掌心彩虹·焰之苍兰 ✨ (Hand-Yan)

这是一份为明天的福州花海公园面基专门准备的“程序员式浪漫”礼物。基于 MediaPipe 手势追踪与 Three.js 3D 粒子系统，将虚拟的魔法与真实的满目小苍兰花海完美融合。

![Three.js](https://img.shields.io/badge/Three.js-0.160-black?logo=three.js)
![MediaPipe](https://img.shields.io/badge/MediaPipe-Hands-blue)
![Romantic](https://img.shields.io/badge/Vibe-Romantic-pink)

## 🌸 专属设计元素

- 🌼 **小苍兰 (Freesia)**：金黄与白色的粒子汇聚，呼应福州花海公园的盛景。
- 🌈 **雨后的彩虹**：动态色彩流转的 3D 拱门，致敬她的专属昵称。
- 🔥 **吴陈焰**：发光的立体文字，在掌心上方徐徐绽放。

## ✨ 交互指南

| 交互动作 | 视觉效果 | 浪漫寓意 |
|------|------|------|
| **翻转手掌** | 平摊 -> 侧立 -> 平摊，粒子会在三个专属形状间循环切换 | 将这片花海与彩虹，送入你的掌心 |
| **握紧拳头 (石头)** | 粒子会汇聚在手心并开始高频蓄力发光 | 把所有的心意紧紧握住，蓄势待发 |
| **比个✌️ (剪刀)** | 触发“咔嚓”全屏闪光特效，定格当前的 AR 画面 | 记录下这瞬间的快乐与专属回忆 |

## 📸 拍立得纪念功能

当触发“石头剪刀布”拍照后，系统会自动生成一张带有：
1. **全彩实景背景**（真实的小苍兰花海）
2. **AR 魔法粒子**（当前的 3D 形态）
3. **拍立得专属边框**
4. **纪念落款**：“2026.03.14 · 福州花海公园 · 定格专属魔法”

生成的卡片支持在手机上**长按直接保存**。

## 🚀 快速启动

### 1. 本地运行 (用于测试)
```bash
# 在 hand-yan 目录下启动本地服务器
npx http-server -p 8080
```
访问：`http://localhost:8080/hand-yan/index.html`

### 2. 公网部署 (面基推荐)
建议部署至 **Vercel**, **Netlify** 或 **GitHub Pages**。
> **注意**：由于需要调用摄像头，必须通过 **HTTPS** 协议访问才能正常开启权限。

## 🛠️ 技术栈

- **3D 引擎**: Three.js 0.160 (WebGL)
- **AI 追踪**: MediaPipe Hands (实时手部骨骼追踪)
- **渲染技术**: 
  - `Additive Blending` 粒子发光效果
  - `preserveDrawingBuffer` 截图合成算法
  - `Canvas 2D` 拍立得边框与文案渲染

## 📝 面基小贴士
1. **光照条件**：尽量在光线充足的地方（如公园开阔地）展示，手势识别会更精准。
2. **手机适配**：建议让她用后置摄像头（通常网页会自动调用环境摄像头），由你举着手机，让她伸手进入画面，效果最惊艳。
3. **互动引导**：你可以开玩笑说要和她玩个小游戏，引导她对着镜头先出“石头”，然后再出“剪刀（比个耶）”。
