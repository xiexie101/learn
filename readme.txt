帮我创建一个PC网站，功能如下：
1. 有一个上传图片的入口，可以上传多个图片
2. 有一个合并的按钮，点击合并按钮可以将上传的图片合并在一起显示，合并的图片可以下载


合并图片的区域要实现如下功能：

1、图片可以随便拖拽改变顺序，比如图1在图2的右边，图三在图1的下面

2、你可以做一个类似画板一样的功能


激活环境
conda activate image-merger
启动服务器：
python -m http.server
在浏览器中访问：
http://localhost:8000

# 查看占用 8000 端口的进程
lsof -i :8000

# 会显示类似这样的输出：
# COMMAND  PID   USER   FD   TYPE             DEVICE  SIZE/OFF NODE NAME
# Python   1234  user   31u  IPv4  0x1234567890123456      0t0  TCP *:8000 (LISTEN)

# 使用显示的 PID 终止进程
kill -9 1234  # 替换为实际的 PID


# 先退出环境
conda deactivate

# 删除环境
conda remove -n image-merger --all


主要新功能：
图片可以自由拖拽到画布的任何位置
每个图片都可以调整大小
图片可以叠加，点击时会自动置顶
添加了清空画布的功能
合并时会保持图片的位置、大小和层级关系
使用方法：
上传图片后，图片会以可拖拽的形式出现在画布中
可以拖动图片到任意位置
可以通过右下角的手柄调整图片大小
点击图片会自动置顶
完成排列后点击"合并图片"即可生成最终图片
